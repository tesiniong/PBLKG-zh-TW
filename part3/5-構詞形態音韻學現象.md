# 5\. 構詞形態音韻學現象（DARYBOS MORFONOLOGINIAI REIŠKINIAI）

## 5.1. 詞根元音交替（Šaknies balsių kaita）

對比處於直接構詞關係中的詞語（基底詞與派生詞）可以發現，派生詞與基底詞的詞根元音往往不同。\[^III-99] 元音交替（balsių kaita）在所有詞類的構詞過程中均可能發生。元音交替的種類很多，但有些較為常見，有些則較為罕見：

1）i : y，比較 *ryšulỹs*（: *rìšti*）[^III-100]、*ryšỹs*（: *rìšti*）、*gỹmis*（: *gìmti*）、*skrỹdis*（: *skrìdo*, *skrìsti*）、*gylė*（: *gilùs*, *-i̇̀*）；y : i，比較 *septiñtas*, *-à*（: *septyni̇̀*, *-ios*）；

2）i : ie，比較 *liẽptas*（: *lìpti*）、*sniẽgas*（: *snìgti*）；ie : i，比較 *žvigdýti*（: *žviẽgti*）、*nemigà*（: *nemiẽga*, *niemiegóti*）；

3）i : ė，比較 *kėbė*「糾纏的一堆」（: *kìbti*）；ė : i，比較 *surìkti*（: *rėkti*）、*rikdýti*（: *rėkti*）；

4）i : e，比較 *nértėti*（: *nir̃to*, *nir̃sti*）；e : i，比較 *vìrkdyti*（: *ver̃kti*）、*kilnóti*（: *kélti*）、*gìrdyti*（: *gérti*）；

5）i : ei，比較 *bréizgėti*「芽枝亂長」（: *brìgzti*）、*géižėti*「變酸」（: *gìžti*）；

6）i : a，比較 *maršùs*, *-i̇̀*「記性差的」（: *mir̃šti*「不記得」）、*dárbas*（: *dìrbti*）、*ardýti*（: *ìrti*）；a : i，比較 *mìltai*（: *málti*）；

7）i : ai，比較 *káišioti*（: *kìšti*）、*láipioti*（: *lìpti*）；

8）i : o，比較 *lomà*（: *lìmti*「彎曲、傾斜」）；

9）i : u，比較 *sluñkius*, *-ė*（: *slìnkti*）；

10）y : ie，比較 *žiẽdas*（: *žýdi*, *žydė́ti*）、*lietùs*（: *lýti*）；ie : y，比較 *svỹdis*「拋擲；拋擲距離」（: *sviédė*, *sviésti*）、*są́lytis*（: *su(si̇̀)lietė*, *su(si)liésti*）；

11）y : ei，比較 *pléišėti*（: *plýšti*）；ei : y，比較 *saulė́lydis*（: *sáulė*, *léidosi*, *léistis*）；

12）y : ai，比較 *klaidà*（: *klýdo*, *klýsti*）、*kláidžioti*（: *klýdo*, *klýsti*）、*klaidi̇̀nti*（: *klýdo*, *klýsti*）；

13）ie : ai，比較 *pašaipà*（: *pašiẽpti*）、*graibýti*（: *griẽbti*）、*vaiši̇̀nti*（: *viẽši*, *viešė́ti*）、*raidà*（: *riẽda*, *riedė́ti*「流淌，溜走（關於時間）」）；

14）ė : y，比較 *rýkauti*「時而叫喊」（: *rėkti*）、*slypė́ti*（: *slėpti*）；

15）ė : e，比較 *rẽplės*（: *rėpti*）；e : ė，比較 *tėkmė*（: *tẽka*, *tekė́ti*）、*núotėkis*（: *nùteka*, *nutekė́ti*）；

16）ė : a，比較 *sajùs*, *-i̇̀*「易於播種的」（: *sė́ja* / *sė́jo*, *sė́ti*）、*šlakstýti*（: *šlėkti*「澆、灑」）；

17）ė : ai，比較 *bráižyti*（: *brė́žti*）、*brãižas*（: *brė́žti*）、*apýbraiža*（: *apibrė́žti*）；

18）ė : o，比較 *sodi̇̀nti*（: *sė́sti*）、*sóstas*（: *sė́sti*）；

19）ė : u，比較 *sriubà*（: *srėbti*）\[^III-101]；

20）ė : ū，比較 *sriū́bčioti*（: *srėbti*）；

21）ė : uo，比較 *úodas*（: *ė́da* / *ė́dė*, *ė́sti*）；

22）e : a，比較 *sagà*（: *sègti*）、*sagti̇̀s*（: *sègti*）、*naštà*（: *nèšti*）、*var̃tai*（: *vérti*）、*var̃žtas*（: *ver̃žti*）、*pašalpà*（: *pašel̃pti*）、*pastabà*（: *pàstebi*, *pastebė́ti*）、*pažangà*（: *pažeñgti*）、*pãšaras*（: *pašérti*）、*talkà*（: *tel̃kti*）、*trąšà*（: *trę̃šti*）、*lañkas*（: *leñkti*）、*nãras*（: *nérti*）、*vãdas*（: *vẽda* / *vẽdė*, *vèsti*）、*garbė*（: *ger̃bti*）；

23）ei : ai，比較 *pãkaitalas*（: *pàkeitė*, *pakei̇̃sti*）、*lai̇̃das*（: *léido*, *léisti*）、*sai̇̃kas*（: *séikėti*）\[^III-102]、*išlaidùs*, *-i̇̀*（: *išléido*, *išléisti*）；

24）a : o，比較 *nakti̇̀gonė*（: *nakti̇̀s*, *gãno* / *gãnė*, *ganýti*）、*grõžis*（: *gražùs*, *-i̇̀*）、*grožýbė*（: *gražùs*, *-i̇̀*）、*pašolỹs*（: *pašálti*）、*į̇́protis*（: *įprãto*, *įpràsti*）、*i̇̀šmonė*（: *išmãno* / *išmãnė*, *išmanýti*）；o : a，比較 *skabýti*（: *skõbti*）；

25）ai : ui，比較 *rui̇̃šis*「韁繩」（: *rai̇̃šti*）\[^III-103]；

26）au : u，比較 *niekdžiugỹs*, *-ė̃*（: *niekai̇̃*, *džiaũgtis*）、*pjudýti*（: *pjáutis*）；u : au，比較 *kiáuras*, *-à*（: *kiùro*, *kiùrti*）；

27）au : ū，比較 *šū̃kis*（: *šaũkti*）、*pjūti̇̀s*（: *pjáuti*）、*spūsti̇̀s*（: *spáustis*）；ū : au，比較 *sraunus*, *-i*（: *srū́ti*）；

28）o : ū，比較 *smū̃gis*（: *smõgti*）、*krūvà*（: *króvė*, *kráuti*）；ū : o，比較 *džiovà*（: *džiū́vo*, *džiū́ti*）；

29）o : uo，比較 *klúonas*「穀倉」（: *klóti*）；uo : o，比較 *dosnùs*, *-i̇̀*（: *dúoti*）；

30）u : ū，比較 *mū̃šis*（: *mùšti(s)*）、*pósūkis*（: *pa(si)sùkti*）；

31）u : uo，比較 *guõlis*（: *gùli*, *gulė́ti* / *gul̃ti*）\[^III-104]；uo : u，比較 *čiužinė́ti*（: *čiuõžti*）；

32）ū : ui，比較 *buiti̇̀s*（: *bū́ti*）\[^III-105]；

33）uo : ū，比較 *pačiū́ža*（: *pačiuõžti*）。

## 5.2. 詞根輔音交替（Šaknies priebalsių kaita）

在構詞過程中（主要是在構成後綴和詞尾派生詞時），詞根輔音有時也會發生變化：[^III-106]

1）t : č（račiùkas、rãčius : rãtas；balčius : báltas, -à）；č : t（bažnýtinis, -ė : bažnýčia；pė́stininkas, -ė : pė́sčias, -ià）；

2）d : dž（puõdžius : púodas；juodžiùkas : júodas, -a；skur̃džius : skurdùs, -i̇̀；smir̃džius : smi̇̀rdi, smirdė́ti；valdžià : val̃do / val̃dė, valdýti）；dž : d（pažodi̇̀nis, -ė : pažodžiui̇̃）。

## 5.3. 語素連接（morfemų sankloda）

**語素連接**（morfonologinis mazgas，形態音韻節點）是指相鄰語素的融合\[^III-107]，例如，後綴 -sena 的開頭與詞根末尾融合（即 s 同時屬於詞根和後綴）：drį̇̃/s/ena (: drį̇̃sti)、plė́/š/ena (: plė́šti)。\[^III-108] 比較不發生語素融合的派生詞，其中 s 只屬於後綴 -sena：ei̇̃sena (: ei̇̃ti)、mąstýsena (: mąstýti)。某些後綴 -ininkas, -ė 派生詞中，語素 in 也同時屬於基底詞和後綴，例如：i̇̀šimt/in/inkas, -ė（其中 in 來自基底詞 i̇̀šimtinė）；maš/i̇̀n/inkas, -ė (: mašinà)；ord/in/inkas, -ė (: ordinas)。比較同一構詞類型中不發生語素融合的派生詞：mi̇̀šk-ininkas, -ė (: mi̇̀škas)、jū́r-ininkas, -ė (: jū́ra)、móksl-ininkas, -ė (: mókslas)。

## 5.4. 換調（metatonija）

在構詞過程中，換調（kirčio ir priegaidės kaita，重音和音調的交替）十分常見。換調是同根詞語中韻律成分（重音和音調）的交替。在單音節的構詞詞幹中，發生音調的交替；在多音節（雙音節）的詞幹中，發生重音位置的變化，例如，比較前綴 be- 的派生詞 bežiẽdis, -ė、bepini̇̀gis, -ė 與其基底詞 ži̇́edas、pi̇̀nigas；後綴 -inti 的派生詞 várginti、skaitmẽninti（另一個可能的變體是 skai̇̃tmeninti\[^III-109]）與其基底詞 var̃gti、skaitmenà、skai̇̃tmeną。

換調分為重音2型換調（cirkumfleksinė metatonija）——基底詞的重音1（銳音）在派生詞中變為重音2（抑揚音）——和重音1型換調（akūtinė metatonija）——重音2變為重音1。重音2型換調更為常見。

自古以來，重音2型換調出現在以下情況：a）帶有前綴 be- 的形容詞以及複合詞的第二詞組成分中，例如：*bedãrbis*（: *dárbas*）、*bendrãdarbis*（: *beñdras*, *-à*, *dárbas*）；*beplaũkis*（: *pláukas*）、*baltaplaũkis*（: *báltas*, *-à*, *pláukas*）；*bešir̃dis*（: *širdi̇̀s*, *ši̇̀rdį*）、*kietašir̃dis*（: *ki̇́etas*, *-à*, *širdi̇̀s*, *ši̇̀rdį*）；b）詞尾 -is 的派生詞中：*geltõnis*（: *geltónas*, *-à*）、*stõris*（: *stóras*, *-à*）、*šaltis*（: *šáltas*, *-à*）。在構成新詞時，這一換調規律並不總是嚴格適用，比較重音1的傾向：*be-kãmeris*, *-ė* 1（: *kãmera* 1）、*be-vi̇̀dpadis*, *-ė* 1（: *vi̇̀dpadis* 1）、*daug-ia-kãmeris*, *-ė* 1、*dv-i-ãtramis*, *-ė* 1、*daug-ia-žánris*, *-ė* 1、*Áuksė*（: *áuksas*，人名）\[^III-110]。

重音2型換調和重音1型換調均可出現在由名詞（a）、形容詞（b）、動詞（c）派生的名詞中，比較：

a）*kal̃n-ius*（: *kálnas*）、*añt-inas*（: *ántis*）、*vìlk-ė*（: *vil̃kas*）；

b）*ménk-ė*（: *meñkas*, *-à*）、*méil-ė*（: *meilùs*, *mei̇̃lų*）以及 *mar̃g-is*（: *márgas*, *-à*）、*jaũn-is*, *-ė*（: *jáunas*, *-à*）；

c）*giñ-klas*（: *gìnti*）、*bėg-is*（: *bė́gti*）、*švi̇̀lp-a*（: *švil̃pti*）、*bùrb-a*「多嘴饒舌者」（: *bur̃ba*, *burbė́ti*）。

重音位置移動的例子：*be-asmẽnis*, *-ė*（: *asmeñs*, *ãsmenį*, *asmuõ*）、*asmén-vardis*（: *asmeñs*, *ãsmenį*, *asmuõ*, *vardas*）、*riešùt-medis*（: *ri̇́ešutas*, *mẽdis*；也可說 *ri̇́ešutmedis*）\[^III-111]；*Pa-nemùnė*（: *Nẽmunas*）、*vasãr-is*（: *vãsara*）、*vieni̇̀š-ius*, *-ė*（: *vi̇́enišas*, *-à*）或 *vien-ì-šius*, *-ė*（: *vi̇́enas*, *-à*）\[^III-112]、*lietùv-is*, *-ė*（: *Lietuvà*, *Lì̇etuvą*）、*vidur̃-kis*（: *vidurỹs*, *vì̇durį*）、*gelẽž-tė*（: *geleži̇̀s*, *gẽležį*）。

換調是構詞的輔助手段，其作用與詞根元音交替相似：元音交替和換調都不涵蓋某一構詞類型的所有派生詞（關於構詞類型見6.3節），即使用同一構詞形式素構成的派生詞，有些保留基底詞的重音和音調，有些則發生換調。（關於此現象更多見 Mikulėnienė 2005；比較 KTŽ: 124；LKE: 350；Barauskaitė 1985: 24–25, 34–35；Barauskaitė, Čepaitienė, Mikulėnienė, Pabrėža, Petkevičienė 1995: 172–180；Kazlauskaitė 2016。）

## 5.5. 複合詞連接元音（dūrinių jungiamieji balsiai）

複合詞的連接元音在第 2.4 節中已有概述。這裡補充說明：連接元音的選取通常取決於第一成分的形態特點（名詞或形容詞的語法性別、變格型等），但也有不規律的情況。在立陶宛語中，最常見的連接元音是 -a- / -ia-；-i- 相對常見；其他連接元音（-o-、-u-、-ė-、-y-、-ū-、-uo-）則較為罕見（詳見 Urbutis 2009: 278–290）。
