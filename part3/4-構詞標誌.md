# 4. 構詞標誌（Darybos formantas）

## 4.1. 單純與複合構詞標誌（Grynasis ir sudėtinis darybos formantas）

**構詞標誌**（formantas）是使**構詞產物**（darinys）區別於**詞基**（pamatinis žodis）或多個詞基的所有形式**構詞手段**（darybos priemonės）的總和 (Urbutis 2009: 247)[^fn88]。構詞標誌可以由單一個**詞綴**（afiksas）組成——**前綴**（priešdėlis）（1）、**後綴**（priesaga）（2）、**詞尾**（galūnė）（3）[^fn89]。這樣的構詞標誌被稱為**單純構詞標誌**（grynasis formantas），例如：
1) *ati-dúoti*「歸還」(: *dúoti*「給予」), *į-nèšti*「帶入」(: *nèšti*「攜帶」), *pó-dukra*「繼女」(: *dukrà*「女兒」), *ne-der̃lius*「歉收」(: *der̃lius*「收成」), *ne-gẽras*, *-à*「不好的」(: *gẽras*, *-à*「好的」), *nebe-jáunas*, *-à*「不再年輕的」(: *jáunas*, *-à*「年輕的」), *pa-i̇̀lgas*, *-a*「稍長的」(: *i̇̀lgas*, *-à*「長的」), *ùž-pernai*「前年」(: *pérnai*「去年」)；
2) *nauj-ai̇̃*「新地」(: *naũjas*, *-à*「新的」), *suprañtam-ai*「可理解地」(: *suprañtamas*, *-a*「可理解的」), *kit-ur̃*「在別處」(: *kìtas*, *-à*「其他的」)；
3) *sti̇̀kl-ius*「玻璃匠」(: *sti̇̀klas*「玻璃」), *blõg-is*「壞處 / 邪惡」(: *blõgas*, *-à*「壞的」), *trauk-à*「吸引力」(: *tráukti*「拉 / 吸引」), *pakil-ùs*, *-i̇̀*「高昂的 / 崇高的」(: *paki̇̀lti*「升起」)。

構詞產物更常具有**複合構詞標誌**（sudėtinis darybos formantas），即除了**主要詞綴**（pagrindinis afiksas）[^fn90]之外，還具有某種輔助構詞手段。值得注意的是，**構詞法**（darybos būdas）是根據主要構詞手段來決定的。詞尾極常作為輔助構詞手段。複合構詞標誌可以由以下部分組成：
1.  主要詞綴為後綴，且輔助構詞手段為詞尾（這類構詞產物被稱為**後綴派生詞**（priesagų vediniai）），例如：*aukó-***toj***-as*, *-a*「捐獻者」(: *aukóti*「捐獻」), *netek-***t***-i̇̀s*「喪失」(: *netèkti*「失去」), *išleis-***tùv***-ės*「畢業典禮 / 歡送會」(: *išléisti*「送出 / 發行」), *ajer-***ýn***-as*「菖蒲叢」(: *ãjeras*「菖蒲」), *vasar-***i̇̀n***-is*, *-ė*「夏天的」(: *vãsara*「夏天」), *liet-***i̇̀ng***-as*, *-a*「多雨的」(: *lietùs*「雨」), *peñk-***t***-as*, *-à*「第五」(: *penki̇̀*, *-ios*「五」), *kel-***er***-i̇̀*, *-ios*「幾個的（集合數詞）」(: *keli̇̀*, *-ios*「幾個」)。
2.  主要詞綴為前綴，且輔助構詞手段為詞尾（這類構詞產物被視為**前綴派生詞**（priešdėlių vediniai）），例如：***añt***-kakl-is*「項圈」(: *kãklas*「脖子」), ***pa***-kel-ė̃*「路邊」(: *kẽlias*「路」), ***ùž***-jūr-is*「海外」(: *jū́ra*「海」), ***ùž***-miest-is*「郊外」(: *miẽstas*「城市」), ***pó***-žem-is*「地下」(: *žẽmė*「土地」), ***pró***-vag-ė*「溝間空地」(: *vagà*「犁溝」), ***apý***-graž-is*, *-ė*「稍美的」(: *gražùs*, *-i̇̀*「美麗的」), ***be***-bai̇̃m-is*, *-ė*「無畏的」(: *báimė*「恐懼」), ***pó***-nauj-is*, *-ė*「稍新的」(: *naũjas*, *-à*「新的」)。
3.  主要詞綴為前綴，且輔助構詞手段為反身詞綴（這類構詞產物被視為前綴派生詞），例如：***pér-si***-valgyti*「吃得過飽」(: *válgyti*「吃」), ***už-si***-žiopsóti*「發呆忘神」(: *žiopsóti*「發呆 / 目瞪口呆」)[^fn91]。
4.  主要構詞手段為**詞幹結合**（sandūra），且輔助構詞手段為詞尾（這類構詞產物被視為**複合詞**（dūriniai））。如同2.4節所述，複合詞可以不帶**連接元音**（jungiamasis balsis）構成（a），或帶有連接元音[^fn92]（b）：
    a) *septyn-mỹl-is*, *-ė*「七英里的」(: *septyni̇̀*, *-ios*「七」, *mylià*「英里」), *rãt-lank-is*「輪輞」(: *rãtas*「輪子」, *lañkas*「環 / 箍」), *sén-vag-ė*「舊河道」(: *sẽnas*, *-à*「老的」, *vagà*「河床 / 溝」), *vien-marški̇̀n-is*, *-ė*「只穿襯衫的」(: *vi̇́enas*, *-à* (*vieni̇̀*)「一 / 單獨」, *marškiniai̇̃*「襯衫」), *nulėp-aũs-is*, *-ė*「垂耳的」(: *nulėpti*「下垂」, *ausi̇̀s*「耳朵」)；
    b) *ginkl-***a***-neš-ỹs*, *-ė*「武器攜帶者 / 侍從」(: *giñklas*「武器」, *nèšti*「攜帶」), *gandr-***ã***-lizd-is*「鸛鳥巢」(: *gañdras*「鸛鳥」, *li̇̀zdas*「巢」), *daug-***ia***-bùt-is*「多公寓的 / 多層住宅」(: *daũg*「多」, *bùtas*「公寓」), *tušč-***ia***-vidùr-is*, *-ė*「空心的」(: *tùščias*, *-ià*「空的」, *vidurỹs*「中間 / 內部」) (另見 Barauskaitė 1985: 29–30; 比較 5.5)[^fn93]。

## 4.2. 後綴（Priesaga）

後綴可以是唯一的構詞手段，即在 *ger-ai̇̃*「好地」(: *gẽras*, *-à*「好的」)、*ger-ỹn*「變好」(: *gẽras*, *-à*「好的」) 類型的構詞產物中作為單純構詞標誌，但更常見的是後綴與詞尾一起出現，即後綴作為複合構詞標誌的基礎，例如：*neš-***ė́j***-as*, *-a*「搬運工」(: *nèšti*「攜帶」), *plauk-***i̇̀k***-as*, *-ė*「游泳者」(: *plaũkti*「游泳」), *jū́r-***inink***-as*, *-ė*「水手」(: *jū́ra*「海」)。有時，後綴會進入這樣一種構詞標誌，其中它與前綴或**複合**（dūryba）處於同等地位（談到**混合構詞法**（mišrusis darybos būdas）的構詞產物時）：在第一種情況下，這是**前後綴構詞產物**（prefiksacijos-sufiksacijos dariniai），例如 ***pra***-meč-***iui̇̃***「隔年」(: *mẽtai*「年」)；在第二種情況下，是**複合-後綴構詞產物**（kompozicijos-sufiksacijos dariniai），例如 *pel-***ė́***-kau-***t***-ai*「捕鼠器」(: *pelė̃*「老鼠」, *káuti*「敲打 / 殺死」)。

某些後綴會發生變化，但要區分同一後綴的**變體**（variantai）並不總是容易。例如，**動詞**（veiksmažodis）抽象名詞後綴 *-imas* 和 *-ymas* 應被視為同一後綴的變體，例如：*neš-i̇̀mas*「攜帶（名詞）」(: *nèšti*「攜帶（動詞）」), *sup-i̇̀mas*「搖晃（名詞）」(: *sùpti*「搖晃（動詞）」)；*rãš-ymas*「寫作（名詞）」(: *rãšo* / *rãšė*, *rašýti*「寫（動詞）」), *dãr-ymas*「製作（名詞）」(: *dãro* / *dãrė*, *darýti*「製作（動詞）」)。變體還有**指小後綴**（deminutyvų priesagos）*-elis*, *-ė* 和 *-ėlis*, *-ė*，例如：*nam-ẽlis*「小房子」(: *nãmas*「房子」), *vaik-ẽlis*「小孩子」(: *vai̇̃kas*「孩子」)；*sūnait-ėlis*「小孫子」(: *sūnáitis*「孫子」), *vaikuž-ėlis*「小寶貝」(: *vaik-ùžis*「小孩子」)。同一後綴的變體還包括**變形**（atmainos）*-svas*, *-a* 和 *-švas*, *-a*，例如：*žal̃-svas*, *-à*「微綠的」(: *žãlias*, *-ià*「綠色的」), *pil̃k-švas*, *-à*「微灰的」(: *pi̇̀lkas*, *-à*「灰色的」)[^fn94]。**擴展後綴**（išplėstinės priesagos）（如 *-intelis*, *-ė*、*-uotinas*, *-ė* 等，例如：*nauj-iñtelis*, *-ė*「嶄新的」(: *naũjas*, *-à*「新的」), *vis-úotinas*, *-a*「普遍的」(: *vi̇̀sas*, *-à*「全部的」)[^fn95]）需要與**後綴叢集**（priesagų telkiniai）（多個構詞後綴）區分開來，例如，作為三個**構詞層級**（darybos pakopos）結果的 *rūpintojėlis*, *-ė*「小照顧者 / 愁容木雕」擁有三個獨立的構詞後綴 *-ėlis*, *-ė*、*-tojas*, *-a* 和 *-inti(s)*：*rūpintoj-ė̃lis*, *-ė*「小照顧者 / 愁容木雕」(: *rū́pintojas*, *-a*「照顧者」), *rū́pin-tojas*, *-a*「照顧者」(: *rū́pintis*「照顧 / 擔心」), *rū́p-intis*「擔心」(: *rū̃pi*, *rūpė́ti*「使...擔心」)[^fn96]。

## 4.3. 前綴（Priešdėlis）

前綴比後綴更常單獨作為構詞標誌。「純前綴派生最廣泛的領域是帶有前綴的動詞構詞」（Urbutis 2009: 259），例如：***at***-áugti*「重新生長 / 長回」(: *áugti*「生長」), ***su***-kviẽsti*「召集」(: *kviẽsti*「邀請」), ***pér***-auklėti*「重新教育」(: *áuklėti*「教育 / 撫養」) 等。派生詞通常由同一個基底動詞加上賦予各種意義色彩的前綴構成，例如從動詞 *eiti*「走」派生出 ***ap***-ei̇̃ti*「繞走」、***į***-ei̇̃ti*「走入」、***iš***-ei̇̃ti*「走出」、***nu***-ei̇̃ti*「走去」、***pa***-ei̇̃ti*「走一會兒 / 能走」、***par***-ei̇̃ti*「走回」、***pér***-eiti*「穿越」、***pra***-ei̇̃ti*「走過 / 經過」、***pri***-ei̇̃ti*「走近」、***su***-ei̇̃ti*「聚攏」、***už***-ei̇̃ti*「順道拜訪」。在**名詞**（daiktavardis）和**形容詞**（būdvardis）的構詞中，純前綴派生要罕見得多。在**靜詞**（vardažodis）的構詞中，更常見的是這種前綴派生：前綴的添加伴隨著詞尾（**屈折範式**（kaitybos paradigma））的改變，即前綴是複合構詞標誌的主要部分，而詞尾是輔助構詞手段（4.1），例如：***apý***-aušr-is*「黎明時分」(: *aušrà*「黎明」), ***be***-dar̃b-is*, *-ė*「失業者」(: *dárbas*「工作」), ***pa***-pilv-ė*「下腹部」(: *pil̃vas*「肚子」), ***pó***-rūgšt-is*, *-ė*「微酸的」(: *rūgštùs*, *-i̇̀*「酸的」)。如2.5節所述，前綴也會是混合構詞法（前後綴構詞）標誌的一部分，例如：***prieš***-kar-***i̇̀nis***, *-ė*「戰前的」(: *kãras*「戰爭」), ***prieš***-tánk-***inis***, *-ė*「反坦克的」(: *tánkas*「坦克」), ***pa***-srov-***iui̇̃***「順流地」(: *srovė̃*「水流」)[^fn97]。

## 4.4. 詞尾（Galūnė）

詞尾可以是唯一的構詞手段，即唯一的詞綴，亦即單純構詞標誌，這出現在諸如 *apžiūr-***à***「檢查（名詞）」(: *apžiū̃ri*, *apžiūrė́ti*「檢查（動詞）」)、*grei̇̃t-***is***「速度」(: *grei̇̃tas*, *-à*「快的」)、*bail-***ỹs***, -***ė̃***「膽小鬼」(: *bailùs*, *-i̇̀*「膽小的」)、*sẽn-***is***, -***ė***「老人」(: *sẽnas*, *-à*「老的」)、*stãl-***ius***「木匠」(: *stãlas*「桌子」)、*kib-***ùs***, -***i̇̀***「有黏性的 / 愛纏人的」(: *ki̇̀bti*「黏附 / 糾纏」)、*vakãr-***is***, -***ė***「晚上的 / 西風」(: *vãkaras*「晚上」) 等類型的構詞產物中，但更常見的是，在構成前綴派生詞、後綴派生詞或複合詞時，詞尾是作為次要的構詞手段，例如：*per̃-pet-ė*「肩帶 / 斜挎帶」(: *petỹs*「肩膀」), *ãpskait-inink-as*, *-ė*「會計員」(: *apskaità*「會計 / 記帳」), *burõk-lap-is*「甜菜葉」(: *burõkas*「甜菜」, *lãpas*「葉子」)[^fn98]。
