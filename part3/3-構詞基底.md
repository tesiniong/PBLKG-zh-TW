# 3. 構詞基礎（DARYBOS PAMATAS）

## 3.1. 詞基形式的選擇（Pamatinio žodžio formos parinkimas）

**詞基**（pamatinis žodis）是指在其他詞的**構詞法**（žodžių daryba）中所依賴的詞，或者是衍生出其他詞的詞。為了形成**構詞產物**（darinys），通常不以整個可**詞形屈折**（kaitomas）的詞或**語法形式**（forma）作為基礎，而是以**詞幹**（kamienas）作為基礎（Keinys 1999: 110; Urbutis 2009: 223）。詞基與構詞產物所共有的詞語部分，被稱為**基礎詞幹**（pamatinis kamienas）。**派生詞**（vedinys）具有一個基礎詞幹，例如：*ąžuol-ýnas*「橡樹林」(: *ą́žuolas*「橡樹」)、*núospaud-a*「壓痕 / 老繭」(: *nuspáudė*「壓下（過去單次時）」, *nuspáusti*「壓下（不定式）」)，而**複合詞**（dūrinys）具有兩個基礎詞幹，例如：*mėlyn-ãk-is, -ė*「藍眼睛的（陽性，陰性）」(: *mė́lynas, -a*「藍色的（陽性，陰性）」, *aki̇̀s*「眼睛」) (KTŽ: 142; Keinys 1999: 17)。相對而言，**名詞**（daiktavardis）和**形容詞**（būdvardis）在選擇基礎詞幹方面提供的自由度較小，因為它們所有形式的詞幹通常是相同的。不過，由第五**變格型**（linksniuotė）名詞所形成的構詞產物，既可以依賴短詞幹（**單數**（vienaskaita）**主格**（Vardininkas）），也可以依賴長詞幹（所有其餘**格**（linksnis））。大多數構詞產物具有長的**構詞詞幹**（darybos kamienas）（*akmen-*「石頭」、*dukter-*「女兒」、*piemen-*「牧羊人」），例如：*akmen-áuti*「撿石頭」、*akmen-ėlis*「小石頭」、*akmen-ỹnė*「多石的地方」、*ãkmen-iškas, -a*「像石頭的（陽性，陰性）」、*dukter-ė́čia*「姪女 / 外甥女」、*dukter-i̇̀nis, -ė*「女兒的（陽性，陰性）」、*piemen-áitis, -ė*「年輕的牧羊人（陽性，陰性）」、*piemen-ỹstė*「牧羊人的工作 / 放牧」、*piemen-galỹs, -ė*「牧童（陽性，陰性）」、*pi̇́emen-iškas, -a*「牧羊人的（陽性，陰性）」、*piemen-áuti*「當牧羊人」，plg. 具有短構詞詞幹的指小詞有 *dukt-ýtė*「小女兒」、*dukt-ùlė*「小女兒」、*dukt-ẽlė*「小女兒」、*dukt-ùkė*「小女兒」、*dukt-ùžė*「小女兒」，以及對現代語言來說不常見的構詞產物 *akm-uvà*「水中的石頭（罕用構詞）」、*akm-úotas, -a*「多石的（陽性，陰性）」[^fn58] (daugiau žr. Urbutis 2009, 224)。也存在構詞產物傾向於名詞單數或**複數**（daugiskaita）的情況，例如，依賴單數形式 *žmogùs*「人」的構詞產物有 *žmog-ẽlis*「小人物」、*žmog-i̇́ena*「人肉」、*žmog-ystà*「人（略帶貶義或特指）」、*žmog-žudỹs, -ė*「殺人犯（陽性，陰性）」、*žmõg-iškas, -a*「人性的（陽性，陰性）」、*žmog-ė́ti*「變得像人」、*žmõg-inti*「使具人性」；依賴複數 *žmónės*「人們」的構詞產物有 *žmon-ijà*「人類」、*žmón-iškas, -a*「人性的 / 人類的（陽性，陰性）」、*žmón-inti*「使成為人」。當詞基是形容詞時，基礎詞幹的選擇不會變化，因為在這種情況下通常所有形式的詞幹都是一樣的。雖然構詞產物通常不依賴形容詞**比較級（範疇）**（laipsnis）的屈折詞幹，但也存在帶有**比較級**（aukštesnysis laipsnis）或**最高級**（aukščiausiasis laipsnis）詞幹 *vyresn-*「較年長的」、*vyriaus-*「最年長的」的構詞產物，例如：*vyresn-ėlis, -ė*「長者 / 較年長的孩子（陽性，陰性）」、*vyresn-ýbė*「長官 / 上級」、*vyresn-ùmas*「年長 / 資歷」、*vyresn-iáuti*「居長 / 充當上級」[^fn59]。

有時作為構詞產物的構詞基礎的不是主格，而是另一個格，在這個格中發生了詞基**詞根**（šaknis）的**輔音交替**（priebalsių kaita）（plg. 5.2），例如，名詞 *jauč-iùkas*「小公牛」不是依賴主格 *jáutis*「公牛」，而是依賴詞根**輔音**（priebalsis）發生變化的格（*jáučio*「公牛（屬格）」, *jáučiui*「公牛（與格）」...）；**副詞**（prieveiksmis）*bli̇̀zganč-iai*「閃亮地」不是依賴主格 *bli̇̀zgantis, -i*「閃亮的（陽性，陰性）」，而是依賴**分詞**（dalyvis）**後綴**（priesaga）*-ant-* 具有**變體**（variantas） *-anč-* 的格（*bli̇̀zgančio*「閃亮的（陽性屬格）」, *bli̇̀zgančiam*「閃亮的（陽性與格）」...）。有時構詞產物依賴外來語名詞較短的詞幹。例如某些帶後綴的名詞和形容詞：*ãkc-ininkas, -ė*「股東（陽性，陰性）」；*avar-i̇̀ngas, -a*「事故多發的（陽性，陰性）」、*revoliuc-i̇̀ngas, -a*「革命的（陽性，陰性）」；*fonèt-iškas, -a*「語音的（陽性，陰性）」；*akadèm-inis, -ė*「學術的（陽性，陰性）」、*tèchn-inis, -ė*「技術的（陽性，陰性）」等。詞基 *ãkcija*「股份 / 行動」、*avãrija*「事故」、*revoliùcija*「革命」、*fonètika*「語音學」、*akadèmija*「學院」、*tèchnika*「技術」的後綴 *-ij-*、*-ik-* 不會進入到派生詞中。 

## 3.2. 動詞詞幹的選擇（Veiksmažodžio kamieno parinkimas）

**動詞**（veiksmažodis）具有三種詞幹形式。通常由動詞派生出的構詞產物依賴兩個屈折詞幹：**不定式**（bendratis）（通常在**構詞詞綴**（darybos afiksas）以輔音開頭時）（1）或**過去單次時**（būtasis kartinis laikas）（通常在構詞詞綴以**元音**（balsis）開頭時）（2）： 
1) *bū́-klė*「狀態」(: *bū́ti*「是 / 存在（不定式）」, plg. *bū̃na*「是（現在時）」, *bùvo*「是（過去時）」)、*judė́-sena*「動作 / 活動方式」(: *judė́ti*「動（不定式）」, plg. *jùda*「動（現在時）」, *judė́jo*「動（過去時）」)、*turė-klas*「扶手」(: *turė́ti*「有 / 握有（不定式）」, plg. *tùri*「有（現在時）」, *turė́jo*「有（過去時）」)、*kraty-tùvas*「搖動器 / 篩子」(: *kratýti*「搖動（不定式）」, plg. *krãto*「搖動（現在時）」, *krãtė*「搖動（過去時）」)、*važiuõ-klė*「底盤」(: *važiúoti*「乘車 / 行駛（不定式）」, plg. *važiúoja*「行駛（現在時）」, *važiãvo*「行駛（過去時）」)、*išlaiký-tinis, -ė*「受撫養者（陽性，陰性）」(: *išlaikýti*「撫養（不定式）」, plg. *išlai̇̃ko*「撫養（現在時）」, *išlai̇̃kė*「撫養（過去時）」)、*móky-tojas, -a*「教師（陽性，陰性）」(: *mókyti*「教（不定式）」, plg. *móko*「教（現在時）」, *mókė*「教（過去時）」)、*sužadė-tùvės*「訂婚」(: *susižadė́ti*「訂婚（不定式）」, plg. *susi̇̀žada*「訂婚（現在時）」, *susižadė́jo*「訂婚（過去時）」)[^fn60]；
2) *skyn-i̇̀mas*「採摘 / 砍伐的空地」(: *skýnė*「採摘（過去時）」, plg. *ski̇̀nti*「採摘（不定式）」, *ski̇̀na*「採摘（現在時）」)、*važiãv-imas*「行駛」(: *važiãvo*「行駛（過去時）」, plg. *važiúoti*「行駛（不定式）」, *važiúoja*「行駛（現在時）」)、*siùt-as*「狂怒」(: *siùto*「發怒（過去時）」, plg. *siùsti*「發怒（不定式）」, *siuñta*「發怒（現在時）」)、*rad-inỹs*「發現物」(: *rãdo*「找到（過去時）」, plg. *ràsti*「找到（不定式）」, *rañda*「找到（現在時）」)。 

有時構詞產物依賴動詞**現在時**（esamasis laikas）詞幹，例如：*ded-ẽklė*「產卵的母雞」(: *dẽda*「下蛋（現在時）」, plg. *dė́ti*「放 / 下蛋（不定式）」, *dė́jo*「放（過去時）」)、*duod-i̇̀ngas, -a*「容易給予的，慷慨的」(: *dúoda*「給（現在時）」, plg. *dúoti*「給（不定式）」, *dãvė*「給（過去時）」)、*ein-ùs, -i̇̀*「不在同一個住所或工作地點久留的」(: *ei̇̃na*「走（現在時）」, plg. *ei̇̃ti*「走（不定式）」, *ė̃jo*「走（過去時）」)、*blizg-ùtis*「亮片」(: *bli̇̀zga*「閃爍（現在時）」, plg. *blizgė́ti*「閃爍（不定式）」, *blizgė́jo*「閃爍（過去時）」)、*baršk-ùlis*「撥浪鼓 / 響板」(: *bárška*「嘎嘎響（現在時）」, plg. *barškė́ti*「嘎嘎響（不定式）」, *barškė́jo*「嘎嘎響（過去時）」)、*varvẽ-klis*「冰柱 / 滴下物」(: *var̃va*「滴下（現在時）」, plg. *varvė́ti*「滴下（不定式）」, *varvė́jo*「滴下（過去時）」)。

總體而言，考慮到源自動詞詞幹的構詞產物與動詞某一個詞幹或同時與幾個屈折詞幹的一致性，可以將基礎詞幹本質上一致的構詞產物分為以下幾組：
1) 與所有動詞詞幹（不定式、現在時和過去單次時）一致，例如：*lẽs-alas*「鳥食」(: *lèsti*「啄食（不定式）」, *lẽsa*「啄食（現在時）」, *lẽsė*「啄食（過去時）」)、*drož-i̇̀kas, -ė*「雕刻工（陽性，陰性）」(: *dróžti*「雕刻（不定式）」, *dróžia*「雕刻（現在時）」, *dróžė*「雕刻（過去時）」)、*suk-inỹs*「旋轉物 / 卷」(: *sùkti*「旋轉（不定式）」, *sùka*「旋轉（現在時）」, *sùko*「旋轉（過去時）」)、*rėk-snỹs, -ė*「愛大喊大叫的人（陽性，陰性）」(: *rėkti*「喊叫（不定式）」, *rėkia*「喊叫（現在時）」, *rėkė*「喊叫（過去時）」)、*neš-ė́jas, -a*「搬運工（陽性，陰性）」(: *nèšti*「拿 / 搬（不定式）」, *nẽša*「拿（現在時）」, *nẽšė*「拿（過去時）」)、*kep-snỹs*「烤肉」(: *kèpti*「烤（不定式）」, *kẽpa*「烤（現在時）」, *kẽpė*「烤（過去時）」)、*kaũk-smas*「嚎叫聲」(: *kaũkti*「嚎叫（不定式）」, *kaũkia*「嚎叫（現在時）」, *kaũkė*「嚎叫（過去時）」)； 
2) 與過去單次時詞幹一致，例如：*gėr-alas*「飲料」(: *gė́rė*「喝（過去時）」, plg. *gérti*「喝（不定式）」, *gẽria*「喝（現在時）」)、*šėr-i̇̀kas, -ė*「飼養員（陽性，陰性）」(: *šė́rė*「餵養（過去時）」, plg. *šérti*「餵養（不定式）」, *šẽria*「餵養（現在時）」)、*buv-i̇̀mas*「存在 / 待著」(: *bùvo*「是（過去時）」, plg. *bū́ti*「是（不定式）」, *yrà*「是（現在時）」)、*krit-ėsiai̇̃*「落下的東西 / 廢料」(: *kri̇̀to*「掉落（過去時）」, plg. *kri̇̀sti*「掉落（不定式）」, *kriñta*「掉落（現在時）」)、*put-muo͊*「腫塊 / 浮腫」(: *pùto*「膨脹（過去時）」, plg. *pùsti*「膨脹（不定式）」, *puñta*「膨脹（現在時）」)、*i̇̀šbrid-a*「泥濘難行之處 / 跋涉的痕跡」(: *išbri̇̀do*「涉水走出（過去時）」, plg. *išbri̇̀sti*「涉水走出（不定式）」, *i̇̀šbrenda*「涉水走出（現在時）」)、*ėm-ė́jas, -a*「拿取者 / 收受者（陽性，陰性）」(: *ėmė*「拿（過去時）」, plg. *im̃ti*「拿（不定式）」, *i̇̀ma*「拿（現在時）」)、*pardav-ė́jas, -a*「售貨員（陽性，陰性）」(: *par̃davė*「賣（過去時）」, plg. *pardúoti*「賣（不定式）」, *pardúoda*「賣（現在時）」)、*išdav-i̇̀kas, -ė*「叛徒（陽性，陰性）」(: *i̇̀šdavė*「出賣（過去時）」, plg. *išdúoti*「出賣（不定式）」, *išdúoda*「出賣（現在時）」)； 
3) 與不定式詞幹一致，例如：*duo-muõ*「數據 / 資料」(: *dúoti*「給（不定式）」, plg. *dúoda*「給（現在時）」, *dãvė*「給（過去時）」)、*griáu-medis*「蛀木蟲」(: *griáuti*「破壞（不定式）」, plg. *griáuna*「破壞（現在時）」, *grióvė*「破壞（過去時）」, *mẽdis*「木頭」)、*ry-klỹs*「鯊魚」(: *rýti*「吞咽（不定式）」, plg. *rỹja*「吞咽（現在時）」, *ri̇̀jo*「吞咽（過去時）」)、*ùžsė-lis*「播種過的地方」(: *užsė́ti*「播種（不定式）」, plg. *užsė́ja*「播種（現在時）」, *užsė́jo*「播種（過去時）」)、*bes-tùvas*「插具 / 錐子」(: *bèsti*「插（不定式）」, plg. *bẽda*「插（現在時）」, *bẽdė*「插（過去時）」)、*áuklė-tinis, -ė*「受教育者 / 學生（陽性，陰性）」(: *áuklėti*「教育（不定式）」, plg. *áuklėja*「教育（現在時）」, *áuklėjo*「教育（過去時）」)； 
4) 與現在時詞幹一致，例如：*iešk-i̇̀klis*「尋找器 / 探測器」(: *i̇́eško*「尋找（現在時）」, plg. *ieškóti*「尋找（不定式）」, *ieškójo*「尋找（過去時）」)、*bud-ỹnė*「守夜 / 守靈」(: *bùdi*「守衛（現在時）」, plg. *budė́ti*「守衛（不定式）」, *budė́jo*「守衛（過去時）」)、*nemók-ša*「無知的人」(: *nemóka*「不會（現在時）」, plg. *nemokė́ti*「不會（不定式）」, *nemokė́jo*「不會（過去時）」)、*varv-ulỹs*「流鼻涕的人 / 滴水」(: *var̃va*「滴下（現在時）」, plg. *varvė́ti*「滴下（不定式）」, *varvė́jo*「滴下（過去時）」)、*palyd-õvas, -ė*「伴隨者 / 衛星（陽性，陰性）」(: *palỹdi*「伴隨（現在時）」, plg. *palydė́ti*「伴隨（不定式）」, *palydė́jo*「伴隨（過去時）」)、*stri̇̀ks-ė*「蹦跳的女孩 / 母兔」(: *stri̇̀ksi*「蹦跳（現在時）」, plg. *striksė́ti*「蹦跳（不定式）」, *striksė́jo*「蹦跳（過去時）」)、*mieg-ãlius, -ė*「愛睡覺的人（陽性，陰性）」(: *miẽga*「睡覺（現在時）」, plg. *miegóti*「睡覺（不定式）」, *miegójo*「睡覺（過去時）」)； 
5) 與現在時和過去單次時詞幹一致，例如：*abejój-imas*「懷疑」(: *abejója*「懷疑（現在時）」, *abejójo*「懷疑（過去時）」, plg. *abejóti*「懷疑（不定式）」)、*ėd-alas*「飼料」(: *ė́da*「吃（動物）（現在時）」, *ė́dė*「吃（動物）（過去時）」, plg. *ė́sti*「吃（動物）（不定式）」)、*grand-inỹs*「鏈條」(: *grándo*「刮（現在時）」, *grándė*「刮（過去時）」, plg. *grándyti*「刮（不定式）」)、*liùdij-imas*「證明」(: *liùdija*「證明（現在時）」, *liùdijo*「證明（過去時）」, plg. *liùdyti*「證明（不定式）」)、*raš-éiva*「愛寫的人 / 劣等作家」(: *rãšo*「寫（現在時）」, *rãšė*「寫（過去時）」, plg. *rašýti*「寫（不定式）」)、*raš-muõ*「字母 / 書寫符號」(: *rãšo*「寫（現在時）」, *rãšė*「寫（過去時）」, plg. *rašýti*「寫（不定式）」)、*siuv-yklà*「裁縫店」(: *siùva*「縫（現在時）」, *siùvo*「縫（過去時）」, plg. *siū́ti*「縫（不定式）」)、*žarst-ẽklis*「火鉗 / 撥火棍」(: *žar̃sto*「撥動（現在時）」, *žar̃stė*「撥動（過去時）」, plg. *žarstýti*「撥動（不定式）」)； 
6) 與不定式和過去單次時詞幹一致，例如：*vi̇̀r-alas*「湯 / 煮好的食物」(: *vi̇̀rti*「煮（不定式）」, *vi̇̀rė*「煮（過去時）」, plg. *vérda*「煮（現在時）」)、*kirp-yklà*「理髮店」(: *kir̃pti*「剪（不定式）」, *kir̃po*「剪（過去時）」, plg. *ker̃pa*「剪（現在時）」)、*lik-ùtis*「剩餘物」(: *li̇̀kti*「留下（不定式）」, *li̇̀ko*「留下（過去時）」, plg. *liẽka*「留下（現在時）」)、*į̇́dub-a*「凹陷處」(: *įdùbti*「凹陷（不定式）」, *įdùbo*「凹陷（過去時）」, plg. *įdum̃ba*「凹陷（現在時）」)； 
7) 與不定式和現在時詞幹一致，例如：*ùžpil-as*「浸液 / 澆注料」(: *užpi̇̀lti*「倒在...上（不定式）」, *ùžpila*「倒在...上（現在時）」, plg. *užpýlė*「倒在...上（過去時）」)、*tri̇̀n-alas*「擦劑」(: *tri̇̀nti*「摩擦（不定式）」, *tri̇̀na*「摩擦（現在時）」, plg. *trýnė*「摩擦（過去時）」)、*šùn-kara*「絞死狗的人 / 惡棍」(: *šuñs*「狗（屬格）」, *šuõ*「狗（主格）」, *kárti*「吊（不定式）」, *kãria*「吊（現在時）」, plg. *kórė*「吊（過去時）」)。

3) *-(i)o-ti*、*-(i)oj-a*、*-(i)oj-o*：*žvej-ýba*「捕魚」 (: *žvej**ó**ti*「捕魚（不定式）」, *žvej**ó**ja*「捕魚（現在時）」, *žvej**ó**jo*「捕魚（過去時）」)、*saug-i̇̀klis*「保險絲 / 保護器」 (: *sáug**o**ti*「保護（不定式）」, *sáug**o**ja*「保護（現在時）」, *sáug**o**jo*「保護（過去時）」)、*klaj-ū̃nas*「流浪者」 (: *klaj**ó**ti*「流浪（不定式）」, *klaj**ó**ja*「流浪（現在時）」, *klaj**ó**jo*「流浪（過去時）」)、*núosij-os*「篩下的東西 / 穀殼」 (: *nusij**ó**ti*「篩下（不定式）」, *nusij**ó**ja*「篩下（現在時）」, *nusij**ó**jo*「篩下（過去時）」)[^fn65]、*kaital-ỹnė*「不斷更換 / 變化無常」 (: *kaitali**ó**ti*「不斷改變（不定式）」, *kaitali**ó**ja*「不斷改變（現在時）」, *kaitali**ó**jo*「不斷改變（過去時）」)；

## 3.3. 反身詞綴是否進入構詞產物（Sangrąžos afikso (ne)perėjimas į darinius）

通常，**反身詞綴**（sangrąžos afiksas）（又稱質詞、**小品詞**（dalelytė））[^fn69]會從原形動詞進入構詞產物中，例如：*apsiblaũs-ėlis, -ė*「陰鬱的人（陽性，陰性）」 (: *apsiblaũsti*「變得陰鬱（不定式）」)、*elg-i̇̀masis*「行為」 (: *el̃gtis*「表現（不定式）」)、*įsikūr-i̇̀mas*「安頓 / 建立」 (: *įsikū́rė*「安頓（過去時）」, *įsikùrti*「安頓（不定式）」)、*pasimãt-ymas*「約會 / 見面」 (: *pasimãto*「見面（現在時）」 / *pasimãtė*「見面（過去時）」, *pasimatýti*「見面（不定式）」)。有時構詞產物依賴**反身動詞**（sangrąžinis veiksmažodis），但反身詞綴被省略（沒有進入構詞產物），例如：*el̃g-sena*「行為方式」 (: *el̃gtis*「表現（不定式）」)、*įkur-tùvės*「喬遷之喜的慶祝或宴席」 (: *įsikùrti*「安頓（不定式）」)、*ãtžvilg-is*「方面 / 視角」 (: *atsižvel̃gti*「考慮（不定式）」)、*są́mok-slas*「陰謀」 (: *susimóko*「密謀（現在時）」 / *susimókė*「密謀（過去時）」, *susimókyti*「密謀（不定式）」)[^fn70]、*ilg-esỹs*「渴望 / 思念」 (: *il̃gisi*「想念（現在時）」, *ilgė́tis*「想念（不定式）」)、*pãraš-as*「簽名」 (: *pasirãšo*「簽名（現在時）」 / *pasirãšė*「簽名（過去時）」, *pasirašýti*「簽名（不定式）」)、*pagyr-ū̃nas*「吹牛者」 (: *pasigýrė*「吹牛（過去時）」, *pasigi̇̀rti*「吹牛（不定式）」)、*pãstang-os*「努力」 (: *pasisténgti*「努力（不定式）」)[^fn71]。有時候很難斷定構詞產物是依賴於反身動詞還是非反身動詞，因為從本質上講，它可能依賴於兩者，例如，派生詞 *pósūk-is*「（物體的）轉動、旋轉」，「（道路的）轉彎」可以與 *pasisùkti*「（自己）轉身」以及 *pasùkti*「轉動（某物）」形成**構詞對立**（darybos opozicija），因此，在標明詞基時，反身詞綴通常寫在括號中，即：*pósūk-is* (: *pa(si)sùkti*「轉動 / 轉身」)；**後綴派生**（priesaginė daryba）詞 *pérver-smas*「（發展、體制的）突然改變或被改變」既可以與 *pérversti*「推翻 / 翻轉」形成構詞對立，也可以與 *pérsiversti*「翻過來」形成對立，因此寫為 *pérver-smas* (: *pér(si)versti*「推翻 / 翻轉」)；其他類似情況還有：*ãpav-as*「穿在腳上的東西，鞋類」 (: *àp(si)avė*「穿鞋（過去時）」, *ap(si)aũti*「穿鞋（不定式）」)、*atram-à*「支撐物或被倚靠的物體，支柱」，「椅背」，（體育）「支撐」 (: *at(si)remti*「支撐 / 倚靠（不定式）」)、*ãtloš-as*「椅子、長凳等的靠背」 (: *at(si)lóšti*「向後靠（不定式）」)、*póslink-is*「位置的改變」，「顯著的變化」 (: *pa(si)sliñkti*「移動（不定式）」)、*núopeln-as*「功勞，贏得的東西」 (: *nu(si)pel̃no*「掙得（現在時）」 / *nu(si)pel̃nė*「掙得（過去時）」, *nu(si)pelnýti*「掙得（不定式）」)、*pašnek-õvas, -ė*「交談的參與者 / 談話對象（陽性，陰性）」 (: *pa(si̇̀)šneka*「交談（現在時）」, *pa(si)šnekė́ti*「交談（不定式）」)、*súokalb-is*「協議 / 勾結」 (LKŽe) (: *su(si̇̀)kalba*「商定（現在時）」, *su(si)kalbė́ti*「商定（不定式）」) (daugiau žr. Urbutis 2009: 231–234)[^fn72]。

## 3.4. 前綴的變化（Priešdėlių pakitimai）

後綴和**詞尾**（galūnė）派生詞通常由帶**前綴**（priešdėlis）的動詞構成。有時構詞產物中的前綴會發生改變，即前綴發生變異。必須認識到，這些是同一個前綴的變體，而不是不同的前綴[^fn73]，請比較構詞產物及其詞基的前綴[^fn74]：
1) *ap-* / *api-* / *apy-*：*apýsak-a*「中篇小說」 (: *apsãko*「講述（現在時）」 / *apsãkė*「講述（過去時）」, *apsakýti*「講述（不定式）」)、*apýbraiž-a*「特寫 / 輪廓」 (: *apibrė́žti*「劃定（不定式）」)[^fn75]； 
2) *at-* / *ata-* / *ati-* / *ato-*：*atãbrad-as*「淺灘」 (: *atbri̇̀do*「涉水而來（過去時）」, *atbri̇̀sti*「涉水而來（不定式）」)、*atãtvar-as*「圍欄」 (: *atitvérti*「隔開（不定式）」)、*atãtrank-a*「反衝 / 後坐力」 (: *atsitreñkti*「撞擊（不定式）」)[^fn76]、*atótrūk-is*「間隙 / 脫節」 (: *atitrū́kti*「斷開 / 脫離（不定式）」)； 
3) *į-* / *in-* / *im-*：*iñtak-as*「支流」 (: *į̇̃teka*「流入（現在時）」, *įtekė́ti*「流入（不定式）」)[^fn77]、*im̃pil-as*「裝枕頭或被褥羽毛的內袋 / 褥套」 (: *įpi̇̀lti*「倒入（不定式）」)；
