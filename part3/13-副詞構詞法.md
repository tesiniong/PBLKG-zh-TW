# 13. 副詞構詞法（PRIEVEIKSMIŲ DARYBA）

**副詞**（prieveiksmis）具有兩種主要的構詞法——後綴派生和前綴派生，也有以**混合構詞法**（mišrusis darybos būdas）構成的副詞（見表3）。多數副詞是已經或正在使用後綴構成的[^fn372]。詞尾派生對副詞並不典型，勉強可以談論個別副詞的複合[^fn373]。副詞的詞基通常是**形容詞**（būdvardis）或**形容詞化**（būdvardėjimas）的**分詞**（dalyvis），極少數是**名詞**（daiktavardis）。

## 13.1. 後綴派生詞（Priesagų vediniai）

後綴派生詞最常見的**構詞標誌**（formantas）是 *-(i)ai*。從意義的角度來看，帶有 *-(i)ai* 的副詞是異質的。它們通常具有品質意義，例如：*begė́dišk-ai*、*gõdž-iai*[^fn374]、*įdom-iai̇̃*、*miglót-ai*，但也有表示**比較級**（laipsnis）、**強度**（intensyvumas）等數量的副詞，例如：*kelerióp-ai*、*nepaprast-a*、*maž-a*、*menk-a*。某些 *-(i)ai* 派生詞具有時間、空間或地點意義，例如：*amžin-a*、*dažn-a*、*apýdažn-iai*、*grei̇̃t-ai*、*sen-iai̇̃*；*aukšt-ai̇̃*、*artutė̃l-iai*、*nuošal-iai̇̃*、*tolók-ai*、*žem-ai̇̃*。使用後綴 *-(i)ai*，多數副詞可以由**品質形容詞**（kokybinis būdvardis）構成（偶爾也由**關係形容詞**（santykinis būdvardis）構成），許多派生詞以**主動語態分詞**（veikiamosios rūšies dalyvis）和**被動語態分詞**（neveikiamosios rūšies dalyvis）為基礎[^fn375]。帶有 *-(i)ai* 的副詞由各種結構和各種來源的詞構成——既有**原生詞**（pirminiai žodžiai），也有構詞產物。後綴變體 *-ai* 通常用於由詞尾為 *-as* 的形容詞、被動語態分詞和**需要分詞**（reikiamybės dalyvis）構成的派生詞，而 *-iai* 則通常用於由詞尾為 *-ias*、*-is*、*-us* 的形容詞和主動語態分詞構成的副詞（見表13）。

許多副詞具有後綴 *-yn*。「這些副詞與動詞 *eti* 的形式連用，其意義的動態性可以說等同於帶有後綴 *-ė́ti* 的動詞」（DLKG 2005: 430）。所有後綴 *-yn* 的派生詞都表示增強、擴大的動作特徵。這些派生詞通常以可比較級的雙音節品質形容詞為基礎，例如：*aišk-ỹn* (: *áiškus, -i*)（比較 *aiškė́ti*），*ger-ỹn* (: *gẽras, -à*)，*minkšt-ỹn* (: *mnkštas, -à*)，*puik-ỹn* (: *puikùs, -*)，*šalt-ỹn* (: *šáltas, -à*)。也有由其他詞類構成的派生詞——副詞、動詞（分詞），個別由名詞構成，例如：*art-ỹn* (: *art*)，*daug-ỹn* (: *daũg*)，*tol-ỹn* (: *tol*)；*išdyk-ỹn* (: *išdỹkęs, -usi*)，*pasiut-ỹn* (: *pasiùtęs, -usi*)；*vasar-ỹn*「更加進入夏天」(: *vãsara*)，*niek-ỹn*「變差，變糟」(: *niẽkas*「微小、劣質的東西；不重要的事情；小事」)。

很大一部分帶有能產後綴 *-(i)om(is)* 的副詞，在構詞上與動詞相關聯。在立陶宛語中，經常使用這個後綴的多個派生詞變體（見第8章），例如：*apčiúop-omis* / *apčiúop-om* / *apčiuop-õm* (: *apčiúopti*)，*apgraib-omi̇̀s* / *apgraib-õm* (: *apgrai̇̃bo* / *apgrai̇̃bė*, *apgraibýti*)，*nemač-ioms* / *nemač-iõm* (: *nemãto* / *nemãtė*, *nematýti*)[^fn376]，*sėd-oms* / *sėd-õm* (: *sė́da* / *sė́do*, *sė́sti*)。「此外，也會遇到帶有擴展後綴 *-čiom(is)* 的派生詞（Keinys 1999: 100），例如：*bėg-čioms* / *bėg-čiõm* (: *bė́gti*)。

有些帶有 *-(i)om(is)* 的副詞是由形容詞、名詞構成的，少數是由副詞或數詞構成的，例如：*atbul-oms* / *atbul-õm* (: *ãtbulas, -à*)，*atatupst-oms* / *atatupst-õm* (: *atãtupstas, -a*)，*bas-oms* / *bas-õm* (: *bãsas, -à*)，*rait-oms* / *rait-õm* (: *ratas, -à*)，*tekin-oms* / *tekin-õm* (: *tẽkinas, -à*)；*juok-oms* / *juok-õm* (: *juõkas*)；*greit-oms* / *greit-õm* (: *grei̇̃tai*)；*ketur-ioms* / *ketur-iõm* (: *ketur, -ios*)。

標準語中有極少數副詞是使用後綴 *-(i)ais*、*-(i)a*、*-čia* 構成的，例如：*prešok-iais* (: *prišókti*)[^fn380]，*šnibžd-à* (: *šnbžda, šnibždė́ti*)，*šžarg-a* (: *išžegti*)[^fn381]，*vog-čià* (: *võgti*)。具有各種後綴或同一後綴的變體、且意義相同的副詞，是**構詞同義詞**（darybiniai sinonimai）或變體（見第8章），例如：*nejuč-ià* – *nejuč-ioms* / *nejuč-iõm*，*paslapč-ià* – *paslapč-ioms* / *paslapč-iõm*，*risč-ià* – *risč-ioms* / *risč-iõm*[^fn382]，*bėg-čià* – *bėg-čioms* / *bėg-čiõm*，*vog-čià* – *vog-čioms* / *vog-čiõm*。

使用後綴 *-(i)aip*、*-(i)ur*、*-ada*（較少使用變體 *-ados*、*-ad*）構成代詞性副詞，例如：*an-ap* (: *anàs, -à*)，*k-ap* (: *kàs*)，*kit-ap* (: *ktas, -à*)，*sav-ap* (: *savs*)，*š-iap* (: *šs, -*)，*št-aip* (: *štas, -à*)；*bet k-adà* (: *bet kàs*)，*kažk-adà* (: *kažkàs*)，*niek-adà* / *niek-adõs* / *niẽk-ad* (: *niẽkas*)；*kit-u* (: *ktas, -à*)，*niẽk-ur* (: *niẽkas*)，*vis-u* (: *vsas, -à*)，副詞 *svet-u*[^fn383] 以形容詞 *svẽčias, -ià*（「陌生的」）為基礎。「有一小批數詞性副詞是使用後綴 *-iese* 構成的」（Keinys 1999: 101），例如：*dv-ese* (: *dv, dù*)，*ketur-ese* (: *ketur, -ios*)，*penk-ese* (: *penk, -ios*)，副詞 *tr-isè* (: *trỹs*) 具有後綴 *-iese* 的變體 *-ise*。除了數詞性副詞，還有幾個由其他詞類（代詞、副詞）構成的派生詞，例如：*kel-ese* (: *kel, -ios*)，*netol-ese* (: *netol*)。

## 13.2. 前綴派生詞（Priešdėlių vediniai）

副詞中真正的前綴派生詞只有幾個，例如：*ùž-poryt* (: *porýt*)，*ùž-vakar* (: *vãkar*)，*po-rýt* (: *rýt*)，*pró-risčia* (: *risčià*)（LKE: 432）。關於由帶有後綴 *-(i)ai* 的副詞加上**前綴**（priešdėlis）*ne-*、*nebe-* 所構成的副詞構詞法，S. Keinys（1999, 101–102）的書中寫道：前綴 *ne-* 賦予與詞基相反的意義，而前綴 *nebe-* 的派生詞還表明特徵已發生改變，例如：*ne-jaunai̇̃* (: *jaunai̇̃*)，*ne-jausmi̇̀ngai* (: *jausmi̇̀ngai*)，*ne-mãtomai* (: *mãtomai*)，*ne-suñkiai* (: *suñkiai*)，*ne-tiesmuka* (: *tiesmuka*)，*ne-tiñkamai* (: *tiñkamai*)；*nebe-tiñkamai* (: *tiñkamai*)，*nebe-suprañtamai* (: *suprañtamai*)。但幾乎所有帶有前綴 *ne-*、*nebe-* 的副詞，也可以被視為由帶有這些前綴的詞基構成的後綴 *-(i)ai* 派生詞，也就是說，也可能存在其他的**構詞對立**（darybos opozicija），例如：*ne-jaun-a* (: *nejáunas, -à*)，*nemãtom-ai* (: *nemãtomas, -a*)，*nebesuprañtam-ai* (: *nebesuprañtamas, -a*) 等。因此，基本上帶有前綴 *ne-*、*nebe-* 的副詞，既可以視為前綴派生詞，也可以視為後綴派生詞：*ne-drsiai* (: *drsiai*) 或 *nedrs-iai* (: *nedrąsùs, -*)，*ne-saũgiai* (: *saũgiai*) 或 *nesaũg-iai* (: *nesaugùs, -*)[^fn384]。

## 13.3. 混合構詞法的副詞（Mišriosios darybos prieveiksmiai）

許多副詞是同時使用前綴和後綴構成的。這些構詞產物通常帶有後綴 *-(i)ui*，較少帶有後綴 *-(i)ais*、*-(i)ai*。在前綴中，*pa-* 很常見，但也有帶有其他前綴（*apy-*、*pra-* 等）的構詞產物，例如：*pa-dien-iu* (: *dienà*)，*pa-kel-iu* (: *kẽlias*)，*pa-plauk-iu* (: *pláukas*)，*pa-vėj-u* (: *vė́jas*)，*pa-punkč-iu* (: *pùnktas*)，*pra-meč-iu* (: *mẽtai*)，*pra-nakč-iu* (: *nãkčiai, nakts*)，*pa-žodž-iu* (: *žõdžio, žõdis*)[^fn385]；*į-saul-iu* (: *sáulė*)，*pa-kiem-ias* (: *kiẽmas*)，*pró-rėksm-(i)ais* (: *rėksmas*)，*pró-verksm-iais* (: *veksmas*)；*apý-daug-iai* (: *daũg*)，*pó-daug-iai* (: *daũg*)；*pra-maiš-iu* (: *mašo* / *mašė*, *maišýti*)[^fn386]。
