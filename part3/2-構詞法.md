# 2. 構詞法（ŽODŽIŲ DARYBOS BŪDAI）

詞彙的**構詞法**（žodžių darybos būdas）是**語素**（morfema）構成詞彙的一種形式變體，由**構詞標誌**（formantas）的性質決定。構詞法首先分為**派生**（vedyba / derivacija）與**複合**（dūryba / kompozicija）：**派生**是由單一詞彙加上**詞綴**（afiksas）（**前綴**（priešdėlis）、**後綴**（priesaga）與**詞尾**（galūnė））構成詞彙的方式，而**複合**是由兩個**詞基**（pamatinis žodis）構成詞彙的方式。立陶宛語中的**派生詞**（vedinys）或**複合詞**（dūrinys）可透過四種主要的構詞方式構成——分為**後綴派生**（priesaginė daryba / priesagų daryba / sufiksacija）（2.1）、**前綴派生**（priešdėlinė daryba / priešdėlių daryba / prefiksacija）（2.2）、**詞尾派生**（galūninė daryba / galūnių daryba / paradigm(iz)acija / fleksinė derivacija）（2.3）與**複合**（dūryba / kompozicija / sandūra / sudurtinė daryba）[^fn39]（2.4）。此外還有**混合構詞法**（mišrusis darybos būdas / parasintezė），即同時出現兩種構詞方式，通常是前綴與後綴派生（prefiksacija-sufiksacija）以及複合與後綴派生（kompozicija-sufiksacija）（2.5）（Keinys 1999: 22–23, 82–83, 102, 114; Urbutis 2009: 333–342; plg. LKE: 121; KTŽ: 52, 143, 155, 176, 196; Ružė 2008: 12–15; Murmulaitytė, Aleksaitė 2021）。值得一提的是，學界對構詞法也有其他不同觀點（Paulauskienė 1994: 71–97; 2007: 20–22），例如在具代表性的構詞法著作中（包含本語法書），像 *be-dãlis, -ė*「無份的」、*pa-griovỹs*「溝邊」、*pó-dirvis*「底土」、*ùž-ribis*「界外」等詞被視為前綴的派生詞，但在 A. Paulauskienė 的形態學著作（1994: 92–97）中則被視為複合詞。A. Smetona (2005)[^fn40] 也抱持類似觀點。關於**反身動詞**（sangrąžiniai veiksmažodžiai）的看法也不一致。儘管通常認為**動詞**（veiksmažodis）具有兩種構詞法——前綴派生與後綴派生[^fn41]，但有些文獻[^fn42]還會提到第三種——反身標記——動詞構詞法。在本語法書中，反身標記不被視為獨立的構詞手段，而是被當作附加的（次要的）構詞手段（plg. 3.3, 4.1, 14.3）[^fn43]。所有構詞方式的派生詞與複合詞都具備相同的語素音韻現象（**詞根**（šaknis）**元音**（balsis）與**輔音**（priebalsis）交替、**換調**（metatonija）等[^fn44]）。各個**詞類**（kalbos dalys）（**名詞**（daiktavardis）、**形容詞**（būdvardis）、動詞、**副詞**（prieveiksmis）、**數詞**（skaitvardis）、**代詞**（įvardis））常見的主要構詞方式範例詳見表3。

## 2.1. 後綴派生（Priesaginė daryba / sufiksacija）

有非常大量的派生詞是透過各種後綴構成的。這種構詞法適用於所有詞類，尤其是名詞、形容詞、動詞的後綴派生最為發達。後綴可以是唯一的構詞手段，例如：*nauj-aĩ*「新地」 (: *naũjas, -à*「新的」)，*suman-iaĩ*「聰明地」 (: *sumanùs, -ì*「聰明的」)；但大多數情況下，它會與詞尾（或是動詞**不定式**（bendratis）後綴）一起出現（plg. 4.1）。儘管如此，這類詞語仍被視為後綴派生的派生詞，因為決定構詞法的主要手段是後綴，例如：*ger-ùmas*「善良」 (: *gẽras, -à*「好的」)，*maĩšt-ininkas, -ė*「叛亂者」 (: *maĩštas*「叛亂」)，*puš-ýnas*「松樹林」 (: *pušìs*「松樹」)，*med-ìnis, -ė*「木製的」 (: *mẽdis*「樹木」)，*suk-inė́ti*「反覆轉動」 (: *sùkti*「轉動」)，*juod-úoti*「發黑」 (: *júodas*「黑色的」)[^fn47]。

## 2.2. 前綴派生（Priešdėlinė daryba / prefiksacija）

這種構詞法常見於動詞、形容詞、名詞、副詞：加上前綴構成的動詞與形容詞非常多，名詞則少得多，只有極少數的前綴派生副詞，而數詞與代詞完全不使用此法構成。前綴可以是派生詞唯一的構詞手段，例如：*ap-rašýti*「寫下」 (: *rašýti*「寫」)，*pa-nèšti*「搬一下」 (: *nèšti*「搬」)，*ne-gẽras, -à*「不好的」 (: *gẽras, -à*「好的」)，*nebe-švarùs, -ì*「不再乾淨的」 (: *švarùs, -ì*「乾淨的」)，*tebe-sūrùs, -ì*「仍然鹹的」 (: *sūrùs, -ì*「鹹的」)，*į́-brolis*「繼兄弟」 (: *brólis*「兄弟」)，*ne-láimė*「不幸」 (: *láimė*「幸福」)，*už-porýt*「大後天」 (: *porýt*「後天」)；但也經常與次要構詞手段（即詞尾）一起出現，例如：*be-nãmis, -ė*「無家可歸者」 (: *namaĩ*「家」)，*pa-tė́vis*「繼父」 (: *tė́vas*「父親」)，*pa-meistrỹs, -ė̃*「副手」 (: *méistras, -ė*「師傅」)，*pa-vakarė̃*「傍晚」 (: *vãkaras*「晚上」)，*apý-geris, -ė*「相當好的」 (: *gẽras, -à*「好的」)，*apý-kreivis, -ė*「有點歪的」 (: *kreĩvas, -à*「彎曲的」)，*be-baĩmis, -ė*「無畏的」 (: *báimė*「恐懼」)，*pó-sausis, -ė*「稍乾的」 (: *saũsas, -à*「乾燥的」)，*pó-rūgštis, -ė*「稍酸的」 (: *rūgštùs, -ì*「酸的」)，*príe-kurtis, -ė*「微聾的」 (: *kur̃čias, -à*「聾的」)[^fn48]，*pró-retis, -ė*「稍稀疏的」 (: *rẽtas, -à*「稀疏的」)[^fn49]（關於單純與複合的構詞標誌詳見 4.1）。在標準語中，帶有常見前綴 *apy-* 與 *po-* 的形容詞會採用 *-is*, *-ė* 詞尾（這類派生詞不保留原本作為詞基的形容詞詞尾），例如前綴 *apy-* 的派生詞 *apý-aukštis, -ė*「相當高的」與其詞基 *áukštas, -à*「高的」詞尾不同；前綴 *po-* 的派生詞 *pó-bjauris, -ė*「有點糟糕的」與其詞基 *bjaurùs, -ì*「糟糕的」詞尾亦不同。其他例子：*apý-baltis, -ė*「略白的」, *apý-bjauris, -ė*「略醜的」, *apý-blogis, -ė*「略壞的」, *apý-brangis, -ė*「稍貴的」, *apý-bukis, -ė*「略鈍的」, *apý-dailis, -ė*「稍美的」, *apý-darbštis, -ė*「稍勤勞的」, *apý-dažnis, -ė*「稍頻繁的」, *apý-didis, -ė*「稍大的」... *pó-bukis, -ė*「略鈍的」, *pó-didis, -ė*「稍大的」, *pó-geris, -ė*「略好的」, *pó-gilis, -ė*「略深的」, *pó-ilgis, -ė*「略長的」, *pó-kvailis, -ė*「略傻的」, *pó-mažis, -ė*「略小的」, *pó-minkštis, -ė*「略軟的」, *pó-piktis, -ė*「稍怒的」... 在方言中，前綴 *po-* 還有帶有其他詞尾的變體——即保留其形容詞詞基之詞尾的派生詞，例如：*pó-karštas, -a*「有點熱的」, *pó-brangus, -i*「有點貴的」（更多範例請見 LKŽe）。值得注意的是，由前綴 *apy-* 與 *po-* 構成且表示不完全具備某屬性的同根形容詞常常是同義詞（關於派生同義詞與變體詳見第8章）。

## 2.3. 詞尾派生（Galūninė daryba / paradigm(iz)acija / fleksinė derivacija）

透過詞尾構成的詞包含許多名詞、部分形容詞以及極少數的數詞與代詞，但此構詞法不構成副詞與動詞（見表3），例如：*bríed-ė*「母鹿」 (: *bríedis*「公鹿」)，*jaũn-is, -ė*「年輕人」 (: *jáunas, -à*「年輕的」)，*krat-à*「搜查」 (: *krãto* / *krãtė*「搜查（現/過）」, *kratýti*「搜查」)，*trauk-à*「吸引力」 (: *tráukti*「吸引」)，*zìrz-a*「愛嘟囔的人」 (: *zir̃zti*「嘟囔」)，*dar-ùs, -ì*「能幹的」 (: *dãro* / *dãrė*「做（現/過）」, *darýti*「做」)，*ùždar-as, -à*「封閉的」 (: *uždãro* / *uždãrė*「關閉（現/過）」, *uždarýti*「關閉」)，*túkstant-as, -a*「第一千」 (: *túkstantis*「一千」)，*keliasdešim̃t-as, -à*「第幾十」 (: *kẽliasdešimt*「幾十」)。在生成詞尾派生詞時，經常會發生詞根元音交替（相關說明見5.1），例如：*į́kait-as, -ė*「人質」 (: *į̃keitė*「抵押（過）」, *įkeĩsti*「抵押」)，*krūv-à*「堆」 (: *króvė*「堆積（過）」, *kráuti*「堆積」)，*skõn-is*「味道」 (: *skanùs, -ì*「好吃的」)，*stỹg-ius*「缺乏」 (: *stìgti*「缺乏」)，*išlaid-ùs, -ì*「浪費的」 (: *išléido*「花費（過）」, *išléisti*「花費」)，*talp-ùs, -ì*「容量大的」 (: *til̃pti*「容納」)[^fn50]。

## 2.4. 複合（Dūryba / kompozicija / sandūra）

複合本身構成了一種不需再細分的構詞法。這是詞彙形態學的主要構詞方式之一。它可用於構成名詞、形容詞以及部分數詞與代詞（見表3），但不構成動詞與副詞。文法書中通常指出，複合詞大多以結構簡單的詞基為基礎，但對新詞的研究顯示，也有以結構相當複雜的詞基（即派生詞本身）為基礎的趨勢（Murmulaitytė 2021: 82–87; plg. Aleksaitė 2022: 107–141, 149–156）。複合詞的詞尾有時會與充當第二組成部分的基礎詞之詞尾一致，例如：*bendr-a-keleĩvis, -ė*「同路人」 (: *beñdras, -à*「共同的」, *keleĩvis, -ė*「旅客」)，*rug-ia-gėlė*「矢車菊」 (: *rugỹs*「黑麥」, *gėlė̃*「花」)[^fn51]。但大多數情況下，複合詞獲得的詞尾與充當第二組成部分的詞尾<u>不一致</u>[^fn52]，比較以形容詞 *beñdras, -à*「共同的」為首組成部分、而第二組成部分為不同詞基的複合詞詞尾：*bendr-a-áutoris, -ė*「合著者」 (但詞基為 *áutorius, -ė*「作者」)；*bendr-a-butis*「宿舍」 (但詞基為 *bù-tas*「公寓」)；*bendr-a-dar̃bis, -ė*「同事」 (但詞基為 *dárbas*「工作」)；*bendr-a-keliõnis, -ė*「旅伴」 (但詞基為 *keliõnė*「旅行」)；*bendr-a-klãsis, -ė*「同班同學」 (但詞基為 *klãsė*「班級」)；*bendr-a-mõkslis, -ė*「同學」 (但詞基為 *mókslas*「學習」)；*bendr-a-núomininkas, -ė*「合租人」 (但詞基為 *núomininkas, -ė*「租客」)；*bendr-a-nuõmis, -ė*「合租房屋」 (但詞基為 *núoma*「租賃」)。在複合詞的兩個詞幹之間通常可以分出一個元音成分，稱為**連接元音**（jungiamasis balsis）[^fn53]。因此，複合詞的組成部分可以直接複合而無連接元音（1），或是使用連接元音（2）；立陶宛語中已知的連接元音包含：*-a-* / *-ia-*, *-i-*, *-o-* / *-io-*, *-u-* / *-iu-*, *-ė-*, *-y-*, *-ū-* / *-iū-*, *-uo-* (Urbutis 2009: 280, plg. Keinys 1999: 73, 81)[^fn54]，例如：
1) *keliõn-pinigiai*「差旅費」 (: *keliõnė*「旅行」, *pinigaĩ*「錢」)，*kãt-pėdė*「貓爪草」 (: *katė*「貓」, *pėdà*「腳」)，*sén-daiktis*「古董」 (: *sẽnas, -à*「老的」, *dáiktas*「物品」)，*del̃s-pinigiai*「滯納金」 (: *del̃sti*「拖延」, *pinigaĩ*「錢」)，*šùn-mazgis*「死結」 (: *šuñs, šuõ*「狗」, *mãzgas*「結」)，*gelton-mar̃gis, -ė*「黃花斑的」 (: *geltonaĩ*「黃色」, *márgas, -à*「花斑的」)，*sald-rū̃gštis, -ė*「酸甜的」 (: *saldùs, -ì*「甜的」, *rūgštùs, -ì*「酸的」)，*dúon-riekis, -ė*「切麵包刀」 (: *dúona*「麵包」, *riẽkti*「切片」)，*vìs-žalis, -ė*「常綠的」 (: *vìsas, -à*「全部的」, *žãlias, -ià*「綠的」)；
2) *krauj-ã-gyslė*「血管」 (: *kraũjas*「血」, *gýsla*「靜脈」)，*sidabr-ã-žolė*「委陵菜」 (: *sidãbras*「銀」, *žolė̃*「草」)，*ilg-a-rañkis, -ė*「長臂的」 (: *ìlgas, -à*「長的」, *rankà*「手」)，*pil-iã-kalnis*「山丘堡壘」 (: *pilìs*「城堡」, *kálnas*「山」)，*daug-ia-am̃žis, -ė*「長壽的」 (: *daũg*「多」, *ámžius*「世紀/年齡」)，*av-ì-kailis*「羊皮」 (: *avìs*「羊」, *káilis*「皮毛」)，*nakt-ì-piečiai*「夜宵」 (: *naktìs*「夜」, *piẽtūs*「午餐」)[^fn55]，*viet-ó-vardis*「地名」 (: *vietà*「地方」, *var̃das*「名字」)，*virš-ù-kalnė*「山頂」 (: *viršùs*「頂端」, *kálnas*「山」)，*saul-ė́-tekis*「日出」 (: *sáulė*「太陽」, *tẽka*「升起（現）」, *tekė́ti*「升起」)，*darb-ý-metis*「農忙季節」 (: *dárbas*「工作」, *mẽtas*「時間」)，*koj-ū́-galis*「腳端」 (: *kóója*「腳」, *gãlas*「末端」)，*valdž-ió-troška*「權力慾」 (: *valdžià*「權力」, *tróško*「渴望（過）」, *trókšti*「渴望」)。

從例子中可以看出，無論是否有連接元音，都能構成相同構詞類型（相關說明見 6.3）的複合詞（名詞 + 名詞、形容詞 + 名詞、名詞 + 動詞等）。有時由相同的詞基構成的複合詞，會因連接元音不同而產生不同的詞彙意義，例如：*dien-ó-raštis*「日記」（寫在特定書本或筆記本上的日常事務或印象紀錄）與 *diẽn-raštis*「日報」（（幾乎）每天發行的報紙）；但許多時候，由相同詞基構成的複合詞會具有非常相似（或相同）的詞彙意義，成為構詞的變體形式（關於變體見第8章），例如：*ak-ì-mojis* / *ak-ý-mojis* / *ak-iū́-mojis*「瞬間」，*basl-ì-galis* / *basl-iù-galis*「木樁頭」，*vasar-ó-šiltis, -ė* / *vasár-šiltis, -ė*「如夏天般溫暖的」[^fn56]。

## 2.5. 混合構詞法（Mišrioji daryba）

混合構詞法的派生詞可由前綴與後綴構成，例如：*į́-pėd-inis, -ė*「繼承人」 (: *pėdà*「腳」)，*prieš-šveñt-inis, -ė*「節前的」 (: *šveñtė*「節日」)，*pra-nakč-iuĩ*「過夜」 (: *naktìs*「夜」)；或是透過複合並加上後綴構成，例如：*pel-ė́-kau-tai*「捕鼠器」 (: *pelė̃*「老鼠」, *káuti*「殺死」)，*žmog-ė́d-**ra***「食人族」 (: *žmogùs*「人」, *ė́da* / *ė́dė*「吃（現/過）」, *ė́sti*「吃（動物）」)[^fn57]。雖然文獻通常寫道，混合構詞法在構詞中的作用相對較小（Keinys 1999: 23; LKE: 121; Urbutis 2009: 341），但立陶宛語中這類派生詞的數量正在增加（新詞研究也顯示了這點）。此外，有時會針對詞尾的作用提出質疑，進而探討混合構詞法是否可能構成如 a) *apýrankė*「手鐲」，*beuodẽgis, -ė*「無尾的」，*nuoširdùs, -i*「真誠的」；b) *briedkriaũnis, -ė*「具鹿角柄的」，*penkialãpis, -ė*「五葉的」這類派生詞。V. Urbutis (2009: 341–342) 認為，本質上在這裡也可以看出混合構詞法。儘管如此，在文法書、S. Keinis (1999) 的著作等文獻中，這類詞語仍分別被視為前綴派生詞（*apý-rankė* (: *rankà*「手」)，*be-uodẽgis, -ė* (: *uodegà*「尾巴」)，*nuo-širdùs, -i* (: *širdìs*「心」)）以及複合詞（*bried-kriaũnis, -ė* (: *bríedis*「鹿」, *kriaũnos*「刀柄」)，*penk-ia-lãpis, -ė* (: *penkì, -ios*「五」, *lãpas*「葉子」)）。本語法書亦採取此一立場。
