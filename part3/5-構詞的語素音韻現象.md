# 5. 構詞的語素音韻現象（DARYBOS MORFONOLOGINIAI REIŠKINIAI）

## 5.1. 詞根元音交替（Šaknies balsių kaita）

比較具有直接構詞關係的詞彙（**詞基**（pamatinis žodis）和**派生詞**（darinys））可以看出，派生詞和詞基的**詞根**（šaknis）**元音**（balsis）通常有所不同[^fn99]。**元音交替**（balsių kaita）可能出現在所有**詞類**（kalbos dalys）的**構詞法**（žodžių daryba）過程中。元音交替的多樣性很大，但其中有些交替較常見，有些較罕見：
1) *i* : *y*，例如 *ryšulỹs*「包裹」(: *ri̇̀šti*「綁」)[^fn100]，*ryšỹs*「聯繫」(: *ri̇̀šti*「綁」)，*gỹmis*「本性」(: *gi̇̀mti*「出生」)，*skrỹdis*「飛行」(: *skri̇̀do*「飛(過去時)」, *skri̇̀sti*「飛」)，*gylė*「深處」(: *gilùs, -i̇̀*「深的」)；*y* : *i*，例如 *septiñtas, -à*「第七」(: *septyni̇̀, -ios*「七」)；
2) *i* : *ie*，例如 *liẽptas*「小橋」(: *li̇̀pti*「踏/爬」)，*sniẽgas*「雪」(: *sni̇̀gti*「下雪」)；*ie* : *i*，例如 *žvigdýti*「使尖叫」(: *žviẽgti*「尖叫」)，*nemigà*「失眠」(: *nemiẽga*「不睡覺」, *nemiegóti*「不睡覺」)； 
3) *i* : *ė*，例如 *kėbė*「互相勾住的堆積物」(LKŽe) (: *ki̇̀bti*「勾住」)；*ė* : *i*，例如 *suri̇̀kti*「大叫出聲」(: *rėkti*「叫喊」)，*rikdýti*「使叫喊」(: *rėkti*「叫喊」)；
4) *i* : *e*，例如 *nértėti*「發怒」(: *nir̃to*「發怒(過去時)」, *nir̃sti*「發怒」；見 *niršti*「發怒」)；*e* : *i*，例如 *vi̇̀rkdyti*「使哭泣」(: *ver̃kti*「哭泣」)，*kilnóti*「舉起」(: *kélti*「舉起」)，*gi̇̀rdyti*「給...喝水」(: *gérti*「喝」)；
5) *i* : *ei*，例如 *bréizgėti*「磨損/散開」(LKŽe) (: *bri̇̀gzti*「磨損」)，*géižėti*「變酸」(LKŽe) (: *gi̇̀žti*「變酸」)；
6) *i* : *a*，例如 *maršùs, -i̇̀*「健忘的」(: *mir̃šti*「忘記」)，*dárbas*「工作」(: *di̇̀rbti*「工作(動詞)」)，*ardýti*「拆解」(: *i̇̀rti*「散開」)；*a* : *i*，例如 *mi̇̀ltai*「麵粉」(: *málti*「磨碎」)；
7) *i* : *ai*，例如 *káišioti*「到處塞/插」(: *ki̇̀šti*「塞」)，*láipioti*「到處爬」(: *li̇̀pti*「爬」)；
8) *i* : *o*，例如 *lomà*「低窪地」(: *li̇̀mti*「彎曲/傾斜」)； 
9) *i* : *u*，例如 *sluñkius, -ė*「懶漢」(: *sliñkti*「潛行」)； 
10) *y* : *ie*，例如 *žiẽdas*「花朵/戒指」(: *žýdi*「開花(現在時)」, *žydė́ti*「開花」)，*lietùs*「雨」(: *lýti*「下雨」)；*ie* : *y*，例如 *svỹdis*「投擲；投擲距離」(: *sviédė*「投擲(過去時)」, *sviésti*「投擲」)，*dažãsvydis*「漆彈」(: *dažai̇̃*「顏料」, *sviédė*「投擲(過去時)」, *sviésti*「投擲」)，*są́lytis*「接觸」(: *su(si̇̀)lietė*「互相接觸(過去時)」, *su(si)liésti*「互相接觸」)； 
11) *y* : *ei*，例如 *pléišėti*「裂開」(: *plýšti*「撕裂」)；*ei* : *y*，例如 *saulė́lydis*「日落」(: *sáulė*「太陽」, *léidosi*「落下(過去時)」, *léistis*「落下」)；
12) *y* : *ai*，例如 *klaidà*「錯誤」(: *klýdo*「犯錯/迷路(過去時)」, *klýsti*「犯錯/迷路」)，*kláidžioti*「遊蕩/迷路」(: *klýdo*「犯錯/迷路(過去時)」, *klýsti*「犯錯/迷路」)，*klaidi̇̀nti*「誤導」(: *klýdo*「犯錯/迷路(過去時)」, *klýsti*「犯錯/迷路」)； 
13) *ie* : *ai*，例如 *pašaipà*「嘲笑」(: *pašiẽpti*「嘲笑」)，*graibýti*「抓取」(: *griẽbti*「抓」)，*vaiši̇̀nti*「招待」(: *viẽši*「作客(現在時)」, *viešė́ti*「作客」)，*raidà*「發展」(: *riẽda*「滾動/流逝(現在時)」, *riedė́ti*「滾動/流逝(指時間或年份)」)；
14) *ė* : *y*，例如 *rýkauti*「時而叫喊」(: *rėkti*「叫喊」)，*slypė́ti*「隱藏(狀態)」(: *slėpti*「隱藏(動作)」)； 
15) *ė* : *e*，例如 *rẽplės*「鉗子」(: *rėpti*「抓住」)，*e* : *ė*，例如 *tėkmė*「水流」(: *tẽka*「流(現在時)」, *tekė́ti*「流」)，*núotėkis*「流失/洩漏」(: *nùteka*「流走(現在時)」, *nutekė́ti*「流走」)；
16) *ė* : *a*，例如 *sajùs, -i̇̀*「豐收的(播種)」(LKŽe) (: *sė́ja*「播種(現在時)」 / *sė́jo*「播種(過去時)」, *sė́ti*「播種」)，*šlakstýti*「灑水」(: *šlėkti*「灑水」)；
17) *ė* : *ai*，例如 *bráižyti*「畫/刮」(: *brė́žti*「畫線/刮」)，*brai̇̃žas*「筆跡/特徵」(: *brė́žti*「畫線/刮」)，*apýbraiža*「草圖/特寫」(: *apibrė́žti*「勾勒」)；
18) *ė* : *o*，例如 *sodi̇̀nti*「種植/使坐下」(: *sė́sti*「坐下」)，*sóstas*「寶座」(: *sė́sti*「坐下」)；
19) *ė* : *u*，例如 *sriubà*「湯」(: *srėbti*「喝湯」)[^fn101]；
20) *ė* : *ū*，例如 *sriū́bčioti*「小口喝」(: *srėbti*「喝湯」)；
21) *ė* : *uo*，例如 *úodas*「蚊子」(: *ė́da*「吃(現在時)」 / *ė́dė*「吃(過去時)」, *ė́sti*「吃」)；
22) *e* : *a*，例如 *sagà*「鈕扣」(: *sègti*「扣上」)，*sagti̇̀s*「扣環」(: *sègti*「扣上」)，*naštà*「負擔」(: *nèšti*「攜帶」)，*var̃tai*「大門」(: *vérti*「開關大門」)，*var̃žtas*「螺栓」(: *ver̃žti*「鎖緊」)，*pašalpà*「津貼」(: *pašel̃pti*「救濟」)，*pastabà*「評論/注意」(: *pàstebi*「注意到(現在時)」, *pastebė́ti*「注意到」)，*pažangà*「進步」(: *pažeñgti*「前進」)，*pãšaras*「飼料」(: *pašérti*「餵食」)，*talkà*「集體勞動」(: *tel̃kti*「召集」)，*trąšà*「肥料」(: *trę̃šti*「施肥」)，*lañkas*「弓/環」(: *leñkti*「彎曲」)，*nãras*「潛水員」(: *nérti*「潛水」)，*vãdas*「指揮官/首領」(: *vẽda*「帶領(現在時)」 / *vẽdė*「帶領(過去時)」, *vèsti*「帶領」)，*garbė*「榮譽」(: *ger̃bti*「尊重」)；
23) *ei* : *ai*，例如 *pãkaitalas*「替代品」(: *pàkeitė*「更換(過去時)」, *pakei̇̃sti*「更換」)，*lai̇̃das*「電線」(: *léido*「允許/放下(過去時)」, *léisti*「允許/放下」)，*sai̇̃kas*「測量/適度」(: *séikėti*「測量」)[^fn102]，*išlaidùs, -i̇̀*「揮霍的」(: *išléido*「花費(過去時)」, *išléisti*「花費/發行」)；
24) *a* : *o*，例如 *nakti̇̀gonė*「夜間放牧」(: *nakti̇̀s*「夜晚」, *gãno*「放牧(現在時)」 / *gãnė*「放牧(過去時)」, *ganýti*「放牧」)，*grõžis*「美麗(名詞)」(: *gražùs, -i̇̀*「美麗的」)，*grožýbė*「美貌/美景」(: *gražùs, -i̇̀*「美麗的」)，*pašolỹs*「寒意」(: *pašálti*「結冰」)，*į̇́protis*「習慣」(: *įprãto*「習慣(過去時)」, *įpràsti*「習慣」)，*i̇̀šmonė*「虛構/巧思」(: *išmãno*「理解(現在時)」 / *išmãnė*「理解(過去時)」, *išmanýti*「理解」)；*o* : *a*，例如 *skabýti*「採摘」(: *skõbti*「挖/拔」)； 
25) *ai* : *ui*，例如 *rui̇̃šis*「綁帶」(: *rai̇̃šti*「綁」)[^fn103]； 
26) *au* : *u*，例如 *niekdžiugỹs, -ė̃*「幸災樂禍者」(: *niekai̇̃*「小事/無價值之物」, *džiaũgtis*「高興」)，*pjudýti*「挑唆」(: *pjáutis*「互相撕咬/爭吵」)；*u* : *au*，例如 *kiáuras, -à*「有洞的」(: *kiùro*「破洞(過去時)」, *kiùrti*「破洞」)； 
27) *au* : *ū*，例如 *šū̃kis*「口號/呼聲」(: *šaũkti*「呼喊」)，*pjūti̇̀s*「收割」(: *pjáuti*「割/切」)，*spūsti̇̀s*「擁擠」(: *spáustis*「擠壓」)；*ū* : *au*，例如 *sraunus, -i*「水流湍急的」(: *srū́ti*「流動」)； 
28) *o* : *ū*，例如 *smū̃gis*「打擊」(: *smõgti*「打擊」)，*krūvà*「堆」(: *króvė*「堆積(過去時)」, *kráuti*「堆積/裝載」)；*ū* : *o*，例如 *džiovà*「結核病/乾燥」(: *džiū́vo*「變乾(過去時)」, *džiū́ti*「變乾」)；
29) *o* : *uo*，例如 *klúonas*「穀倉」(: *klóti*「鋪設」)；*uo* : *o*，例如 *dosnùs, -i̇̀*「慷慨的」(: *dúoti*「給予」)； 
30) *u* : *ū*，例如 *mū̃šis*「戰鬥」(: *mùšti(s)*「打擊/打鬥」)，*pósūkis*「轉彎/轉折」(: *pa(si)sùkti*「轉向」)； 
31) *u* : *uo*，例如 *guõlis*「床舖/窩」(: *gùli*「躺(現在時)」, *gulė́ti*「躺」 / *gul̃ti*「躺下」)[^fn104]；*uo* : *u*，例如 *čiužinė́ti*「滑冰」(: *čiuõžti*「滑動」)； 
32) *ū* : *ui*，例如 *buiti̇̀s*「生活/存在狀態」(: *bū́ti*「在/是」)[^fn105]； 
33) *uo* : *ū*，例如 *pačiū́ža*「溜冰鞋」(: *pačiuõžti*「滑行」)。

## 5.2. 詞根輔音交替（Šaknies priebalsių kaita）

在構詞法過程中（主要是形成**後綴**（priesaga）和**詞尾**（galūnė）的派生詞時），有時詞根的**輔音**（priebalsis）也會發生變化[^fn106]：
1) *t* : *č*，例如 *račiùkas*「小輪子/車匠」, *rãčius*「車匠」(: *rãtas*「輪子」)，*bal̃čius*「白馬/白毛動物」(: *báltas, -à*「白色的」)；*č* : *t*，例如 *bažnýtinis, -ė*「教會的」(: *bažnýčia*「教堂」)，*pė́stininkas, -ė*「步兵/行人」(: *pė́sčias, -ià*「步行的」)； 
2) *d* : *dž*，例如 *puõdžius*「陶工」(: *púodas*「鍋/罐」)，*juodžiùkas*「黑皮膚/黑毛的人或動物」(: *júodas, -a*「黑色的」)，*skur̃džius*「窮人」(: *skurdùs, -i̇̀*「貧窮的」)，*smir̃džius*「臭鼬/發臭的人」(: *smi̇̀rdi*「發臭(現在時)」, *smirdė́ti*「發臭」)，*valdžià*「權力/政府」(: *val̃do*「統治(現在時)」 / *val̃dė*「統治(過去時)」, *valdýti*「統治」)；*dž* : *d*，例如 *pažodi̇̀nis, -ė*「逐字的」(: *pažodžiui̇̃*「逐字地」)。 

## 5.3. 語素融合（Morfemų sankloda）

**語素融合**（morfemų sankloda）（又稱為**語素音韻節點**（morfonologinis mazgas））指的是相鄰**語素**（morfema）的融合[^fn107]，例如，在派生詞 *drį̇́/s/ena*「膽量」(: *drį̇́sti*「敢於」)，*plė́/š/ena*「撕裂物/皮」(: *plė́šti*「撕裂」)[^fn108] 中，後綴 *-sena* 的開頭與詞根的結尾融合在一起（即 *s* 既屬於詞根，也屬於後綴）。比較沒有語素融合的派生詞，其中 *s* 僅屬於後綴 *-sena*：*ei̇̃-sena*「步態」(: *ei̇̃ti*「走」)，*mąstý-sena*「思維方式」(: *mąstýti*「思考」)。在一些帶有後綴 *-ininkas, -ė* 的派生詞中，語素 *in* 也同時是詞基和後綴的一部分，例如：*i̇̀šimt/in/inkas, -ė*「領取養老份額者」(: *i̇̀šimtinė*「接管農場的子女給予父母的贍養物」)，*maš/i̇̀n/inkas, -ė*「打字員/機械師」(: *mašinà*「機器」)，*ord/in/inkas, -ė*「受勳者」(: *ordinas*「勳章」)。比較相同**構詞類型**（darybos tipas）但沒有語素融合的派生詞，例如：*mi̇̀šk-ininkas, -ė*「林業員」(: *mi̇̀škas*「森林」)，*jū̃r-ininkas, -ė*「水手」(: *jū̃ra*「海洋」)，*móksl-ininkas, -ė*「科學家」(: *mókslas*「科學」)。 

## 5.4. 換調（Metatonija）

在構詞法過程中，**換調**（metatonija）——即**重音**（kirtis）和**音調**（priegaidė）的交替——並不罕見。
換調是同源詞中**韻律要素**（prozodiniai elementai）（重音和音調）的交替。在單**音節**（skiemuo）的構詞詞幹中發生的是音調變化，而在多音節（雙音節）詞幹中則是重音位置的變化，比較**前綴**（priešdėlis）*be-* 的派生詞 *bežiẽdis, -ė*「無戒指的是/無花果的」，*bepini̇̀gis, -ė*「無錢的」以及它們的詞基 *ži̇́edas*「戒指/花朵」，*pini̇̀gas*「金錢」；後綴 *-inti* 的派生詞 *várginti*「使疲倦」，*skaitmẽninti*「數位化」（另一個可能的變體是 *skai̇̃tmeninti*[^fn109]）以及它們的詞基 *var̃gti*「受苦」，*skaitmenà*「數字」，*skai̇̃tmeną*「數字(賓格)」。
當詞基的**重音1型**（tvirtapradė priegaidė）（換句話說，**銳音**（akūtas））在派生詞中變成**重音2型**（tvirtagalė priegaidė，即**抑揚音**（cirkumfleksas））時，稱為**重音2型換調**（cirkumfleksinė metatonija）；反之，當重音2型音調變成重音1型音調時，稱為**重音1型換調**（akūtinė metatonija）。重音2型換調較為常見。 
自古以來，重音2型換調表現於以下情況：
a) 由前綴 *be-* 構成的**形容詞**（būdvardis），以及複合詞中第二個**靜詞性複合詞成分**（vardažodiniai dūrinių dėmenys），例如：*be-dar̃bis*「失業的」(: *dárbas*「工作」)，*bendr-a-dar̃bis*「同事」(: *beñdras, -à*「共同的」, *dárbas*「工作」)；*be-plaũkis*「無毛的」(: *pláukas*「毛髮」)，*balt-a-plaũkis*「白髮的」(: *báltas, -à*「白色的」, *pláukas*「毛髮」)；*be-šir̃dis*「無情的」(: *širdi̇̀s*「心」, *ši̇̀rdį*「心(賓格)」)，*kiet-a-šir̃dis*「鐵石心腸的」(: *ki̇́etas, -à*「堅硬的」, *širdi̇̀s*「心」, *ši̇̀rdį*「心(賓格)」)；
b) 詞尾 *-is* 的派生詞中：*geltõn-is*「黃疸/黃色」(: *geltónas, -à*「黃色的」)，*stõr-is*「厚度」(: *stóras, -à*「厚的」)，*šal̃t-is*「寒冷」(: *šáltas, -à*「冷的」)。
在創造新詞時，這種換調並不總是有規律的。比較以下傾向的重音標記方式：*be-kãmeris, -ė* 1「無內胎的」(: *kãmera* 1「內胎」)，*be-vi̇̀dpadis, -ė* 1「無內底的」(: *vi̇̀dpadis* 1「內底」)，*daug-ia-kãmeris, -ė* 1「多腔室的」(比較 *kãmera* 1「腔室」)，*dv-i-ãtramis, -ė* 1「雙支撐的」(比較 *ãtramą*「支撐(賓格)」, *atramà* 3b「支撐」)，*daug-ia-žánris, -ė* 1「多體裁的」(比較 *žánras* 1「體裁」)[^fn110]，*Áuk-sė*「奧克塞(女子名)」(: *áuksas*「黃金」；名字)。

重音2型換調和重音1型換調都有可能出現在由**名詞**（daiktavardis）(a)、形容詞(b)、**動詞**（veiksmažodis）(c) 構成的名詞中，例如：
a) *kal̃n-ius*「山區居民」(: *kálnas*「山」)，*añt-inas*「公鴨」(: *ántis*「鴨子」)，*vi̇̀lk-ė*「母狼」(: *vil̃kas*「狼」)；
b) *ménk-ė*「鱈魚」(: *meñkas, -à*「微小的/劣質的」)，*méil-ė*「愛」(: *meilùs*「可愛的/親切的」, *mei̇̃lų*「可愛的(賓格)」) 以及 *mar̃g-is*「斑駁的動物」(: *márgas, -à*「斑駁的」)，*jaũn-is, -ė*「青少年」(: *jáunas, -à*「年輕的」)；
c) *giñ-klas*「武器」(: *gi̇̀nti*「防禦」)，*bėg-is*「軌道/檔位」(: *bė́gti*「跑」)，*švi̇̀lp-a*「哨子」(: *švil̃pti*「吹哨」)，*bùrb-a*「喋喋不休的人」(LKŽe) (: *bur̃ba*「嘟囔(現在時)」, *burbė́ti*「嘟囔」)。
重音位置變化的例子：*be-asmẽnis, -ė*「無人稱的」(: *asmeñs*「人(屬格)」, *ãsmenį*「人(賓格)」, *asmuõ*「人」)，*asmén-vardis*「人名」(: *asmeñs*「人(屬格)」, *ãsmenį*「人(賓格)」, *asmuõ*「人」, *vardas*「名字」)，*riešùt-medis*「胡桃木」(: *ri̇́ešutas*「堅果」, *mẽdis*「樹」；另一個可能的變體是 *ri̇́ešutmedis*)[^fn111]；*Pa-nemùnė*「帕內穆內(地名)」(: *Nẽmunas*「涅曼河」)，*vasãr-is*「二月」(: *vãsara*「夏天」)，*vieni̇̀š-ius, -ė*「孤獨者」(: *vi̇́enišas, -à*「孤獨的」) 或 *vien-i̇̀šius, -ė*「孤獨者」(: *vi̇́enas, -à*「一個的」)[^fn112]，*lietùv-is, -ė*「立陶宛人」(: *Lietuvà*「立陶宛」, *Li̇́etuvą*「立陶宛(賓格)」)，*vidur̃-kis*「平均數」(: *vidurỹs*「中間」, *vi̇̀durį*「中間(賓格)」)，*gelẽž-tė*「刀刃」(: *geleži̇̀s*「鐵」, *gẽležį*「鐵(賓格)」)。
換調是一種輔助的構詞手段——它的作用類似於詞根元音交替：無論是元音交替還是換調，都不會涵蓋某個構詞類型的所有派生詞（關於這些構詞類型，見第 6.3 節），也就是說，帶有相同**構詞標誌**（darybos formantas）構成的派生詞，可能會保留詞基的重音和音調，但也可能會發生換調。（關於該現象的更多資訊，見 Mikulėnienė 2005，比較 KTŽ: 124；LKE: 350；Barauskaitė 1985: 24–25, 34–35；Barauskaitė, Čepaitienė, Mikulėnienė, Pabrėža, Petkevičienė 1995: 172–180；Kazlauskaitė 2016。） 

## 5.5. 複合詞的連接元音（Dūrinių jungiamieji balsiai） 

在**複合詞**（dūrinys）的第一個和第二個**成分**（sandas）之間，通常可以分出一個元音元素，這被稱為**連接元音**（jungiamasis balsis）。**複合詞成分**（dūrinio dėmuo）可以通過以下元音（或**雙元音**（dvibalsis））進行連接：*-a-* / *-ia-*, *-i-*, *-o-* / *-io-*, *-u-* / *-iu-*, *-ė-*, *-y-*, *-ū-* / *-iū-*, *-uo-*（比較 2.4 節，「**形態學**（morfologija）」部分 1.2.7 節）。關於連接元音在構詞中的作用，尚未有統一的看法，但是應該指出，很難看出該元音具備構詞功能——連接元音的隨意性首先體現在：同一構詞類型[^fn113]的複合詞可以帶有連接元音構成，也可以不帶連接元音構成。例如，由兩個名詞構成的 *auks-a-šir̃dis, -ė*「善良的人」(: *áuksas*「黃金」, *širdi̇̀s*「心」)，*turg-ã-vietė*「市場」(: *tur̃gus*「市場」, *vietà*「地方」) 帶有連接元音，而由兩個名詞構成的類似構詞複合詞 *met-raštis*「編年史」(: *mẽtai*「年」, *rãštas*「文字/文件」)，*nauji̇́en-laiškis*「電子報」(: *nauji̇́ena*「新聞」, *láiškas*「信件」) 則沒有連接元音；由形容詞和名詞構成的複合詞 *ger-a-bū̃dis, -ė*「性情好的」(: *gẽras, -à*「好的」, *bū̃das*「性格」)，*sen-a-mãdis, -ė*「守舊的」(: *sẽnas, -à*「老的」, *madà*「時尚」) 的成分由元音 *-a-* 連接，而類似的複合詞 *bráng-akmenis*「寶石」(: *brangùs, -i̇̀*「昂貴的」, *akmeñs*「石頭(屬格)」, *akmuõ*「石頭」)，*sén-vagė*「故河道」(: *sẽnas, -à*「老的」, *vagà*「河床」) 的成分連接時則沒有連接元音。更有甚者——由相同的詞基構成的複合詞，既可以有連接元音也可以沒有，例如 *balt-gal̃vis, -ė*「白髮的」和 *balt-a-gal̃vis, -ė*「白髮的」；這兩個複合詞都基於形容詞 *báltas, -à*「白色的」和名詞 *galvà*「頭」構成（關於**構詞變體**（darybiniai variantai）見第 8 章）。 
在名詞複合詞的構詞法中，最常見的連接元音是 *-a-*，其次依頻率為 *-ia-*, *-i-*, *-o-*, *-u-*, *-ė-*, *-y-*, *-ū-*, *-io-*, *-iu-*, *-uo-*（Keinys 1999: 73）。形容詞複合詞的成分最常以 *-a-* 和 *-ia-* 連接（Keinys 1999: 81；比較 DLKG 2005: 228；Urbutis 2009: 280）。連接元音通常與複合詞第一成分的詞基的**詞幹末音**（kamiengalis）相同，但這只是一種傾向，並非規則[^fn114]。因此，究竟哪種複合詞具備或不具備哪種連接元音，並沒有明確的規律。不過可以說，發音上的便利在這裡也扮演了不容忽視的角色。

值得一提的是，在構詞法過程中經常會發生幾種變化，也就是說，派生詞和詞基在詞根元音或（和）輔音、**反身詞綴**（sangrąža）、前綴變體、重音、音調等方面會有所不同，試比較**詞尾派生**（galūninė daryba）派生詞（名詞）及其詞基（動詞）：*atãskait-a*「報告/結算」(: *atsiskai̇̃to*「結算(現在時)」 / *atsiskai̇̃tė*「結算(過去時)」, *atsiskaitýti*「結算」)，*apývart-a*「周轉/營業額」(: *ap(si̇̀)vertė*「翻轉/周轉(過去時)」, *ap(si)ver̃sti*「翻轉/周轉」)，*núotaik-a*「情緒」(: *nu(si)tei̇̃kti*「調整心態」)，*pri̇́eglaud-a*「避難所」(: *pri(si̇̀)glaudė*「依偎/避難(過去時)」, *pri(si)glaũsti*「依偎/避難」)，*pri̇́esag-a*「後綴」(: *prisègti*「附加/扣上」)，*sándraug-a*「聯邦/友誼」(: *susidraugáuti*「交朋友」, *susidraugáuja*「交朋友(現在時)」, *susidraugãvo*「交朋友(過去時)」)，*sámbrūzd-is*「騷動」(: *subrùzdo*「騷動(過去時)」, *subrùzti*「騷動」)，*į̇́mon-ė*「企業」(: *įmãno*「能做到(現在時)」 / *įmãnė*「能做到(過去時)」, *įmanýti*「能做到」)，*núomon-ė*「意見」(: *nu(si)mãno*「理解(現在時)」 / *nu(si)mãnė*「理解(過去時)」, *nu(si)manýti*「理解」)，*sántyk-is*「關係」(: *suti̇̀kti*「同意/遇見」)，*pósūk-is*「轉彎/轉折」(: *pa(si)sùkti*「轉向」)，*núopeln-as*「功勞」(: *nusipel̃no*「值得(現在時)」 / *nusipel̃nė*「值得(過去時)」, *nusipelnýti*「值得」)[^fn115] 以及其他。
