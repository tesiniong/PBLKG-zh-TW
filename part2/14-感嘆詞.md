# 14. 感嘆詞（JAUSTUKAS）

**感嘆詞**（jaustukas，拉 interactio，語言學 jaustukas）是**不變化的詞類**（nekaitoma kalbos dalis），由直接表達情感、意志行為或促動，而不對其進行命名的詞組成（LKG 1971: 698–733; Paulauskienė 1994: 417–420; LKE 1999: 279; DLKG 2006: 462–467; Jakaitienė 2009: 23）。感嘆詞不會命名情感或意志，而只是表達它們，因此它們與非語言元素（面部表情、手勢）密切相關，語音**語調**（intonacija）也發揮著非常重要的功能。

感嘆詞可以表達正面和負面情緒，以及禁止、克制、安靜、禮貌、呼喚或驅趕動物和鳥類。根據這些區別，可以劃分出充滿情感的呼喊，例如：*à, ã, ak, ái, éi, ói, brr*「（形容冷）」, *òt*；執行意志、命令、指示、促動功能的詞語，例如：*žiū̃, še, te, šèkit, tèkit, valiõ*。部分感嘆詞可以歸入呼喚和驅趕動物、鳥類的詞，即所謂的**呼喚語**（invokacijos），例如：*ciū ciū ciū*「（喚狗聲）」, *kac kac kac*「（喚貓聲）」, *štiš*「（趕雞聲）」, *škac*「（趕貓聲）」；**交際用語**（etiketo žodžiai），例如：*ãčiū*「謝謝」, *dė̃kui*「謝謝」, *prãšom*「請 / 不客氣」, *sudiẽ*「再見」, *lãbas*「你好」, *sveiki̇̀ gyvi̇̀*「你好」；**詈語**（keiksmažodžiai），例如：*rùpūže*「（詈語）癩蛤蟆」, *velniai̇̃*「（詈語）惡魔」, *ei̇̃k velnióp*「（詈語）見鬼去吧」（Zabarskaitė 2009: 91–109）。

感嘆詞的句法功能有兩種：它可以代替句子（在這方面類似於**自立詞類**（savarankiškosios kalbos dalys）），例如：*Ačiū.*「謝謝。」 *Valio.*「太棒了。」 *Džiugu,*「真高興，」 也可以是句子成分的附屬部分，例如：*Brr, kaip šalta.*「brr（形容冷），好冷啊。」

感嘆詞的結構多種多樣。這可以是形式上與一個或幾個聲音（通常相同）重合的極短詞，例如：*ã, à, ė, ái, ak, ek, et, ša, šš, tš, ach, ech, brr*「（形容冷）」, *ojė́, ajajai* 等；也可以是由幾個詞組成的組合，例如：*po šimts kalakutų*「見鬼（字面：一百隻火雞）」, *Dieve gink.*「上帝保佑。」 感嘆詞可以是單獨的，例如：*ai, ak, oi*，也可以是重複的，例如：*ajaj, ojoj, ohoho, kac kac kac*「（喚貓聲）」。

在起源與構詞的層面上，感嘆詞也各不相同。許多**原生感嘆詞**（pirminiai jaustukai），例如：*ai, ei, oi, ak, ot* 等，起源於情感驚呼，這些感嘆詞無法與立陶宛語的其他詞彙建立聯繫。**次生感嘆詞**（antriniai jaustukai）是透過功能性派生的方式，從僵化的獨立詞彙形式起源。有很大一部分感嘆詞起源於名詞的**呼格**（Šauksmininkas），例如：*diẽ < Dieve*「天哪 < 上帝（呼格）」, *brol < broli*「兄弟 < 兄弟（呼格）」, *dievuliau*「上帝（呼格）」, *Jėzau*「耶穌（呼格）」, *Marija*「瑪利亞（呼格）」, *rupūže*「癩蛤蟆（呼格）」等。此外，不少感嘆詞與**動詞**（veiksmažodis）形式有關，例如：*žiū̃ < žiūrėk*「看啊 < 看（祈使語氣）」, *palà < palauk*「等一下 < 等（祈使語氣）」, *valio < valiok*「太棒了 < 能（祈使語氣）」, *eik*「去（祈使語氣）」。源自形容詞的感嘆詞有 *sveikas*「你好（字面：健康的）」, *labas*「你好（字面：好的）」。小品詞 *še, te* 既可以歸類為感嘆詞，也可以歸為情感小品詞，因為它們的區分基礎非常相似（當指示意義模糊時，上述小品詞會被區分為感嘆詞，但確定這一點並不總是容易的）。有相當一部分感嘆詞是由詞組或短句形成的，例如：*dievaži̇̀, dievàž, dievažiñ < Dievas žino*「天曉得 < 上帝知道」；*amžinãtilsį < amžiną atilsį*「安息吧 < 永遠的安息」。感嘆詞的功能也可以由慣用語來執行，例如：*po šimts pypkių*「見鬼（字面：一百根煙斗）」, *velniai rautų.*「該死（字面：魔鬼拔掉）」。感嘆詞也可以是借詞，例如：*brãvo*「太棒了（借詞）」, *bis.*「安可（借詞）」。

從用法的角度來看，感嘆詞涵蓋了非常廣泛的表達範圍——它們可以表達懷疑、驚訝、自豪、厭惡、憐憫、痛苦、意志行為和促動。這類詞經常在自由風格、**文學作品**（grožinė literatūra）和情感性質的交流中使用。正式文體通常只具有表達禮貌的感嘆詞。
