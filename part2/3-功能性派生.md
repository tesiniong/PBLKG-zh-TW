# 3\. 功能性（句法性）派生（FUNKCINĖ (SINTAKSINĖ) DARYBA）

**功能性（句法性）派生**（funkcinė/sintaksinė daryba）是一種與語素學和構詞法層面相關的現象：在不添加任何派生詞綴（詞語的語素結構不變，形式上保持不變）的情況下，一個詞類的詞語開始以另一個詞類的意義發揮功能。

例如，比較以下兩個句子：（1）Ši stiklinė vaza man labai patinka.「這個玻璃花瓶我很喜歡。」和（2）Ši stiklinė man dukters dovanota.「這個玻璃杯是我女兒送的。」——詞語 stiklinė 在第一個句子中是形容詞（作定語），在第二個句子中則是名詞（作主語）。這些詞語「由於語義、形態或其他變化而開始屬於不同的詞類」（Markevičius 2012: 141）。

確定某一詞類是否以另一詞類的意義發揮功能，需要依據以下主要識別和區分標準：

1. **詞彙標準**（確定意義）
2. **形態標準**（可能使用該詞類不常見的形式）
3. **句法標準**（必須確定具體用法中的句法功能）
4. **使用頻率標準**（功能性派生的例子是否在標準語中有記錄）

功能性派生不如形態構詞法那麼活躍，但它是一個重要的、持續發生的現象。

主要的功能性派生類型：

**名詞化**（daiktavardėjimas）——其他詞類以名詞意義使用，尤其是形容詞和分詞：priimamasis「候診室、接待室」（< 被動分詞 priimamasis「被接受的」），miegamasis「臥室」，ginamasis「被辯護的人」，teisiamasis「被審判的人」。

**形容詞化**（būdvardėjimas）——主要是分詞轉化為形容詞，失去時態和語態範疇的意義，而獲得穩定特徵的意義：pažįstamas, -a「熟悉的」，žinomas, -a「著名的」，riestas, -a「彎曲的」，lenktas, -a「彎的」；pasiutęs, pasiutusi「瘋狂的、極度的」，pasiturintis, -i「富裕的」。名詞化通常以形容詞化為先行過程（分詞 → 形容詞 → 名詞的轉化序列）。

**代詞化**（įvardėjimas）——其他詞語以代詞意義使用，例如名詞 pats, pati 在口語中表示「丈夫、妻子」，形容詞 pastarasis「後者」（以指示功能使用）。

**副詞化**（prieveiksmėjimas）——其他詞類（有時只是其特定形式）轉化為副詞。副詞化是副詞構成的主要方式，被視為最具生產力的副詞構成途徑。常見的副詞化例子：drauge「一起（副詞）」（< 工具格 drauge），greitai「快速地」（< 形容詞後綴 -ai），ilgam「長期地」，ryt-oj「明天」，tyčia「故意地」，šiandien「今天」。副詞化通常使詞尾的語法功能失效，成為凝固詞尾（sustabarėjusi galūnė）。

**非自立詞類的功能性派生**——其他詞類以小詞（dalelytė）、介詞（prielinksnis）和連詞（jungtukas）的意義使用。許多非自立詞類都是通過功能性派生形成的，在共時語言學中有時難以確定具體功能詞來自哪個詞類。
