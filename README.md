## 前言

近十幾年來，未有較為完整的立陶宛語語法描述問世。既有的語法著作多為學術性質，對語言現象分類詳盡、例證豐富，卻鮮少分析和解釋語言的多樣面貌。針對語言系統各個分支，亦有面向高等教育的教材或專題性出版物問世，然而仍缺乏一部對語言使用者切實有用的語法描述——一部足以讓人了解或核實語言用法相關資訊的著作。類似性質的實用語法曾於 2003 年出版（Meilutė Ramonienė 與 Joana Pribušauskaitė 著，《立陶宛語實用語法》，維爾紐斯：Baltų lankų 出版社），但其目標在於協助學習立陶宛語的外籍人士，因此在材料選取與呈現方式上，與傳統語法著作有相當大的差異。實用語言手冊所提供的資訊過於簡略，無法讓語言使用者了解語言系統各單位的典型性，也無從核實邊緣表達方式的功能性。

確立各種語言使用規範的規則，通常描述語言單位，並提及區分正確用法與錯誤用法的特徵。語言規範有時只是將較為適切的用法與不那麼適切（但並非錯誤）的用法並列呈現。本書《實用立陶宛標準語語法》（以下簡稱《語法》）的首要目的，在於幫助語言規範的應用者評估所討論語言單位在語法系統中的位置，更好地理解具體情況下的規範描述，並能將規範應用於書中未提及的類似語言用法情形。

《語法》的目標，是描述語法單位或現象的主要特徵，並盡可能詳細地討論邊緣現象，幫助使用者——包括語言規範制定者——確定語言系統的邊界，識別違反語言系統的用法案例。但本書並不制定語法規範，也不對具體用法的正確性作詳細評論，僅在必要時提供規範來源的參考，或附上與所述語言單位相關的簡短規範性說明。歸屬於所描述的語法系統本身，就是判定規範的標準。一些較為複雜的用法問題或規範化議題，在獨立的小節中另行討論，並提出以系統性為基礎的詮釋。

本書的語言語法系統描述，並不旨在解決理論性的語言問題，但會討論那些在詮釋上語言學家意見分歧的現象，採納編者較為認可的立場，並加以論證。本書遵循立陶宛語語法的傳統概念框架——學校教材正是依此框架描述語法現象，語言正確性規範的制定、拼寫與標點規則的擬定，也都以這一詮釋為基礎。

書中所描述的各語言層次單位系統，不涵蓋已不再適用於標準語的情形，即：已廢棄的形式或構詞元素，或多或少在方言中流行、但不適用於標準語且規範來源不建議使用的語言單位。

《語法》提供系統性的概要描述，可供各層次的教學參考，對有志深入了解語言的母語者也不無裨益，但其材料的編排方式並非以教學目的為導向。

《語法》共分四個部分：一、語音學（Fonetika，Rūta Kazlauskaitė 著），二、形態學（Morfologija，Asta Balčiūnienė 著），三、構詞學（Žodžių daryba，Jolanta Vaskelienė 著），四、句法學（Sintaksė，Albinas Drukteinis 著）。每部分開頭均設有引言，介紹該部分的結構、基本概念以及與該語法領域相關的共同問題；末尾附有術語索引。書中所用例證大多為新設，並非引自其他語法描述；例句取自各種文體的文本，並常經過改編，以更清晰地說明所討論的現象，因此不附出處引文。

《語法》依據國家立陶宛語委員會發起的《〈實用立陶宛標準語語法〉編撰》計畫（主持人：Albinas Drukteinis）編寫完成，由「國家語言規範化、使用、教育與推廣」計畫提供資金支持。
