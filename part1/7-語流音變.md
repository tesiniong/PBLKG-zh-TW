# 語流音變（Garsų prisitaikymas kalboje）

極少會發出單一、孤立的語音。由於受中樞神經系統控制，**發音器官**（kalbos padargai）同步運作，接連發出的語音會適應相鄰的語音。語音的適應、一個語音對另一個語音的影響稱為**協同發音**（koartikuliacija）（拉丁語 co-「與、共；一起」+ articulātio < articulo「我清晰發音；我切分」）(Hardcastle, Hewlett 1999: 1)。協同發音的任務是探究大腦、中樞神經系統如何控制移動發音器官的肌肉。**語音學**（fonetika）關注由協同發音引起的變化，因為語音在互相適應時會產生強弱不等的變化。改變後的語音發音更簡單、更舒適。

語音變化在時間維度上有兩種：**歷時性**（diachroniniai）（歷史性的）與**共時性**（sinchroniniai）（同時期的）。

## 歷時用語音學過程（Diachroniniai fonetiniai procesai）

歷時性變化發生在一段時間內。由於這些變化，一種共時系統最終會被另一種取代。從現在的立場來看，歷時性讓我們能理解語言及其具體現象的本質，特別是能解釋各種不一致與例外。舉例來說，**動詞**（veiksmažodis）形式 *mès-ti* [ˈmʲɛsʲtʲɪ]「投擲（不定式）」與 *mẽ-ta* [2ˈmʲæːtɐ]「投擲（現在時）」、*mẽ-tė* [2ˈmʲæːtʲeː]「投擲（過去時）」的**詞根**（šaknis）末尾**輔音**（priebalsis）並不相同。它們不同，是因為**不定式**（bendratis）中新引入了 *s*；如 *mẽt-a* [2ˈmʲæːtɐ] 與 *mẽt-ė* [2ˈmʲæːtʲeː] 所示，較古老的不定式形式是 *\*mèt-ti*。相鄰的兩個**塞音**（sprogstamasis priebalsis）*tt* 中，第一個根據其一項特徵（空氣通過的方式）發生了改變（異化），變成了**摩擦音**（pučiamasis priebalsis）*s*。

歷時性過程包含**量的元音交替**（kiekybinė balsių kaita）與**質的元音交替**（kokybinė balsių kaita）、**異化**（disimiliacija）、**塞擦化**（afrikacija）、**異位**（metatezė），以及語音脫落與添加現象：**消除**（eliminacija）、**首音省略**（aferezė）、**中音省略**（sinkopė）、**末音省略**（apokopė）、**收縮**（kontrakcija）、**省略**（elizija）、**同音節省略**（haplologija）與**增音**（protezė）、**插入音**（epentezė）。

**量的元音交替**（kiekybinė balsių kaita）——是指在同源詞的詞根或同一詞語的不同形式中，同一個歷史短**元音**（balsis）與長元音互相替換的現象。例如：

`[ɪ] (< *ĭ) : [iː] (< *ī)` – *di̇̀delis* [ˈdʲɪdʲɛlʲɪs]「大的」 : *dỹdis* [2ˈdʲiːdʲɪs]「大小」；*ski̇̀lo* [sʲkʲɪloː]「（他）裂開了」 : *skỹla* [2ˈsʲkʲiːlɐ]「（他）正在裂開」；
`[ʊ] (< *ŭ) : [uː] (< *ū)` – *mùša* [ˈmʊʃɐ]「（他）打」 : *mū̃šis* [2ˈmuːʃʲɪs]「戰鬥」；*dùsti* [ˈdʊsʲtʲɪ]「窒息（不定式）」 : *dū̃sta* [2ˈduːstɐ]「（他）正在窒息」；
`[ɛ] / [æː] (< *ĕ) : [eː] (< *ē)` – *vèžti* [ˈʋʲɛʃʲtʲɪ]「運送（不定式）」, *vẽža* [2ˈʋʲæːʒɐ]「（他）運送」 : *vėžės* [2ˈʋʲeːʒʲeːs]「車轍（複數）」；*geriù* [ɡʲɛˈrʲʊ̟]「我喝」, *gẽria* [2ˈɡʲæːrʲɛ]「（他）喝」 : *gė́rė* [1ˈɡʲeːrʲeː]「（他）喝了」；
`[ɐ] / [ɑː] (< *ă) : [oː] (< *ā)` – *gražùs* [ɡrɐˈʒʊs]「美麗的」, *grãžų* [2ˈɡrɑːʒuː]「美麗的（單數賓格）」 : *grõžis* [2ˈɡroːʒʲɪs]「美」；*varýti* [ʋɐ1ˈrʲiːtʲɪ]「驅趕（不定式）」, *vãro* [2ˈʋɑːroː]「（他）驅趕」 : *vorà* [ʋoːˈrɐ]「隊伍」。

**質的元音交替**（kokybinė balsių kaita）——是指在同源詞的詞根或同一詞語的不同形式中，替換出現不同音質的元音或**二合音核**（dvigarsis）。
其基礎為短音 *\*ĕ* : *\*ă* 與長音 *\*ē* : *\*ā* 的交替。如前所述，現代立陶宛語中這些短元音有兩種形式：短音 `[ɛ]`、`[ɐ]`，以及因**重音**（kirtis）而延長變成歷史長音的 `[æː]`、`[ɑː]`；而長元音則在音質上發生了變化，變得狹窄。考量歷時變化，交替可標記為：`[ɛ] / [æː] : [ɐ] / [ɑː] : [eː] : [oː] / [uɔ]`，例如：

*vèžti* [ˈʋʲɛʃʲtʲɪ]「運送（不定式）」, *vẽža* [2ˈʋʲæːʒɐ]「（他）運送」 : *važiúoti* [ʋɐ1ˈʒʲu̟ɔtʲɪ]「乘車（不定式）」, *vãžis* [2ˈʋɑːʒʲɪs]「雪橇」 : *vėžės* [2ˈʋʲeːʒʲeːs]「車轍（複數）」；
*slegiù* [sʲlʲɛˈɡʲʊ̟]「我壓」, *slẽgia* [2ˈsʲlʲæːɡʲɛ]「（他）壓」 : *slãktai* [2ˈslɑːktɐɪ]「捕捉老鼠或野獸的器具、陷阱、圈套」 : *slė́gti* [2ˈsʲlʲeːktʲɪ]「壓（不定式）」 : *slogà* [sloːˈɡɐ]「感冒」, *slúoksnis* [1ˈsluɔksʲnʲɪs]「層」。

在上述討論的基礎交替語音之後，可以附加上某個**響輔音**（sklandusis priebalsis）（即 R：`[r l m n]`）或 `[ɪ]`、`[ʊ]`。由此形成另外三組交替系列——元音 *e, i, u*。在語言中並非能找到所有交替系列的成員，此外有些詞彙並不普遍，僅在立陶宛某些地區為人所知。

*   *e* 系列：*eR : aR : ėR : uoR : iR : uR : yR : ūR*，例如：
    *gérti* [1ˈɡʲæˑrʲtʲɪ]「喝（不定式）」 : *prãgaras* [2ˈprɑːɡɐrɐs]「貪吃者、大胃王、不知足的人」 : *gė́rė* [1ˈɡʲeːrʲeː]「（他）喝了」 : *gi̇̀rtas* [1ˈɡʲɪrtɐs]「喝醉的」 : *gùrkšnis* [1ˈɡʊrʲkʃʲnʲɪs]「一口」；
    *tvérti* [1ˈtʲʋʲæˑrʲtʲɪ]「圍起（不定式）」 : *tvártas* [¹ˈtʋɑˑrtɐs]「牲畜棚」 : *tvė́rė* [1ˈtʲʋʲeːrʲeː]「（他）圍起了」 : *tvi̇̀rtas* [1ˈtʲʋʲɪrtɐs]「堅固的」 : *turė́ti* [tʊ1ˈrʲeːtʲɪ]「擁有（不定式）」 : *tūris* [2ˈtuːrʲɪs]「體積」。
*   *i* 系列：*ie, ei (ej) : ai (aj) : ė(j) : i(j) : y*，例如：
    *šliẽti* [1ˈʃʲlʲiɛtʲɪ]「依靠（不定式）」, *šlei̇̃vas* [2ˈʃʲlʲɛɪˑʋɐs]「羅圈腿的」, *šlẽja* [2ˈʃʲlʲæːjɛ]「（他）依靠」 : *šlai̇̃tas* [2ˈʃlɐɪˑtɐs]「斜坡」, *šlãjos* [2ˈʃlɑːjo̟ːs]「工作用雪橇」, *šlėjo* [2ˈʃʲlʲeːjo̟ː]「（他）曾依靠」 : *šli̇̀jo* [ˈʃʲlʲɪjo̟ː]「（他）依靠了」 : *šlýti* [1ˈʃʲlʲiːtʲɪ]「傾斜（不定式）」；
    *šviẽsti* [2ˈʃʲʋʲiɛsʲtʲɪ]「發光（不定式）」, *švei̇̃sti* [2ˈʃʲʋʲɛɪˑsʲtʲɪ]「擦亮（不定式）」 : *ãtšvaitas* [2ˈɑːtʃʋɐɪtɐs]「反光物」 : *švė́stelti* [1ˈʃʲʋʲeːsʲtʲɛlʲtʲɪ]「稍微扔一下、給予」 : *švi̇̀sti* [ˈʃʲʋʲɪsʲtʲɪ]「變亮（不定式）」 : *švytė́ti* [ʃʲʋʲiː1ˈtʲeːtʲɪ]「閃耀（不定式）」。
*   *u* 系列：*ˈau, au (av, ov) : ė(v) : uo : u : ū*，例如：
    *čiáupti* [1ˈʧʲæˑʊpʲtʲɪ]「閉合（不定式）」, *taupýti* [tɒʊ1ˈpʲiːtʲɪ]「節省（不定式）」 : *čiuõpti* [1ˈʧʲu̟ɔpʲtʲɪ]「摸索（不定式）」 : *tuopýti* [tuɔ1ˈpʲiːtʲɪ]「節省」 : *čiùpti* [ˈʧʲʊ̟pʲtʲɪ]「抓住（不定式）」（以及 *čiūpsóti* [1ˈʧʲu̟ːp1ˈsoːtʲɪ]「處於閉嘴狀態」）；
    *blaũstis* [2ˈblɒʊˑsʲtʲɪs]「變得陰沉、起霧」 : *blė́sti* [1ˈbʲlʲeːsʲtʲɪ]「熄滅（不定式）」 : *blúostas* [1ˈbluɔstɐs]「布洛斯塔斯（同源詞）」 : *blùsti* [ˈblʊsʲtʲɪ]「眼皮打架、打盹」 : *blūsà* [bluːˈsɐ]「睡意、瞌睡」；
    *ráuti* [1ˈrɑˑʊtʲɪ]「拔（不定式）」 : *ravė́jo* [rɐ1ˈʋʲeːjo̟ː]「（他）拔除了雜草」, *róvė* [1ˈroːʋʲeː]「（他）拔了」。

有時也會出現混合交替。它包含屬於不同系列的成員，例如：
*plė́šyti* [1ˈpʲlʲeːʃʲiːtʲɪ]「撕裂（不定式）」 : *plúoštas* [1ˈpluɔʃtɐs]「束、把」（*e* 系列）, *pléišėti* [1ˈpʲlʲæˑɪʃʲeːtʲɪ]「裂開（不定式）」 : *atplaišà* [ɐtplɐɪˈʃɐ]「碎片」 : *plýšti* [1ˈpʲlʲiːʃʲtʲɪ]「撕破（不定式）」（*i* 系列）, *plaušai̇̃* [plɒʊ2ˈʃɐɪˑ]「韌皮纖維」（*u* 系列）；
*brė́žti* [1ˈbʲrʲeːʃʲtʲɪ]「劃（不定式）」 : *bri̇̀žės* [ˈbʲrʲɪʒʲeːs]「彈簧耙」 : *brỹžis* [2ˈbʲrʲiːʒʲɪs]「劃痕、線條、記號」 : *brúožas* [1ˈbruɔʒɐs]「特徵、線」（*e* 系列） : *braižýti* [brɐɪ1ˈʒʲiːtʲɪ]「繪圖（不定式）」（*i* 系列） : *brū́žinti* [1ˈbruːʒʲɪnʲtʲɪ]「摩擦（不定式）」 : *nubraũžti* [nʊ2ˈbrɒʊˑʃʲtʲɪ]「擦掉、抹去」（*u* 系列）。

**異化**（disimiliacija）（或稱不相似化）——將兩個相同或相似輔音中的一個，替換為發音位置相近的輔音。例如，當 *t*（`[t]`, `[tj]`）、*d*（`[d]`, `[dj]`）遇到相同的 *t*（`[t]`, `[tj]`）、*d*（`[d]`, `[dj]`）時，會變成 *s*（`[s]`, `[sj]`）（在**清音**（duslusis priebalsis）前）以及 *z*（`[z]`, `[zj]`）（在**濁音**（skardusis priebalsis）前）：
*\*mèt-ti*（比較 *mẽt-ė* [2ˈmʲæːtʲeː]「（他）投擲了」）> *mèsti* [ˈmʲɛsʲtʲɪ]「投擲（不定式）」；*\*met-damas* > *mèsdamas* [ˈmʲɛzdɐmɐs]「投擲著」；*\*išvýd-ti*（比較 *išvýd-o* [ɪʃʲ1ˈʋʲiːdoː]「（他）看見了」）> *išvýsti* [ɪʃʲ1ˈʋʲiːsʲtʲɪ]「看見（不定式）」；*\*vai̇̃d-das*（比較 *vaid-éntis* [ʋɐɪ1ˈdʲæˑnʲtʲɪs]「顯現（不定式）」）> *vai̇̃zdas* [2ˈʋɐɪˑzdɐs]「景象」。
在**方言**（tarmė）中也能聽到這樣的例子，如 *kali̇̀dorius* [kɐˈlʲɪdɔrʲʊ̟s]「走廊」（*rr > lr*）、*léndrė* [1ˈlʲæˑnʲdʲrʲeː]「蘆葦」（*nn > ln*）、*šálkmėtės* [1ˈʃɑˑljkʲmʲeːtʲeːs]「薄荷」（*tt > kt*）。在這些方言例子中，輔音隔著距離互相影響——發生了遠距異化。

**塞擦化**（afrikacija）（拉丁語 affricātio「摩擦」）——塞音變為複合輔音，即**塞擦音**（sprogstamasis-pučiamasis (afrikata)）。塞擦音的開頭（成阻）像塞音，而發音（持阻）與結尾（除阻）則像在相同位置發出的摩擦音。立陶宛語中自有的塞擦音是 *č*（`[tʃj]`）、*dž*（`[dʒj]`）（在 Dzūkija 地方的方言中為 *c*（`[tsj]`）、*dz*（`[dzj]`））。它們源自組合 *\*tj*, *\*dj*，例如：*\*svetjuñ* > *svečių̃* [sʲʋʲɛ2ˈʧʲuː]「客人的（複數屬格）」, *\*gai̇̃djā* > *gai̇̃džio* [2ˈɡɐɪˑʤʲoː]「公雞的（單數屬格）」。

**異位**（metatezė）——同源詞中輔音位置的互換。在塞音前發音為輔音群 *ks, kš, gz, gž*，而在元音前則發音為 *sk, šk, zg, žg*，例如：*pliẽksti* [2ˈpʲlʲiɛkjsʲtʲɪ]「拍打（不定式）」 : *pliẽskia* [2ˈpʲlʲiɛsʲkʲɛ]「（他）拍打」, *réikšti* [1ˈrʲæˑɪkjʃʲtʲɪ]「表達（不定式）」 : *réiškia* [1ˈrʲæˑɪʃʲkʲɛ]「（他）表達」, *règzti* [ˈrʲɛkjsʲtʲɪ]「編織（不定式）」 : *rẽzga* [2ˈrʲæːzɡɐ]「（他）編織」, *džer̃gžti* [2ˈʤʲɛrʲˑkʲʃʲtʲɪ]「發出刺耳聲（不定式）」 : *džer̃žgia* [2ˈʤʲɛrʲˑʒʲɡʲɛ]「（他）發出刺耳聲」。

**消除**（eliminacija）——在構成新詞時丟棄輔音。通常被丟棄的是發在摩擦音 *s*（`[s]`, `[sj]`）或某個響輔音之前的 *t*（`[t]`, `[tj]`）或 *d*（`[d]`, `[dj]`），例如：*mąst-lùs* > *mąslùs* [mɑːˈslʊs]「深思的」, *raud-muõ* > *raumuõ* [rɒʊ2ˈmuɔ]「肌肉」, *smárd-vė* > *smárvė* [1ˈsmɑˑrʲʋʲeː]「臭味」。

**首音省略**（aferezė）——丟棄詞首的語音，例如：*beždžiõnė* [bʲɛ2ˈʒʲʤʲo̟ːnʲeː]「猴子」（白俄羅斯語 абязьяна）, *Zarasai̇̃* [zɐrɐ2ˈsɐɪˑ]「扎拉賽（地名）」< *\*Azarasai*（瑟羅尼語 Ezerasai「多湖的」，參見 Būga 1959: 20–21）。

**中音省略**（sinkopė）——丟棄詞中的無重音短元音，例如：*šitai̇̃* > *štai̇̃* [2ˈʃtɐɪˑ]「這就是」, *kẽlinės* > *kélnės* [1ˈkʲæˑlʲnʲeːs]「褲子」, *vi̇̀sakas* > *vi̇̀skas* [ˈʋʲɪskɐs]「一切」。此類例子在方言中很豐富，這裡丟棄的還包括元音 *e* `[ɛ]`、*u* `[ʊ]`：*šulinỹs* [ʃʊlʲɪ2ˈnʲiːs] > *šulnỹs* [ʃʊlʲ2ˈnʲiːs]「井」, *dárgana* [1ˈdɑˑrɡɐnɐ] > *dárgna* [1ˈdɑˑrɡnɐ]「陰雨天氣」, *di̇̀deliai* [ˈdʲɪdʲɛlʲɛɪ]「非常、猛烈地」> *di̇̀dliai* [ˈdʲɪdʲlʲɛɪ], *są́siuvinis* [1ˈsɑːsʲʊ̟ʋʲɪnʲɪs] > *są́svinis* [1ˈsɑːsʲʋʲɪnʲɪs]「筆記本」等等。

**末音省略**（apokopė）——丟棄詞尾的一個或多個語音，例如：*nẽšas* [2ˈnʲæːʃɐs] < *nẽšasi* [2ˈnʲæːʃɐsɪ]「（他）攜帶著」, *mẽdy* [2ˈmʲæːdʲiː]、*mẽdyj* [2ˈmʲæːdʲiːɪ] < *mẽdyje* [2ˈmʲæːdʲiːjɛ]「在樹上」, *šiañdie* [2ˈʃʲɛnʲˑdʲiɛ] < *šiañdien* [2ˈʃʲɛnʲˑdʲiɛn]「今天」。

**收縮**（kontrakcija）——將兩個不同的語音縮減為一個。被縮減的包含元音（例如：*ne-yrà* > *nėrà* [nʲeːˈrɐ]「沒有」, *grė-(j)i̇̀mas* < *grei̇̃mas* [2ˈɡrʲɛɪˑmɐs]「煮沸牛奶或湯上的薄膜」）和輔音（例如：*mùš-s* > *mùš* [ˈmʊʃ]「（他）將會打」, *lūž-snà* > *lūšnà* [luːˈʃnɐ]「茅屋」（比較 *lūžo* [1ˈluːʒoː]「（他）折斷了」））。拼寫中並沒有顯示詞根末尾輔音與**後綴**（priesaga）輔音的收縮現象（*š < šs / žs*）。

**省略**（elizija）——當下一個詞以相同的元音開頭時，丟棄詞語或**語素**（morfema）結尾的一個元音，例如：*nẽsti* [2ˈnʲæːsʲtʲɪ] < *ne ẽsti*「不吃（動物）」, *nesù* [nʲɛˈsʊ] < *ne esù*「我不是」, *nei̇̃ti* [2ˈnʲɛɪtʲɪ] < *ne ei̇̃ti*「不去」, *amžinãtilsį* [ɐmʲʒʲɪ2ˈnɑːtʲɪlʲsʲiː] < *ámžiną ãtilsį*「願其安息」。

**同音節省略**（haplologija）——丟棄兩個相同或相似**音節**（skiemuo）中的一個。在立陶宛**標準語**（bendrinė kalba）中，有時會丟棄派生詞中後綴 *-ininkas* 的 *-ni-* 音節，例如：*maši̇̀ninkas* [mɐˈʃʲɪnʲɪŋˑkɐs] < *maši̇̀nininkas*「打字員」, *šuliniñkas* [ʃʊlʲɪ2ˈnʲɪŋˑkɐs] < *šulininiñkas*「挖井人」。

**增音**（protezė）——在詞首添加非詞源性的語音。這樣的附加 *s*（`[s]`, `[sj]`）出現在 *stvérti* [1ˈsʲtʲʋʲæˑrʲtʲɪ] < *tvérti*「拿取、抓取」, *skėsti* [2ˈsʲkʲeːsʲtʲɪ] < *kėsti*「擴展」等詞中。帶有增音元音的例子可以在方言中找到：*atkõčius* [ɐtk2ˈoːʧʲʊ̟s]「織工」（波蘭語 tkacz；比較姓氏 *Atkočáitis*, *Atkočiū́nas*）, *uzbõnas* [ʊ2ˈzboːnɐs]「水罐」（波蘭語 dzban, 白俄羅斯語 збaн）。

**插入音**（epentezė）——在詞中插入非詞源性的語音，例如 *žvaigždė* [ʒʋɐɪɡj2ˈʒʲdʲeː]「星星」（比較某些薩莫吉提亞方言的 *žvaizdė*，俄語 звезда）。在方言中使用的有 *vykšnià* [ʋʲiːˈkʲʃʲnʲɛ] < *vyšnià* [ʋʲiːˈʃʲnʲɛ]「櫻桃」, *nuvei̇̃na* [nʊ2ˈʋɛɪnɐ] < *nuei̇̃na* [nʊ2ˈɛɪˑnɐ]「（他）走開」, *pãletas* [2ˈpɑːlʲɛtɐs] > *páltas* [1ˈpɑˑltɐs]「大衣」。

語音變化是由於對語言秩序（節奏）、差異（多樣性）與簡短的追求所決定的。換言之，語音變化是為了減輕發音的負擔。
很大一部分的語音交替規律，可以根據現代語言的數據——即應用特定的**語素音韻學**（morfonologija）規則來解釋。例如，*e* `[ɛ]`, `[æː]` : *ė* `[eː]` 的交替一致表現在如 *kélti* [1ˈkʲæˑlʲtʲɪ]「舉起（不定式）」, *kẽlia* [2ˈkʲæːlʲɛ]「（他）舉起」, *kė́lė* [1ˈkʲeːlʲeː]「（他）舉起了」；*šérti* [1ˈʃʲæˑrʲtʲɪ]「餵養（不定式）」, *šẽria* [2ˈʃʲæːrʲɛ]「（他）餵養」, *šė́rė* [1ˈʃʲeːrʲeː]「（他）餵養了」；*sémti* [1ˈsʲæˑmʲtʲɪ]「舀（不定式）」, *sẽmia* [2ˈsʲæːmʲɛ]「（他）舀」, *sė́mė* [1ˈsʲeːmʲeː]「（他）舀了」等動詞的基本形式中；*e* `[ɛ]`, `[æː]` : *a* `[ɐ]` 的交替則常見於從帶有**前綴**（priešdėlis）的動詞派生而來、帶**詞尾**（galūnė）的名詞中：*pãsaka* [2ˈpɑːsɐkɐ]「童話」（：*pàsekė* [ˈpɐsʲɛkjeː]「（他）講了故事」）, *núolaida* [1ˈnuɔlɐɪdɐ]「折扣」（：*nuléido* [nʊ1ˈlʲæˑɪdoː]「（他）放下了」）, *į̇́tampa* [1ˈiːtɐmpɐ]「電壓、緊張」（：*į̇̃tempė* [2ˈiːtʲɛmʲpʲeː]「（他）拉緊了」）等等。

## 共時用語音學過程（Sinchroniniai fonetiniai procesai）

共時變化是生動的、發生在實時之中的——它是從語言使用者的角度來看待不斷變化的語言。例如，動詞 *mèsti* [ˈmʲɛsʲtʲɪ]「投擲」, *mẽta* [2ˈmʲæːtɐ]「（他）投擲」, *mẽtė* [2ˈmʲæːtʲeː]「（他）投擲了」在**虛擬語氣**（tariamoji nuosaka）形式 *mèstų* [ˈmʲɛstuː] 中發出輔音 *s* `[s]`，而在**過去反覆時**（būtasis dažninis laikas）形式 *mèsdavo* [ˈmʲɛzdɐʋoː] 中則發出 *z* `[z]`。因此，**音位**（fonema）/s/ 由兩個輔音代表：`[s]` 和 `[z]`。第二個輔音 `[z]` 是因為輔音 `[s]` 在濁音 `[d]` 前發生了**濁化**（suskardėjimas）而出現的。
共時過程包含兩個相同語音的**邊界**（sandūra）、**語調短語**（frazė）中（詞內與詞間）的語音順應，以及詞首和詞尾規律。

**相同語音邊界（Vienodų garsų sandūra）**——指的是相同或發音位置極為相近的語音相遇。當一個語素以某個語音結束，而另一個語素以同一個語音開始時，就會產生邊界。
屬於不同音節的兩個相鄰元音稱為**元音間隙**（hiatas）（或 sambalsis）；屬於元音間隙的其中一個元音可以作為二合音核的組成部分。在立陶宛語本源詞中，元音間隙出現在前綴或**反身詞綴**（sangrąžos dalelytė）與詞根之間，以及複合詞的兩個詞根之間（第一個元音通常是 *a*，作為**連接元音**（jungiamasis balsis）），例如：*pa-aštrė́ti* [pɐ.ɐ1ˈʃʲtʲrʲeːtʲɪ]「變鋒利（不定式）」, *susi̇̀-ima* [sʊˈsʲɪ.ɪmɐ]「（他）自我控制」, *po-odi̇̀nis* [poː.oːˈdʲɪnʲɪs]「皮下的」, *su-ur̃gzti* [sʊ.2ˈʊrʲˑɡjsʲtʲɪ]「咆哮（不定式）」, *ne-eiliúotas* [nʲɛ.ɛɪ1ˈlʲu̟ɔtɐs]「非押韻的」；*auksa-ãkė* [ɒʊksɐ.2ˈɑːkʲeː]「草蛉（脈翅目昆蟲）」, *dvylika-aũkštis* [dʲʋʲiːlʲɪkɐ.2ˈɒʊˑkjʃʲtʲɪ]「十二層的」, *dvi-ir̃klis* [dʲʋʲɪ.2ˈɪrʲkjlʲɪs]「雙槳的」。不相同的元音也會相遇，例如：*pa-ū̃žti* [pɐ.2ˈuːʃʲtʲɪ]「呼嘯一陣（不定式）」, *ne-atėjo* [nʲɛ.ɐ2ˈtʲeːjo̟ː]「（他）沒來」, *nu-ãlinti* [nʊ.2ˈɑːlʲɪnʲtʲɪ]「使疲憊（不定式）」, *pri-ar̃tina* [pʲrʲɪ.2ˈɐrʲˑtʲɪnɐ]「（他）使靠近」, *pó-elgis* [1ˈpoː.ɛlʲɡʲɪs]「行為」, *su-úodė* [sʊ.1ˈuɔdʲeː]「（他）嗅出」；*tarpù-akis* [tɐrˈpʊ.ɐkʲɪs]「雙眼中間」, *tri-ei̇̃lis* [tʲrʲɪ.2ˈɛɪˑlʲɪs]「三行的」。兩個元音之間的音節界限在**轉寫**（transkribavimas）中用句點（.）標記。

元音間隙在外來詞的**詞幹**（kamienas）中很常見。在這裡它通常使用在前綴之後與後綴（結尾）之前。例如：*koordinãtės* [kɔ.ɔrʲdʲɪ2ˈnɑːtʲeːs]「座標」, *neoromantizmas* [nʲɛ.ɔrɔmɐnʲˈtʲɪzmɐs]「新浪漫主義」, *triacetãtas* [tʲrʲɪ.ɐʦʲɛ2ˈtɑːtɐs]「三醋酸纖維」, *polietilènas* [pɔlʲɪ.ɛtʲɪˈlʲɛnɐs]「聚乙烯」, *vãkuumas* [2ˈʋɑːkʊ.ʊmɐs]「真空」, *maèstras* [mɐ.ˈɛstrɐs]「大師」, *chaosas* [xɐ.ˈɔsɐs]「混沌」, *aū́las* [ɐ.1ˈuːlɐs]「阿吾勒（高加索山村）」, *teãtras* [tʲɛ.2ˈɑːtrɐs]「劇院」, *eksploatúoti* [ɛksplɔ.ɐ1ˈtuɔtʲɪ]「剝削（不定式）」, *poemà* [pɔ.ɛˈmɐ]「長詩」, *Bornèo* [bɔrʲˈnʲɛ.ɔ]「婆羅洲」, *individualùs* [ɪnʲdʲɪʋʲɪdʊ.ɐˈlʊs]「個人的」, *guãšas* [ɡʊ.2ˈɑːʃɐs]「水粉畫」。在立陶宛語中，可**變格**（linksniuotė）詞彙的詞幹與詞尾相接處不允許出現元音間隙——會插入 `[j]`。例如，名字 *Maria*、*Matias* 讀作 [mɐrjɪˈjɛ] 與 [2ˈmɑːtʲɪjɛs]。

兩個詞相接時元音相聚也是可能的，例如：*nė ė́dė* [nʲeː‿1ˈeːdʲeː]「連吃都沒吃」, *po óda* [poː‿2ˈoːdɐ]「在皮下」, *su ùžrašu* [sʊ‿ˈʊʒrɐʃʊ]「帶著題詞」, *kadà atvažiúosi* [kɐˈdɐ‿ɐtʋɐ1ˈʒʲu̟ɔsʲɪ]「你什麼時候會開車來」。

元音是音節的基礎，因此在標準語中既不會收縮，也不會丟棄。可視為例外的標準語形式有如 *pasilsė́ti* [pɐsʲɪlʲ1ˈsʲeːtʲɪ], *pasi̇̀lsi* [pɐ1ˈsʲɪlʲsʲɪ], *pasilsė́jo* [pɐsʲɪlʲ1ˈsʲeːjo̟ː] < *pasiilsė́ti* [pɐsʲɪ.ɪlʲ1ˈsʲeːtʲɪ]「休息（不定式）」, *pasii̇̀lsi* [pɐsʲɪ.1ˈɪlʲsʲɪ]「（他）休息」, *pasiilsė́jo* [pɐsʲɪ.ɪlʲ1ˈsʲeːjo̟ː]「（他）休息了」；*nei̇̃na* [2ˈnʲɛɪˑnɐ] < *ne ei̇̃na*「（他）不走」；村莊名稱 *Pórijai* [1ˈpoːrʲɪjɛɪ]（由河流名 *Orijà* [oːrʲɪˈjɛ] 加上 *po-* (*pa-*) 派生而來）以及 *Užúožeriai* [ʊ1ˈʒuɔʒʲɛrʲɛɪ]（由**名詞**（daiktavardis）*ẽžeras*「湖泊」的方言形式 *ãžeras* 加上 *užu-* 派生而來）。後面的例子證明了過去發生的語音現象。

兩個相鄰的同發音位置輔音稱為**重複輔音**（geminata）。它通常在詞綴與詞根之間、複合詞的兩個詞根之間，以及非獨立詞（附著詞）與獨立詞之間形成。相鄰的兩個輔音通常只發出第一個，而在書寫時適用形態學拼寫原則——拼寫必須顯示出語素的界限與派生特徵，例如：*pér-rašymas* [1ˈpʲæːrɐʃʲiːmɐs]「重寫」, *iš-šókti* [ɪ1ˈʃoːktʲɪ]「跳出（不定式）」, *už-žymė́jo* [ʊʒʲiː1ˈmʲeːjo̟ː]「（他）標記了」；*púod-dangtis* [1ˈpuɔdɐŋktʲɪs]「鍋蓋」, *dóbil-laukis* [1ˈdoːbʲɪlɒʊkʲɪs]「三葉草地」, *šėm-mar̃gis* [ʃʲeː2ˈmɐrʲˑɡʲɪs]「灰白斑駁的」, *vanden-nešỹs* [ʋɐnʲdʲɛnʲɛ2ˈʃʲiːs]「挑水人」, *pus-septiñto* [pʊsʲɛpʲ2ˈtʲɪnˑtoː]「六點半」, *dešimttūkstanti̇̀nė* [dʲɛʃʲɪmtuːkstɐnʲˈtʲɪnʲeː]「一萬的」, *kiek-kar̃t* [kʲiɛ2ˈkɐrˑth]「多少次」, *taip pàt* [tɐɪ‿ˈpɐth]「同樣」。在如 *užšókti* [ʊ1ˈʃoːkjtʲɪ]「跳上（不定式）」, *išžar̃sto* [ɪ2ˈʒɐrˑstoː]「（他）撒開」, *žỹgkelis* [2ˈʒʲiːkʲɛlʲɪs]「行軍路線」, *rýtdiena* [1ˈrʲiːdʲiɛnɐ]「明天」這樣的例子中，相鄰的輔音變得相同——發生清化或濁化（且通常會融合——消除重複輔音，發生**去重複輔音化**（degeminacija）），這是由於正在發生的**同化**（asimiliacija）：*žš > šš (> š)*, *šž > žž (> ž)*, *gk > kk (> k)*, *td > dd (> d)*。

重複輔音曾經存在於詞根與某些後綴的交界處，但現代拼寫並未證明已發生的收縮。遵循語音拼寫原則，這些動詞形式寫為：
*   不定式，例如：*šniõkšti* [2ˈʃʲnʲo̟ːkjʃʲtʲɪ] < *šniõkšt-ti*「嘶嘶作響（不定式）」（比較 *šniõkštė* [2ˈʃʲnʲo̟ːkjʃʲtʲeː]「（他）嘶嘶作響了」）, *švir̃kšti* [2ˈʃʲʋʲɪrʲˑkjʃʲtʲɪ] < *švir̃kšt-ti*「噴射（不定式）」（比較 *švir̃kštė* [2ˈʃʲʋʲɪrʲˑkjʃʲtʲeː]「（他）噴射了」）；
*   帶有後綴 *-sta* 的**現在時**（esamasis laikas），例如：*vė́sta* [1ˈʋʲeːstɐ] < *vė́s-sta*「（他）變涼」（比較 *vėso* [1ˈʋʲeːsoː]「（他）變涼了」）, *pailstù* [pɐ.ɪlˈstʊ] < *pails-stù*「我感到疲倦」（比較 *pail̃so* [pɐ.2ˈɪlˑsoː]「（他）感到疲倦了」）；
*   **將來時**（būsimasis laikas），例如：*dùs* [ˈdʊs] < *dùs-s*「（他）將會窒息」（比較 *dùsti* [ˈdʊsʲtʲɪ]「窒息（不定式）」）, *liẽsiu* [2ˈlʲiɛsʲʊ̟] < *liẽs-siu*「我將會碰觸」（比較 *liẽsti* [2ˈlʲiɛsʲtʲɪ]「碰觸（不定式）」）；
*   **祈使語氣**（liepiamoji nuosaka），例如：*lėk* [2ˈlʲeːkh] < *lėk-k(i)*「飛吧！」（比較 *lėkti* [2ˈlʲeːkjtʲɪ]「飛（不定式）」）, *juõkis* [2ˈju̟ɔkʲɪs] < *juõk-kis*「笑吧！」（比較 *juõktis* [2ˈju̟ɔkjtʲɪs]「笑（不定式）」）。
派生形式和後綴派生詞的拼寫與不定式和現在時形式相同，即寫一個輔音，例如：*vė́stantis* [1ˈʋʲeːstɐnʲtʲɪs]「變涼的（現在時主動分詞）」, *švir̃kšdavo* [2ˈʃʲʋʲɪrʲˑɡʒdɐʋoː]「（他）以前常噴射」, *švirkšlỹs* [ʃʲʋʲɪrʲkj2ˈʃʲlʲiːs]「注射器」（從 *švir̃kš-ti* 派生出的 *-lys* 詞）。

在立陶宛語中已根深蒂固的外來詞不具有重複輔音。去重複輔音化也顯示在拼寫中，例如：*klãsė* [2ˈklɑːsʲeː]「階級、班級」（比較：拉丁語 classis，英語 class，德語 Klasse，法語 classe，俄語 класc）, *apeti̇̀tas* [ɐpʲɛˈtʲɪtɐs]「食慾」（比較拉丁語 appetitus，英語 appetite，德語 Appetit，法語 Appétit，俄語 appetit）, *Tãlinas* [2ˈtɑːlʲɪnɐs]「塔林」（愛沙尼亞語 Tallinn）。只有在少數構詞較為清晰的國際詞彙中會寫兩個輔音，例如：*superregenerãtorius* [sʊpʲɛrʲɛɡʲɛnʲɛ2ˈrɑːtɔrʲʊ̟s]「超再生接收機」, *siurreali̇̀zmas* [sʲʊ̟rʲɛ.ɐˈlʲɪzmɐs]「超現實主義」, *grammolèkulė* [ɡrɐmɔˈlʲɛkʊlʲeː]「克分子」, *Transsi̇̀biras* [trɐnʲˈsʲɪbʲɪrɐs]「西伯利亞橫貫的」。

當說話緩慢、清晰、帶有強調時，並不總是進行去重複輔音化：要麼剩下的輔音發音較長，要麼兩個輔音都發出來。由於特殊的發音方式，人們尤其傾向保留 *rr* 重複輔音，例如：*keturrãtis* [kʲɛtʊr(+)2ˈrɑːtʲɪs]「四輪車」, *kontrrevoliùcija* [kɔntr(+)rʲɛʋɔˈlʲʊ̟ʦʲɪjɛ]「反革命」。不消除重複輔音的發音（在語素邊界處表現為開放邊界）是一種次要的發音變體。

另一個類似於去重複輔音化的現象是：當輔音 `[p]`, `[b]` 與 `[pʲ]`, `[bʲ]` 作為三個或四個輔音群中的第二個成員時被省略。輔音 `[p]`, `[b]` 與 `[pʲ]`, `[bʲ]` 在 `[m]`, `[mʲ]` 之後，且在 `[s]`, `[sʲ]`, `[t]`, `[tʲ]`, `[d]`, `[dj]` 之前時可以不發音，例如：*klim̃psta* [2ˈkʲlʲɪmˑpstɐ] / [2ˈkʲlʲɪmˑstɐ]「（他）陷進去」, *į̇̃temptas* [2ˈiːtʲɛmptɐs] / [2ˈiːtʲɛmtɐs]「緊張的」, *zvim̃bti* [2ˈzʲʋʲɪmʲˑpʲtʲɪ] / [2ˈzʲʋʲɪmʲˑtʲɪ]「嗡嗡作響（不定式）」, *skam̃btelėti* [2ˈskɐmʲˑpʲtʲɛlʲeːtʲɪ] / [2ˈskɐmʲˑtʲɛlʲeːtʲɪ]「輕響一下（不定式）」, *pampsiùkas* [pɐmpˈsʲʊ̟kɐs] / [pɐmˈsʲʊ̟kɐs]「（人名或小東西，方言）」, *klumpdirbỹs* [klʊmʲbʲdʲɪrʲ2ˈbʲiːs] / [klʊmʲdʲɪrʲ2ˈbʲiːs]「木鞋匠」（DLKG: 31）。當兩個發音位置相同（**雙唇音**（abilūpis priebalsis））的輔音相鄰時，發第二個音會很不方便，因此將其丟棄。

**語調短語中的語音順應（Garsų derinimas frazėje）**——指的是兩個相鄰且互相影響的語音在發音位置上的調整。當發生明顯或至少可察覺的變化時，它們的協同發音特徵由**順應**（akomodacija）和同化現象來界定。

**順應**（akomodacija）（拉丁語 accomodatio「適應」）——這是一種局部的、不會引起音位交替（**中和**（neutralizacija））的相鄰語音調整。這個過程的結果產生了同一個音位的組合變體（**音位變體**（alofonas / fonemos variantas）），例如，音位 /ɐ/ 在雙唇輔音之間的圓唇化變體出現在詞語 *pàbara* [ˈpɐbɐrɐ]「（他）責罵」中，而在**鼻音**（nosinis priebalsis）之間的鼻化變體則出現在詞語 *ramùnė* [rɐˈmʊnʲeː]「洋甘菊」中。由順應引起的語音變化在書寫中不予表達。
較明顯的順應情況有：**前化**（priešakėjimas）與**後化**（užpakalėjimas）、**開音化與閉音化**（atvirėjimas / uždarėjimas）、**圓唇化**（lūpinimas / labializacija）、**鼻化**（nosinimas / nazalizacija）、**軟齶化**（gomurinimas）、塞音爆破位置的改變、**送氣**（aspiracija）。

**前化**（priešakėjimas）是指在發**後元音**（užpakalinės eilės balsis）`[ɐ]`, `[ɑː]`, `[ɔ]`, `[oː]`, `[ʊ]`, `[uː]` 時，舌頭稍微向前移動。比較詞彙 *àkti* [ˈɐkjtʲɪ]「失明（不定式）」與 *làkti* [ˈlɐkjtʲɪ]「舔飲（不定式）」；*gãro* [2ˈɡɑːroː]「蒸汽的（屬格）」與 *dãro* [2ˈdɑːroː]「（他）做」中第一個音節的元音音質。在後一組動詞中，`[ɐ]` 和 `[ɑː]` 更靠前，因為它們跟在**舌前齒音**（liežuvio priešakinis dantinis）輔音 `[l]`, `[d]` 之後。在**舌後軟齶音**（liežuvio užpakalinis gomurio užpakalinis）輔音 `[k]`, `[ɡ]`, `[x]`, `[ɣ]` 之後，特別是在這類輔音之間，元音 `[ɐ]`, `[ɑː]` 會變得更靠後，例如：*kalvà* [kɐlˈʋɐ]「山丘」, *galvà* [ɡɐlˈʋɐ]「頭」, *chalvà* [xɐlˈʋɐ]「哈爾瓦（甜點）」（比較 *spalvà* [spɐlˈʋɐ]「顏色」）, *hãlė* [2ˈɣɑːlʲeː]「大廳」（比較 *sãlė* [2ˈsɑːlʲeː]「廳」）；*kaktà* [kɐkˈtɐ]「額頭」（比較 *staktà* [stɐkˈtɐ]「門框」, *rastà* [rɐsˈtɐ]「被找到的（陰性）」）, *chakãsai* [xɐ2ˈkɑːsɐɪ]「哈卡斯人（西伯利亞民族）」（比較 *ananãsai* [ɐnɐ2ˈnɑːsɐɪ]「鳳梨」）。
元音在**軟輔音**（minkštasis priebalsis）之後會極度前化；在轉寫中，該現象用記號「 ̟」標示在元音字母下方，例如：*šiõ* [2ˈʃʲo̟ː]「這的（屬格）」, *siū́las* [1ˈsʲu̟ːlɐs]「線」, *važiuõs* [ʋɐ2ˈʒʲu̟ɔs]「（他）將會乘車」。**低元音**（žemutinio pakilimo balsis）`[ɐ]`, `[ɑː]` 在軟輔音之後極度前化並變為**前元音**（priešakinės eilės balsis）`[ɛ]`, `[æː]`，例如：*žalià* [ʒɐˈlʲɛ]「綠色的（陰性）」, *žaliãvo* [ʒɐ2ˈlʲæːʋoː]「（他）呈現綠色」, *kurią̃* [kʊ2ˈrʲæː]「哪一個（陰性賓格）」（發生了元音音位的交替，即中和）。

**開音化**（atvirėjimas）是指以更張開的嘴部發出前元音。在標準語中，尤其明顯的是元音 `[æː]` 位於**硬輔音**（kietasis priebalsis）之前、後元音之前以及詞尾時的音質變化，比較以下詞語的發音：*nẽša* [2ˈnʲæːʃɐ]「（他）攜帶」, *nešù* [nʲɛˈʃʊ]「我攜帶」, *be ãbejo* [bʲɛ‿2ˈɑːbʲɛjo̟ː]「毫無疑問」, *darbè* [dɐrʲˈbʲɛ]「在工作中」。
在軟輔音與隨後的非低元音之前，前元音會更閉合（更狹窄），例如：*neši̇̀* [nʲɛˈʃʲɪ]「你攜帶」, *be eilės* [bʲɛ‿ɛɪ2ˈlʲeːs]「不按順序」。**重音2型（抑揚音）**（tvirtagalė priegaidė）以及無重音雙元音 *ai* `[ɐɪ]`、*ei* `[ɛɪ]` 的第一個組成部分也會在音質上發生改變，例如：*lai̇̃kas* [2ˈlɐɪˑkɐs]「時間」, *keistà* [kʲɛɪˈstɐ]「奇怪的」, *rei̇̃das* [2ˈrʲɛɪˑdɐs]「突襲」。閉合性是由於舌頭在發下一個音時向硬齶抬起所造成的。

**圓唇化**（lūpinimas / labializacija）——**唇音**（lūpinis priebalsis）的影響，即發音時將嘴唇稍微弄圓並向前突出。夾在**圓唇元音**（lūpinis balsis）之間或之前的輔音最容易被圓唇化。在轉寫中，強烈的影響可以用符號「w」來標記，例如：*óro* [1ˈoːrwoː]「空氣的（屬格）」, *rùdo* [ˈrʊdwoː]「棕色的（屬格）」, *po lóva* [poː‿1ˈlwoːʋɐ]「在床下」；*kótas* [1ˈkwoːtɐs]「柄」, *šių̃* [2ˈʃʲwu̟ː]「這些的（屬格）」, *ir rū̃kas* [ɪ‿2ˈrwuːkɐs]「還有霧」。輔音 `[ʋ]`, `[ʋʲ]`, `[f]`, `[fʲ]` 在圓唇元音之前或之間時，經常會變成雙唇音。它們可以相應地標記為 `[β]`, `[βʲ]` 與 `[ɸ]`, `[ɸʲ]`，例如：*bùvo* [ˈbʊβoː]「（他）曾經是」, *sòfos* [ˈsɔɸoːs]「沙發的（屬格）」。抑揚音及無重音雙元音 *au* `[ɒʊ]` 和 *eu* `[ɛʊ]` 的第一個成分會適應第二個成分（圓唇的 *u*），例如：*laũkas* [2ˈlɒʊˑkɐs]「田野」, *Europà* [ɛʊrɔˈpɐ]「歐洲」。
在圓唇元音旁邊或夾在雙唇輔音 `[p]`, `[pʲ]`, `[b]`, `[bʲ]` 之間的**非圓唇元音**（nelūpinis balsis）其圓唇化程度較小，例如：*sùošė* [ˈsʊ.oːʃʲeː]「（他）喧鬧了」, *paū̃š* [pɐ.2ˈuːʃ]「（他）將會呼嘯」；*pàbara* [ˈpɐbɐrɐ]「（他）責罵」, *Bãbtai* [2ˈbɑːptɐɪ]「巴布泰（地名）」。

**鼻化**（nosinimas / nazalizacija）——即鼻輔音 `[m]`, `[mʲ]`, `[n]`, `[nʲ]` 對元音的影響，發音時一部分氣流從鼻腔呼出。夾在鼻輔音之間的短元音鼻化最為明顯，例如：*ramùnė* [rɐˈmʊnʲeː]「洋甘菊」, *namai̇̃* [nɐ2ˈmɐɪˑ]「家」, *ten iñkštė* [tʲɛnʲ‿2ˈɪŋʲˑkʲʃʲtʲeː]「那裡發出呻吟聲」。長元音的鼻化音色較淡，比較：*pirmū̃nė* [pʲɪr2ˈmuːnʲeː]「領先者（陰性）」, *nãmo* [2ˈnɑːmoː]「回家」，因為在發長音時，軟齶抬得較高，關閉了鼻腔的開口。鼻化可以用符號「 ̃」標記，例如：*ramùnė* [rɐˈmʊ͂nʲeː], *namai̇̃* [nɐ͂2ˈmɐɪˑ]。

**軟齶化**（gomurinimas）——作為**混合二合音核**（mišrusis dvigarsis）第二成分的輔音 `[n]`, `[nʲ]` 發音位置發生改變，變為軟齶音 `[ŋ]`, `[ŋʲ]`。通常這些輔音在牙齒附近發音（屬於舌前齒音），但在舌後軟齶音 `[k]`, `[ɡ]`, `[x]` 和**舌中硬齶音**（liežuvio vidurinis gomurio vidurinis）`[kʲ]`, `[ɡʲ]`, `[xʲ]` 之前，舌頭會退縮到口腔後部——相鄰輔音的發音位置變得一致。例如：*liñko* [2ˈlʲɪŋˑkoː]「（他）彎腰了」, *angà* [ɐŋˈɡɐ]「開口」, *bronchi̇̀tas* [brɔŋʲˈxʲɪtɐs]「支氣管炎」。

爆破位置的改變常見於閉塞型塞音 `[p]`, `[pʲ]`, `[b]`, `[bʲ]`, `[t]`, `[tʲ]`, `[d]`, `[dʲ]` 與響輔音相遇時。當雙唇音 `[p]`, `[pʲ]`, `[b]`, `[bʲ]` 出現在發音位置相同的 `[m]`, `[mʲ]` 之前，而舌前齒音 `[t]`, `[tʲ]`, `[d]`, `[dʲ]` 出現在 `[n]`, `[nʲ]` 之前時，氣流會在咽部爆破。咽音爆破（或喉音爆破）的例子如：*ãpmaudas* [2ˈɑːpmɒʊdɐs]「怨恨」, *stabmeldỹs* [stɐbʲmʲɛlʲ2ˈdʲiːs]「偶像崇拜者」, *àtneša* [ˈɐtʲnʲɛʃɐ]「（他）帶來」, *liū̃dnas* [2ˈlʲu̟ːdnɐs]「悲傷的」。當舌前齒音 `[t]`, `[tʲ]`, `[d]`, `[dʲ]` 與 `[l]`, `[lʲ]` 相遇時，氣流會伴隨強烈的爆破從舌頭兩側衝出，例如：*ãtlasas* [2ˈɑːtlɐsɐs]「阿特拉斯（地圖集）」, *riẽdlentė* [2ˈrʲiɛdʲlʲɛnʲtʲeː]「滑板」。這種爆破稱為邊音爆破。

**送氣**（aspiracija）——在發位於詞尾、停頓之前的清音 `[p]`, `[t]`, `[k]` 時額外噴出的一股氣流。送氣用符號「ʰ」標記，例如：*savai̇̃p* [sɐ2ˈʋɐɪˑpʰ]「用自己的方式」, *visàd* [ʋʲɪˈsɐtʰ]「總是」, *ki̇̀tąsyk* [ˈkʲɪtɑːsʲiːkʰ]「下一次」。

語音可以同時在多個方面進行適應。例如，在詞語 *glotnùs* [ɡloːˈtnʊs]「光滑的」中，輔音 `[tn]` 既在圓唇元音之間發生了圓唇化，也經歷了咽音爆破。

**同化**（asimiliacija）（拉丁語 assimilatio「變得相似」）——這種現象中，相鄰或距離相近的語音發生變化，使得音位的**區別性特徵**（distinktyviniai požymiai）改變——也就是發生了音位交替，例如，*atgãvo* [ɐd2ˈɡɑːʋoː]「（他）恢復了」。可以看到，音位 /ɡ/ 的濁音區別性特徵轉移到了另一個音位——清音的 /t/ 上，因此這個音位變成了濁音的 /d/。同化在立陶宛標準語中，在輔音的發音上表現得更一致。元音的同化情況在方言中較為豐富。
同化分為幾種：
*   根據同化程度：**局部同化**（dalinė asimiliacija）與**完全同化**（visiškoji asimiliacija），
*   根據影響方向：**順行同化**（progresyvioji asimiliacija）與**逆行同化**（regresyvioji asimiliacija），
*   根據相互影響語音的距離：**接觸同化**（artimoji asimiliacija）與**遠距同化**（tolimoji asimiliacija）。

在局部同化中，輔音僅部分與相鄰的輔音適應；兩個輔音都會發音（AB → CB），例如：*di̇̀rbti* [1ˈdʲɪrʲpʲtʲɪ]「工作（不定式）」, *strazdžiùkas* [strɐʒʲˈʤʲʊ̟kɐs]「小鶇（鳥名）」。
在完全同化中，語音完全協調、變得相同。相鄰的兩個相同輔音只發一個音，因為形成的重複輔音被消除了（AB → B），例如：*pérraša* [1ˈpʲæːrɐʃɐ]「謄本」, *tri̇̀ssyk* [ˈtʲrʲɪsʲiːkh]「三次」。

順行同化表現在前面的語音影響後面的語音（A → B）。順行同化在標準語中的例子不多。最明顯的例子是將來時**第三人稱**（trečias asmuo）形式：*peš-ti* + *s* > *pešš* > *peš* [ˈpɛʃʲ]「（他）將會拔」（由於詞根 `[ʃ]` 的影響，後綴 `[s]` 變成了 `[ʃ]`）。

逆行同化是指後面的語音影響前面的語音（A ← B），例如：*di̇̀rbti* [1ˈdʲɪrʲpʲtʲɪ]「工作（不定式）」, *išsamiai̇̃* [ɪsɐ2ˈmʲɛɪˑ]「詳盡地」。絕大多數的同化例子都證明了逆行的語音適應方向。

接觸同化包含相鄰的語音（A → B, A ← B），例如：*di̇̀rbti* [1ˈdʲɪrʲpʲtʲɪ]「工作（不定式）」, *pérraša* [1ˈpʲæːrɐʃɐ]「謄本」。

在發生遠距同化時，有一定距離的語音變得相似（相互影響的語音之間至少夾雜了一個不參與同化的語音；A...B → B...B）。遠距同化的例子通常是偶然的口誤，例如 *apši̇̀šlavė* [ɐpʲˈʃʲɪʃlɐʋʲeː]「（他）稍微打掃了自己」（由於詞根第一個輔音 `[ʃ]` 的影響，反身詞綴 `[s]` 變成了 `[ʃ]`）。一些專有名詞顯示了過去發生的同化現象，例如姓氏 *Šli̇̀nkšė* [1ˈʃʲlʲɪŋjkjʃʲeː] < *Slinkšė*（：*sliñk-ti* + *-šė*）。

局部與完全同化有多種表現形式。區分以下同化方式：
*   根據聲帶活動：濁化與清化，
*   根據發音位置——發音位置的改變，
*   根據**軟化性**（palatališkumas）；
*   混合型同化。

根據聲帶活動的局部同化。相鄰的清音與濁音會根據聲帶活動進行適應——前者適應後者，例如：*láukdavo* [1ˈlɑˑʊɡdɐʋoː]「（他）以前常等」（清音 /k/ 由於濁音 /d/ 的影響變成了濁音 /ɡ/——發生了濁化）, *dègs* [ˈdʲɛks]「（他）將會燃燒」（濁音 /ɡ/ 由於清音 /s/ 的影響變成了清音 /k/——發生了清化）。除了響輔音之外，所有輔音都可以發生濁化與清化；響輔音總是濁的，相鄰的清輔音不會影響其濁音特性，比較：*dãgtį* [2ˈdɑːkjtjiː]「燈芯（賓格）」與 *dañtį* [2ˈdɐnjˑtʲˈiː]「牙齒（賓格）」, *dùbti* [ˈdʊpʲtʲɪ]「凹陷（不定式）」與 *dùmti* [1ˈdʊmʲtʲɪ]「吹（不定式）」, *ki̇̀bti* [ˈkʲɪpʲtʲɪ]「黏附（不定式）」與 *ki̇̀mti* [1ˈkʲɪmʲtʲɪ]「變得沙啞（不定式）」。

根據聲帶活動的完全同化。當成對的濁音和清音相遇時，區別性特徵會互相協調，在兩個相同輔音的位置只發第二個音，例如：*ãtdaras* [2ˈɑːdɐrɐs]「敞開的」（清音 /t/ 由於濁音 /d/ 的影響發生濁化，變為 /t/；重複輔音 /tt/ 被去重複輔音化，發出一個 /t/）；*užšãlo* [ʊ2ˈʃɑːloː]「（他）結冰了」（濁音 /ʒ/ 受到清音 /ʃ/ 影響，清化為 /ʃ/；重複輔音 /ʃʃ/ 被去重複輔音化，只發出一個 /ʃ/）。

根據發音位置的局部同化在以下兩種情況下表現得更為一致（當未實現開放邊界時）：
1) 舌前齒音摩擦音 /sʲ/ 和 /zʲ/ 在**舌前齒齦音**（liežuvio priešakinis alveolinis）/ʧʲ/ 和 /ʤʲ/ 之前會變成齒齦音 /ʃʲ/、/ʒʲ/，例如：*mókesčiai* [1ˈmoːkʲɛʃʲʧʲɛɪ]「稅」（齒音 /sʲ/ 適應了後面輔音 /ʧʲ/ 的齒齦發音位置——變成了齒齦音 /ʃʲ/）；*pãvyzdžiui* [2ˈpɑːʋʲiːʒʲʤʲʊ̟ɪ]「例如」（齒音 /zʲ/ 適應了後面輔音 /ʤʲ/ 的齒齦發音位置——變成了齒齦音 /ʒʲ/）；
2) 舌前齒音響輔音 /n/、/nʲ/ 在雙唇音 /p/、/b/ 和 /pʲ/、/bʲ/ 之前會變成雙唇音 /m/、/mʲ/，例如：*sénbernis* [1ˈsʲæˑnʲbʲɛrʲnʲɪs]「老光棍」, *raudonplaũkė* [rɒʊdoːn2ˈplɒʊˑkʲeː]「紅髮女子」（齒音 /nʲ/、/n/ 在發音上適應了後面的雙唇輔音 /bʲ/、/p/，變成了與之相同的雙唇音 /mʲ/、/m/）。

根據發音位置的完全同化。當齒音與齒齦音摩擦音相遇時（通常為清音 /s/ 和 /ʃ/, /sʲ/ 和 /ʃʲ/），發音位置會協調，並且只發第二個輔音，例如，*pùsšimtis* [ˈpʊʃʲɪmʲtʲɪs]「五十」（齒輔音 /sʲ/ 適應了後面的齒齦音 /ʃʲ/，變成了相同的齒齦音 /ʃʲ/；相鄰的兩個相同輔音融合，只發出一個 /ʃʲ/）。

根據軟化性的同化——輔音在前元音之前的齶化（軟化）。在 `[æː]`, `[ɛ]`, `[eː]`, `[iː]`, `[ɪ]` 之前的輔音（書寫中用字母 *e, ę, ė, y, į, i* 標記）會被齶化，因為舌頭會接觸到硬齶邊緣。這種在前元音前的語音調整會引起硬輔音和軟輔音音位的交替，因此被視為一種同化現象，比較 *dedù* [dʲɛˈdʊ]「我放」與 *dedi̇̀* [dʲɛˈdʲɪ]「你放」, *laũkas* [2ˈlɒʊˑkɐs]「田野」與 *laukẽlis* [lɒʊ2ˈkʲæːlʲɪs]「小田野」, *mãto* [2ˈmɑːtoː]「（他）看見」與 *matýti* [mɐ1ˈtʲiːtʲɪ]「看見（不定式）」。

局部同化的混合型同化——當兩個輔音不僅在一個，而是在多個特徵上同時變得相似的現象，例如，*pùsdžiūvis* [ˈpʊʒʲʤʲu̟ːʋʲɪs]「半乾的」。輔音 /sʲ/ 根據聲帶活動是清音，根據發音位置是舌前齒音。後面的 /ʤʲ/ 是濁音和舌前齒齦輔音。因此 /sʲ/ 和 /ʤʲ/ 要在兩個特徵上進行協調。協調過程具體描述如下：清音 /sʲ/ 因濁音 /ʤʲ/ 的影響變成濁音 /zʲ/；齒音 /zʲ/ 因齒齦音 /ʤʲ/ 的影響變成 /ʒʲ/，所以不發 /sʲ/ 和 /ʤʲ/，而是發出在聲帶活動和發音位置上已協調一致的 /ʒʲ/ 和 /ʤʲ/。

在完全同化的混合型同化情況下，相鄰輔音在兩個或三個特徵上完全變得相似，並且在消除差異後，發生去重複輔音化——其中一個輔音不發音。這種同化的例子——名詞 *pùsžiemis* [ˈpʊʒʲiɛmʲɪs]「隆冬」。發生的協調過程具體描述如下：清音 /sʲ/ 在濁音 /ʒʲ/ 的影響下，變成齒齦音 /zʲ/；舌前齒音 /zʲ/ 在齒齦濁音 /ʒʲ/ 的影響下變為 /ʒʲ/；兩個相同的輔音 /ʒʲ/ 和 /ʒʲ/ 相遇，它們融合並只發出一個 /ʒʲ/。

上述討論的同化情況也會表現在詞語的邊界上。當非獨立詞（無重音）和獨立詞相接時，輔音的同化最為一致。在兩個獨立詞的組合中，語音也可以不改變，即同化是可選的（隨意的）。例如：
*iš gãlo* [ɪʒ‿2ˈɡɑːloː]「從頭」, *už pãstato* [ʊʃ‿2ˈpɑːstɐtoː]「在建築物後面」, *tiesiog pali̇̀k* [tʲiɛsʲo̟ːk‿pɐˈlʲɪkh] / *tiesióg pali̇̀k* [tʲiɛ1ˈsʲo̟ːkh pɐˈlʲɪkh]「直接留下」（根據聲帶活動的局部同化）；
*bet dùria* [bʲɛ‿ˈdʊrʲɛ]「但刺痛」, *lig kraštų̃* [lʲɪ‿krɐʃ2ˈtuː]「直到邊緣」, *bus žvalùs* [bʊ‿ʒʋɐˈlʊs] / *bùs žvalùs* [ˈbʊʒ ʒʋɐˈlʊs]「將會充滿活力」（根據聲帶活動的完全同化；最後一個例子中沒有發生去重複輔音化）；
*vis čáižo* [ʋʲɪʃ‿1ˈʧɑˑɪʒoː]「總是鞭打」, *ten pràmušė* [tʲɛm‿ˈprɐmʊʃʲeː] / *teñ pràmušė* [2ˈtʲɛnˑ ˈprɐmʊʃʲeː]「在那裡打穿了」（根據發音位置的局部同化）；
*nors šaũk* [noːr‿2ˈʃɒʊˑk]「即便呼喊」, *mi̇̀škas šlãma* [ˈmʲɪʃkɐʃ 2ˈʃlɑːmɐ]「森林沙沙作響」（根據發音位置的完全同化；最後一個例子中沒有發生去重複輔音化）；
*tik i̇̀šveda* [tʲɪˈkʲ‿ɪʃʲʋʲɛdɐ]「只帶出」, *ir senõliai* [ɪrʲ‿sʲɛ2ˈnoːlʲɛɪ]「還有老人們」, *taip išmókė* [tɐɪpʲ‿ɪʃ1ˈmoːkʲeː] / *tai̇̃p išmókė* [2ˈtɐɪˑph ɪʃ1ˈmoːkʲeː]「就這樣教會了」（根據軟化性的同化）；
*nes džiū̃va* [nʲɛʒʲ‿2ˈʤʲu̟ːʋɐ]「因為變乾」, *ims džèmo* [ɪmʒʲ‿ˈʤʲɛmoː] / *im̃s džèmo* [2ˈɪmˑs ˈʤʲɛmoː]「會拿果醬」（局部同化的混合型同化）；
*kas žãda* [kɐ‿2ˈʒɑːdɐ]「誰答應」, *bemaž stãčias* [bʲɛmɐ‿2ˈstɑːʧʲɛs]（更常讀作 [bʲɛmɐʃ‿2ˈstɑːʧʲɛs]）「幾乎筆直的」（前兩個例子是完全同化的混合型同化）。

**詞首與詞尾規律（Žodžių pradžios ir galo dėsniai）**顯示了在 `[iɛ]` 和停頓之前，響輔音 `[j]`, `[ʋ]` 以及成對的濁音與清音、硬輔音與軟輔音的使用情況。
在詞首，以及在前綴或反身詞綴之後、詞根 `[iɛ]` 之前，會發出附加的 `[j]`，例如：*i̇́etis* [1ˈjiɛtʲɪs]「長矛」, *ieti̇̀galis* [jiɛˈtʲɪɡɐlʲɪs]「矛頭」, *ievà* [jiɛˈʋɐ]「稠李」, *i̇́eško* [1ˈjiɛʃkoː]「（他）尋找」, *pãieškos* [2ˈpɑːjiɛʃkoːs]「搜索」, *sui̇́eškoti* [sʊjiɛ1ˈʃkoːtʲɪ]「尋獲（不定式）」。標記這個輔音的字母 *j* 在書寫中通常不寫出。例外只有幾個詞——*jiẽ*「他們」, *jiẽdu*「他們倆（陽性）」, *jiẽdvi*「她們倆（陰性）」, *jiẽvaras*「懸鈴木」, *Jiẽznas*「耶茲納斯（地名）」, *jiezni̇̀škis*「耶茲納斯人」, *Jiesià*「耶西亞（河流名）」。

在詞尾，換句話說在停頓之前，使用清音和硬輔音。濁輔音在這個位置會清化，例如：*visàd* [ʋʲɪˈsɐtʰ]「總是」, *daũg* [2ˈdɒʊˑkʰ]「多」，而軟輔音則會硬化，例如：*nèšti* [ˈnʲɛʃʲtʲɪ] > *nèšt* [ˈnʲɛʃtʰ]「攜帶（不定式縮略）」, *vėlei* [2ˈʋʲeːlʲɛɪ] > *vėl* [2ˈʋʲeːl]「再次」。
響輔音 `[j]` 和 `[ʋ]` 作為二合音核的尾部成分並在停頓前發音時，會變為元音 `[ʊ]`, `[ɪ]`，例如：*rytój* [rʲiː1ˈtoːɪ]「明天」（比較 *rytójus* [rʲiː1ˈtoːjʊ̟s]「明日」）, *Vi̇̀lniuj* [1ˈʋʲɪlʲnʲʊ̟i]「在維爾紐斯（短形式）」（比較 *Vi̇̀lniuje* [1ˈʋʲɪlʲnʲʊ̟jɛ]「在維爾紐斯」）；*sudiẽv* [sʊ2ˈdʲiɛʊ]「再見」（比較 *su Dievù* [sʊ‿ dʲiɛˈʋʊ]「與上帝同在」）。

非獨立詞（無重音）和獨立詞的邊界被視為同一個**語音詞**（fonetinis žodis）音節的邊界，因此在自然情況下，輔音在這裡也會相互協調——在元音或響輔音（`[m]`, `[mʲ]`, `[n]`, `[nʲ]`, `[l]`, `[lʲ]`, `[r]`, `[rʲ]`, `[j]`, `[ʋ]`, `[ʋʲ]`）之前發清輔音，例如：*kad atei̇̃tų* [kɐt‿ɐ2ˈtʲɛɪˑtuː]「如果（他）來」, *kad ji̇̀s* [kɐtʲ‿ˈjɪs]「如果他」。如果說話帶有強調、口齒清晰，也可以保留濁音：*kad atei̇̃tų* [kɐd‿ɐ2ˈtʲɛɪˑtuː], *kad ji̇̀s* [kɐdʲ‿ˈjɪs]。唯一一個**前置詞**（prielinksnis）*už* 在邊界處的輔音不會清化，例如：*už antrãdienį* [ʊʒ‿ɐn2ˈtrɑːdʲiɛnʲiː]「（感謝）星期二」, *už nãmo* [ʊʒ‿2ˈnɑːmoː]「在房子後面」。
