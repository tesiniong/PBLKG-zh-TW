# 語音在話語中的適應

極少有語音被孤立地發出。依次發出的語音相互適應，因為中樞神經系統控制下的發音器官是協同運作的。語音的相互適應、一個語音對另一個語音的影響稱為**協同發音**（koartikuliacija，源自拉丁文 co-「共同，一起」+ articulātio < articulo「清晰發音；分節」，Hardcastle、Hewlett 1999: 1）。協同發音研究的任務在於闡明大腦和中樞神經系統如何控制帶動發音器官的肌肉。語音學家關注協同發音所引發的語音變化——相互適應的語音或多或少發生改變，而改變後的語音往往更易於發音。

語音隨時間發生的變化分為兩類：**歷時性**（diachroniniai，歷史性）和**共時性**（sinchroniniai，同時性）。

## 歷時性語音過程

歷時性變化發生在一定時間跨度內，由此一個共時系統逐漸被另一個所取代。歷時性研究有助於理解語言和具體語言現象的本質，尤其是從現代立場解釋各種不一致之處和例外。

歷時性過程包括：元音的量的交替和質的交替，異化，塞擦化，異位，以及語音消失和添加現象（消除、首音省略、中音省略、末音省略、收縮、省略、同音節省略，以及增音和插入音）。

**元音的量的交替**（kiekybinė balsių kaita）是同族詞詞根或同一詞的不同詞形中，歷史上同一個短元音與長元音的交替現象，例如：

* \[ɪ]（< \*ĭ）：\[iː]（< \*ī）— di̇̀delis \[ˈdʲɪdʲɛlʲɪs]「大的（形）」：dỹdis \[²ˈdʲiːdʲɪs]「大小」；ski̇̀lo：skỹla
* \[ʊ]（< \*ŭ）：\[uː]（< \*ū）— mùša「打」：mū̃šis「打鬥」；dùsti「氣短」：dū̃sta
* \[ɛ]/\[æː]（< \*ĕ）：\[eː]（< \*ē）— vèžti「運」：vėžės（屬格複數）；geriù：gė́rė
* \[ɐ]/\[ɑː]（< \*ă）：\[oː]（< \*ā）— gražùs「美麗的」：grõžis「美」；varýti「趕」：vorà「排」

**元音的質的交替**（kokybinė balsių kaita）是同族詞詞根或同一詞的不同詞形中，不同質量的元音或二合音核的交替現象。基本規律是短元音 \*ĕ：\*ă 和長元音 \*ē：\*ā 的交替。考慮到歷時變化，交替可記為：\[ɛ]/\[æː]：\[ɐ]/\[ɑː]：\[eː]：\[oː]/\[uɔ]，例如：vèžti、vẽža：važiúoti、vãžis：vėžės；slegiù、slẽgia：slãktai：slė́gti：slogà、slúoksnis。

**異化**（disimiliacija）是兩個相同或相似輔音中的一個被近似發音的輔音所替換。例如，t/d 與 t/d 相遇時，在清輔音前變為 s，在濁輔音前變為 z：\*mèt-ti（比較 mẽt-ė）> mèsti \[ˈmʲɛsʲtʲɪ]；\*vai̇̃d-das（比較 vaid-éntis）> vai̇̃zdas \[²ˈʋɐɪˑzdɐs]。在方言中，遠距異化的例子也可見到。

**塞擦化**（afrikacija）是塞音變為塞擦音的現象。立陶宛語本土塞擦音 č（\[tʃʲ]）和 dž（\[dʒʲ]）來源於 \*tj、\*dj 的組合，例如：\*svetjuñ > svečių̃ \[sʲʋʲɛ²ˈʧʲuː]，\*gai̇̃djā > gai̇̃džio \[²ˈɡɐɪˑʤʲoː]。

**異位**（metatezė）是同根詞中輔音位置互換的現象。在塞音前出現 ks、kš、gz、gž 的組合，在元音前則出現 sk、šk、zg、žg，例如：pliẽksti \[²ˈpʲlʲiɛkjsʲtʲɪ]：pliẽskia \[²ˈpʲlʲiɛsʲkʲɛ]，réikšti \[¹ˈrʲæˑɪkjʃʲtʲɪ]：réiškia \[¹ˈrʲæˑɪʃʲkʲɛ]。

**消除**（eliminacija）是構詞時輔音被刪除的現象，常見的是在摩擦音 s 或某些響輔音之前的 t/d 被刪除：mąst-lùs > mąslùs \[mɑːˈslʊs]，smárd-vė > smárvė \[¹ˈsmɑˑrʲʋʲeː]。

**首音省略**（aferezė）是詞首語音的脫落：beždžiõnė \[bʲɛ²ˈʒʲʤʲo̟ːnʲeː]（源自白俄羅斯語），Zarasai̇̃ \[zɐrɐ²ˈsɐɪˑ] < \*Azarasai。

**中音省略**（sinkopė）是詞中非重讀短元音的脫落：šitai̇̃ > štai̇̃ \[²ˈʃtɐɪˑ]，kẽlinės > kélnės \[¹ˈkʲæˑlʲnʲeːs]，vi̇̀sakas > vi̇̀skas \[ˈʋʲɪskɐs]。

**末音省略**（apokopė）是詞末一個或幾個語音的脫落：nẽšas < nẽšasi，mẽdy，mẽdyj < mẽdyje，šiañdie < šiañdien。

**收縮**（kontrakcija）是兩個不同語音合併為一個的現象——元音（ne-yrà > nėrà \[nʲeːˈrɐ]）或輔音（mùš-s > mùš \[ˈmʊʃ]）均可收縮。

**省略**（elizija）是一個詞或語素末尾的元音在下一個詞以同一元音開頭時被刪除：nẽsti < ne ẽsti，nesù < ne esù，nei̇̃ti < ne ei̇̃ti。

**同音節省略**（haplologija）是兩個相同或相近音節中的一個被刪除：在立陶宛標準語中，有時會刪除後綴 -ininkas 派生詞中的音節 -ni-，如 maši̇̀ninkas < maši̇̀nininkas，šuliniñkas < šulininiñkas。

**增音**（protezė）是詞首添加非詞源性語音的現象：stvérti < tvérti「抓取」，skėsti < kėsti「展開」（增添了 s）；方言中可見詞首添加元音的例子：atkõčius「織工」、uzbõnas「水罐」。

**插入音**（epentezė）是詞中添加非詞源性語音的現象：žvaigždė \[ʒʋɐɪɡj²ˈʒʲdʲeː]（比較某些薩摩吉提亞方言中的 žvaizdė 和俄語 звезда）。

語音的變化受語言節奏性（韻律）、區別性（多樣性）和簡潔性的追求所驅動，換言之，語音的變化是為了簡化發音。大部分語音交替規律可藉現代語言的數據加以解釋，應用一定的語素音韻學規則即可。

## 共時性語音過程

共時性變化是現實時間中發生的、活躍的語音變化——從語言使用者的角度審視正在變化中的語言。例如，動詞 mèsti 虛擬式 mèstų \[ˈmʲɛstuː] 中發音位 /s/ 代表的是 \[s]，而過去常用時 mèsdavo \[ˈmʲɛzdɐʋoː] 中代表的是 \[z]——即音位 /s/ 由兩個輔音 \[s] 和 \[z] 代表，後者是 \[s] 在濁輔音 \[d] 之前濁化的結果。

共時性過程包括：相同語音的相遇、語調短語中的語音協調（詞內和詞間），以及詞首詞末的音韻規律。

### 相同語音的相遇

相同或發音非常相近的語音相遇時，可能出現以下情況：

**母音裂**（hiatas，sambalsas）是屬於不同音節的兩個相鄰元音的結合，其中一個也可以是二合音核的成分。在立陶宛語本土詞彙中，母音裂出現在前綴或反身小詞與詞根之間，以及複合詞的兩個詞根之間（第一個元音通常是 a 作為連接元音）：pa-aštrė́ti \[pɐ.ɐ¹ˈʃʲtʲrʲeːtʲɪ]，susi̇̀-ima \[sʊˈsʲɪ.ɪmɐ]，auksa-ãkė \[ɒʊksɐ.²ˈɑːkʲeː]。在外來詞（借詞）詞根中，母音裂也很常見：koordinãtės \[kɔ.ɔrʲdʲɪ²ˈnɑːtʲeːs]，vãkuumas \[²ˈʋɑːkʊ.ʊmɐs]，teãtras \[tʲɛ.²ˈɑːtrɐs]，poètas \[pɔ.ˈɛtɐs]。在詞與詞的交界處，相鄰元音也可能存在：nė ė́dė \[nʲeː‿¹ˈeːdʲeː]，po óda \[poː‿²ˈoːdɐ]。轉寫中，音節邊界以點（.）標記。

在立陶宛語可屈折詞的詞根與詞尾的交界處，母音裂是不允許的——必須插入 \[j]（如外來名字 Maria 和 Matias 發音為 \[mɐrjɪˈjɛ] 和 \[²ˈmɑːtʲɪjɛs]）。

**重複輔音**（geminatai）的相遇：如果兩個相同的輔音在語素交界處相遇，可簡化為一個（**去重複輔音**，degeminacija）：pérraša \[¹ˈpʲæːrɐʃɐ]，tri̇̀ssyk \[ˈtʲrʲɪsʲiːkh]。出於保持詞義清晰度的需要，有時傾向於保留 rr 重複輔音：keturrãtis \[kʲɛtʊr(+)²ˈrɑːtʲɪs]。此外，輔音組合中的 \[p]、\[b] 有時在 \[m] 之後、\[s]、\[t]、\[d] 等之前可不發音：klim̃psta \[²ˈkʲlʲɪmˑpstɐ] / \[²ˈkʲlʲɪmˑstɐ]，zvim̃bti \[²ˈzʲʋʲɪmʲˑpʲtʲɪ] / \[²ˈzʲʋʲɪmʲˑtʲɪ]。

### 語調短語中的語音協調

語調短語中的語音協調是指兩個相鄰、相互影響的語音的發音調整。其協同發音的特點以**順應**（akomodacija）和**同化**（asimiliacija）現象加以描述。

**順應**（akomodacija，源自拉丁文 accomodatio「適應」）是相鄰語音的局部調整，不引起音位交替（中和）。結果是同一音位的組合變體出現，例如：/ɐ/ 在雙唇輔音之間有圓唇化變體（pàbara \[ˈpɐbɐrɐ]），在鼻音輔音之間有鼻化變體（ramùnė \[rɐˈmʊnʲeː]）。由順應引起的語音變化在書寫中不體現。

主要的順應現象包括：

* **前化**（priešakėjimas）：後元音 \[ɐ]、\[ɑː]、\[ɔ]、\[oː]、\[ʊ]、\[uː] 在舌前齒音 \[l]、\[d] 等之後略向前移動，而在舌後軟齶音 \[k]、\[ɡ]、\[x]、\[ɣ] 之後則更向後移動。在軟輔音之後，後元音高度前化（在轉寫中以「̟」符號表示）：šiõ \[²ˈʃʲo̟ː]，siū́las \[¹ˈsʲu̟ːlɐs]。低元音 \[ɐ]、\[ɑː] 在軟輔音後前化尤為強烈，通常變為前元音 \[ɛ]、\[æː]（發生音位交替或中和）：žalià \[ʒɐˈlʲɛ]，žaliãvo \[ʒɑː²ˈlʲæːʋoː]。
* **後化**（užpakalėjimas）/ **開口化**（atvirėjimas）：前元音 \[æː] 在硬輔音、後元音或詞末位置之前變得更開口（更後化）：nẽša \[²ˈnʲæːʃɐ]，nešù \[nʲɛˈʃʊ]。在軟輔音和後續非低元音之前，前元音更閉口（更窄）：neši̇̀ \[nʲɛˈʃʲɪ]。
* **圓唇化**（lūpinimas，唇化，labializacija）：圓唇語音（圓唇元音和雙唇輔音）對相鄰語音的影響。輔音夾在圓唇元音之間或位於其前時圓唇化最強，可以符號「w」標記：óro \[¹ˈoːrwoː]，kótas \[¹ˈkwoːtɐs]。\[ʋ]、\[ʋʲ]、\[f]、\[fʲ] 在圓唇元音前或之間常變為雙唇音 \[β]、\[βʲ]、\[ɸ]、\[ɸʲ]：bùvo \[ˈbʊβoː]，sòfos \[ˈsɔɸoːs]。重音2型和非重讀雙元音 au、eu 的第一成分受第二成分（圓唇的 u）影響而圓唇化：laũkas \[²ˈlɒʊˑkɐs]，Europà \[ɛʊrɔˈpɐ]。
* **鼻化**（nosinimas，nazalizacija）：鼻輔音 \[m]、\[mʲ]、\[n]、\[nʲ] 對元音的影響，使部分氣流通過鼻腔。位於鼻輔音之間的短元音鼻化最強：ramùnė \[rɐˈmʊnʲeː]，namai̇̃ \[nɐ²ˈmɐɪˑ]。長元音的鼻化色彩較淡。鼻化以符號「̃」標記。
* **軟齶化**（gomurinimas）：構成組合型混合二合音核第二成分的輔音 \[n]、\[nʲ] 在舌後輔音 \[k]、\[ɡ]、\[x] 和舌中輔音 \[kʲ]、\[ɡʲ]、\[xʲ] 之前，發音部位後移，變為軟齶音 \[ŋ]、\[ŋʲ]：liñko \[²ˈlʲɪŋˑkoː]，angà \[ɐŋˈɡɐ]，bronchi̇̀tas \[brɔŋʲˈxʲɪtɐs]。
* **爆破位置的變化**：塞音 \[p]、\[pʲ]、\[b]、\[bʲ]、\[t]、\[tʲ]、\[d]、\[dʲ] 與響輔音相遇時，爆破點發生改變——雙唇音在 \[m]、\[mʲ] 之前，齒音在 \[n]、\[nʲ] 之前，氣流在咽部爆破（咽部爆破）：ãpmaudas \[²ˈɑːpmɒʊdɐs]，àtneša \[ˈɐtʲnʲɛʃɐ]；而 \[t]、\[tʲ]、\[d]、\[dʲ] 在 \[l]、\[lʲ] 之前，氣流從舌側強力突破（側向爆破）：ãtlasas \[²ˈɑːtlɐsɐs]，riẽdlentė \[²ˈrʲiɛdʲlʲɛnʲtʲeː]。
* **送氣**（aspiracija）：詞末清輔音 \[p]、\[t]、\[k] 在停頓前有附加的送氣，以「ʰ」標記：savai̇̃p \[sɐ²ˈʋɐɪˑpʰ]，visàd \[ʋʲɪˈsɐtʰ]。

**同化**（asimiliacija，源自拉丁文 assimilatio「趨同」）是相鄰或相近語音發生變化、改變音位的區別性特徵——即發生音位交替的現象：例如 atgãvo \[ɐd²ˈɡɑːʋoː] 中，音位 /ɡ/ 的濁音特徵傳遞給了清音 /t/，使其變為濁音 /d/。

同化的分類：

* **按趨同程度**：**局部同化**（dalinė，AB → CB，兩個輔音都發音）和**完全同化**（visiškoji，AB → B，兩個相同的輔音合為一個）
* **按影響方向**：**順行同化**（progresyvioji，前音影響後音，A → B）和**逆行同化**（regresyvioji，後音影響前音，A ← B）——絕大多數同化屬逆行同化
* **按距離**：**接觸同化**（artimoji/kontaktinė，A → B 或 A ← B，相鄰語音）和**遠距同化**（tolimoji/distancinė，A...B → B...B，中間隔有不參與同化的語音）

同化的主要類型：

**按聲帶活動的局部同化**：相鄰的清輔音和濁輔音按聲帶活動協調——前者適應後者，例如：láukdavo \[¹ˈlɑˑʊɡdɐʋoː]（清音 /k/ 因濁音 /d/ 而濁化），dègs \[ˈdʲɛks]（濁音 /ɡ/ 因清音 /s/ 而清化）。所有輔音（響輔音除外）均可清化或濁化。

**按聲帶活動的完全同化**：成對的濁音與清音相遇時，區別性特徵消除，兩個相同的輔音去重複化，只發出一個：ãtdaras \[²ˈɑːdɐrɐs]（/t/ + /d/ → /dd/ → /d/），užšãlo \[ʊ²ˈʃɑːloː]（/ʒ/ + /ʃ/ → /ʃʃ/ → /ʃ/）。

**按發音部位的局部同化**，主要有兩種情況：1）齒摩擦音 /sʲ/、/zʲ/ 在齒齦音 /ʧʲ/、/ʤʲ/ 之前變為 /ʃʲ/、/ʒʲ/：mókesčiai \[¹ˈmoːkʲɛʃʲʧʲɛɪ]，pãvyzdžiui \[²ˈpɑːʋʲiːʒʲʤʲʊ̟ɪ]；2）齒響輔音 /n/、/nʲ/ 在雙唇音 /p/、/b/、/pʲ/、/bʲ/ 之前變為 /m/、/mʲ/：sénbernis \[¹ˈsʲæˑmʲbʲɛrʲnʲɪs]。

**按發音部位的完全同化**：相遇的齒摩擦音和齒齦摩擦音（通常是清音）發音部位完全協調，只發出一個輔音：pùsšimtis \[ˈpʊʃʲɪmʲtʲɪs]（/sʲ/ + /ʃʲ/ → /ʃʲ/）。

**按軟化的同化**：輔音在前元音 \[æː]、\[ɛ]、\[eː]、\[iː]、\[ɪ] 之前的軟齶化（軟化），引起硬輔音音位與軟輔音音位的交替：dedù \[dʲɛˈdʊ]：dedi̇̀ \[dʲɛˈdʲɪ]，mãto \[²ˈmɑːtoː]：matýti \[mɐ¹ˈtʲiːtʲɪ]。

**混合趨同**（mišrusis supanašėjimas）：兩個輔音同時在多個特徵上趨同，例如 pùsdžiūvis \[ˈpʊʒʲʤʲu̟ːʋʲɪs] 中：/sʲ/ 既按聲帶活動也按發音部位向 /ʤʲ/ 調整（/sʲ/ → /zʲ/ → /ʒʲ/）；pùsžiemis \[ˈpʊʒʲiɛmʲɪs] 則是完全混合同化（/sʲ/ + /ʒʲ/ → /ʒʲ/）。

上述同化現象也發生在詞語交界處。非自主詞（無重音詞）與自主詞相遇時，輔音同化最為一貫；兩個自主詞相遇時，同化是可選的，例如：iš gãlo \[ɪʒ‿²ˈɡɑːloː]（清濁部分同化），bus žvalùs \[bʊ‿ʒʋɐˈlʊs]（清濁完全同化），vis čáižo \[ʋʲɪʃ‿¹ˈʧɑˑɪʒoː]（部位部分同化），nors šaũk \[noːr‿²ˈʃɒʊˑk]（部位完全同化），tik i̇̀šveda \[tʲɪˈkʲ‿ɪʃʲʋʲɛdɐ]（軟化同化）。
